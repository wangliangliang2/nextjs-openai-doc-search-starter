# 一、推广等级说明

**推广等级共分为 3 档，分别为青铜、白银、黄金，等级越高返佣比例越高，佣金上限越高。新加入的推广者默认为青铜等级。**

# 二、有效客户定义

**1、被邀请人完成个人 / 企业实名认证；**

**2、订单现金实付金额 ＞0元 的用户；**<br></br>



# 三、推广等级判定

**1、等级计算**

核算推广者近 3 个月的推广情况，根据推广者邀请的有效客户数来计算本月等级。（推广等级 = 近3个月累计邀请有效客户数 - 近3个月累计退款客户数）

举例：截止9月某白银推广者近3个月累计有效推广人数为105人，但近3个月共有9位被邀请人发起退款行为。则10月做等级调整时该推广者等级仍为白银等级。

**2、等级更新**

等级按月更新，每月1-3日确认推广者本月等级。

**3、跨级限制**

青铜等级的推广者次月最高升级为白银，不得升级为黄金；升级黄金等级前，原等级不得低于白银。
举例：某推广者初始等级为青铜，当月有效推广客户数为145人，次月做等级调整时，该用户等级为白银，因升级黄金等级前该用户等级不得低于白银，不允许推广者跨级。<br></br>
