# 一、推广资格界定
**1、注册七牛云账号并完成实名认证的个人用户。**

**2、企业用户及渠道代理商不能成为推广者。**

**3、七牛云内部员工不能成为推广者。**

**4、海外用户暂不参与新推官活动。**<br></br>



# 二、好友关联判断
**1、建立关联**

被邀请人通过专属推广链接注册 / 登录就会与推广者建立有效关联。在关联有效期内，被邀请人使用现金部分购买的所有订单产品，推广者均可获得 25% ~ 35% 的返佣奖励（[好友下单前请确定是否已经建立关联](https://portal.qiniu.com/cps/customer)）。

**2、关联失败**

<1>、付费老用户（销售团队直接服务）

<2>、渠道商 / 代理商用户（合作伙伴团队直接对接）

# 三、返佣计算规则
**1、推广佣金计算**

推广奖励佣金 = （被邀请人实付订单金额 - 退款订单金额） × 推广者等级对应的返佣比例

**2、推广返佣范围**

被邀请人在 30 天关联期间内产生的所有有效订单，均可根据推广者当前对应的返佣比例进行返佣奖励。

**3、单笔佣金上限**

被邀请人单笔订单返佣封顶为 5000 元。

举例：被邀请人在与推广者有效关联期间购买单笔订单金额为9万元，根据单笔订单佣金上限5000元的要求，所以该推广者最多在该笔订单中获得返佣金额为5000元。

**4、有效客户定义**

需同时满足以下两点：
1.完成个人 / 企业实名认证，订单现金实付金额 ＞0元 的用户。
2.无任何违法违规行为的用户<br></br>



# 四、推广佣金结算
**1、结算时间**

推广佣金暂不支持后台自主提现，每月1-6日七牛云财务系统会向您的银行卡汇入结算佣金，如遇节假日顺延至下一个工作日，具体到账时间以银行为准。（推广者及时在系统上如实填写个人银行卡信息，避免延误结算）

**2、打款要求**

当月最低打款要求未结算佣金累计 ≥100元。如推广者多笔佣金合计未满足每月最低打款要求，会累计到下个月直至满足最低打款要求后安排统一打款。

**3、结算周期**

佣金结算周期为5期，每期打款比例为返佣金额的20%。

举例：推广者在某月推广了一笔订单，返佣金额为450元并分5期结算，每期结算金额为90元，第一期佣金结算金额为90元，不满足最低打款要求，则累计到与第二期应发佣金一并结算，所以第二期打款金额为180元。第三期结算金额仍为90元，依然不满足最低打款要求并累计到第四期一并结算。剩于第五期的90元佣金需要满足当月最低打款要求后完成打款。

**4、佣金计算**

佣金计算：单笔订单实时计算返佣金额并分为5期打款。

订单退款：如有订单退款行为，系统会实时扣除该笔订单所奖励的佣金。如该笔订单佣金已完成结算，则会在下一期结算金额中扣除。以此类推。

# 5、重要风控事项说明

禁止链接劫持、强制捆绑、违反法律法规等的非正当方式推广。如推广者推广的客户涉及黑产及违规操作，推广者不会获得相应黑产订单产生的佣金。

新推官应在七牛云推广许可范围内，使用正当的手段方式进行推广，不应进行任何欺骗或虚假性质的推广行为，包括但不限于：
1、与其他新推官、或被推广客户相互串通，弄虚作假，恶意刷单，骗取返佣奖励；新推官如使用上述非正当方式开展推广活动，七牛云有权判定该等推广方式所带来的推广结果无效，并扣除佣金；
2、虚假、或夸大陈述七牛云活动的推广政策及优惠（如在其他平台以低于官网活动价形式进行推广售卖）；
3、采取诱导用户访问其网站、并通过放置网页插件、可执行代码等方式，强制与用户建立推广关系的；
4、其他侵犯被推广用户知情权、选择权的不正当推广行为；
5、其他七牛云判定推广手段属于不正当推广行为的；

注意：若发现虚假推广行为，将对月结佣金进行冻结扣除并终止推广合作。若您发现其他新推官采用不正当手段进行推广，支持提供证据进行举报。举报接收邮箱：zhongbowen@qiniu.com。
