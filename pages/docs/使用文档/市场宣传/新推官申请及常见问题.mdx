# 2、如何加入七牛云新推官？

**只需要四步就可以完成推广并获得返佣：**

**1.1、【加入】新推官**

[进入活动页面](https://www.qiniu.com/cps)， 点击 “加入新推官” 按钮，确认推广协议后，成为新推官。

**1.2、【分享】专属链接**

您可以将自己的专属推广链接分享到个人博客、技术论坛、技术网站等页面，引导客户访问并完成注册 / 登录。

**1.3、【关联】并支付订单**

引导客户完成注册 / 登录，实现 30 天有效关联，并在关联期间购买订单。

**1.4、【返利】获得奖励**

推广者在系统内填写好银行卡信息，七牛云会按月发放佣金奖励，打款至推广者的银行账号。<br></br>
