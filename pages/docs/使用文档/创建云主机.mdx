**【注意事项】**

1. 主机创建成功会弹出对话框提示用户云主机的管理员帐号和密码，谨记需要将此信息保存在安全的地方！用户可以通过操作系统对应的远程登录工具（ Linux 类操作系统一般为 ssh，Windows 操作系统推荐 Microsoft Remote Desktop 客户端 )。 

2. 云主机创建成功后立刻进入操作系统启动过程，由于各个操作系统在启动过程中的初始化项不同，可能遇到界面显示云主机已创建成功但仍不可远程登录的过程，用户可以稍等片刻再次尝试。
