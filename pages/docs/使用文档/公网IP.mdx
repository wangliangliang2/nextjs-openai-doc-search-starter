# 公网 IP 简介
七牛云主机系统中公网 IP 地址独立于主机或负载均衡器实例之外，您可以方便的根据需求创建公网 IP 地址，并按需将公网 IP 绑定到某个云主机上。

# 使用范围

公网 IP 可以绑定到主机上，也可以被绑定到负载均衡器上使用，都支持动态的绑定和解绑。
- 一个公网 IP 地址同一时间只能绑定到一个主机或负载均衡器上
- 一个 主机/负载均衡器 同一时间只能绑定一个公网 IP
- 公网 IP 不支持跨区域使用（ 如华北的 IP 不能绑定到华南的实例上 ）
- 目前系统最大带宽上限为 200Mbps，如需更大带宽请联系我们

# 使用说明
### **创建公网 IP**
1. 登录 EVM管理控制台

2. 点击左侧导航栏的 **公网 IP** 进入 公网 IP 管理界面

	![](https://dn-odum9helk.qbox.me/FiSHRXiXlIe_0Kaz8UgM4f5p7oNu)

3. 点击 **新建** 按钮

	![](https://dn-odum9helk.qbox.me/FgJ5mBbYMLDmZqP2M7-RJxd9Ybce)

4. 输入名称、运营商线路、带宽和计费类型信息，点击 **确认创建按钮** 创建

### **绑定公网 IP 到云主机**
1. 进入公网 IP 管理界面

2. 选择一个状态为 **空闲** 的公网 IP，并在列表上方点击 **操作** 按钮

	![](https://dn-odum9helk.qbox.me/Fnlv1umhmazsrCOtOmh74iQ17mfZ)

3. 在出现的下拉菜单中选择 **绑定到主机** 会弹出一个主机列表框

	![](https://dn-odum9helk.qbox.me/Fpf1FNNBbrdqfukJSJTqa8Z7q1Ec)

4. 在弹框中选择需要绑定的云主机，点击确定按钮即可

### **解绑公网 IP**
1. 进入公网 IP 管理界面

2. 选择一个状态为 **使用中** 的 IP，并在列表上方点击 "操作" 按钮

3. 在出现的下拉菜单中选择 **解绑公网 IP**

	![](https://dn-odum9helk.qbox.me/FiKJd58En-Ab-_9LG5OnA641W_dO)

4. 在弹框中点击确定按钮即可

### **调整带宽**
1. 进入公网 IP 管理界面

2. 选择一个按需计费类型的公网 IP，并在列表上方点击 **操作** 按钮

3. 在出现的下拉菜单中选择 **调整带宽**，会弹出调整带宽对话框

	![](https://dn-odum9helk.qbox.me/FrU89BOumLkjWgOOyfBHRdz9rpNi)

4. 在弹框中输入目标带宽（ 带宽上限为200Mbps ），点击确定即可

### **删除**
1. 进入公网 IP 管理界面

2. 选择一个非包月计费类型的公网 IP，并在列表上方点击 **操作** 按钮

3. 在出现的下拉菜单中选择 **删除**，在弹出框输入公网 IP 名称确认后输入用户登录密码即可

### **查看详情**
1. 进入公网 IP 管理界面

2. 点击需要查看详情的公网 IP 条目中的 ID 或者 **详情** 链接，即可进入详情查看页

	![](https://dn-odum9helk.qbox.me/FrLSdS3k326PGd1FhOoBpmtjLnxE)

	![](https://dn-odum9helk.qbox.me/FkFRGJ7_wOYivhUe0gg7QadQBtsU)
    
### **包月或续费**
对于计费类型为 **按需** 的公网 IP 资源可进行包月操作，计费类型为 **包月** 的公网 IP 资源可进行续费操作，操作方式类似 
<font color=red>注意：包月公网 IP 资源在资源到期之前，无法进行修改带宽操作和删除操作</font>
</br>
1. 进入公网 IP 管理界面

2. 选择一个公网 IP，并在列表上方点击 **操作** 按钮

3. 在出现的下拉菜单中选择 **包月 / 续费**，会弹出包月续费对话框

	![](https://dn-odum9helk.qbox.me/FlK8q1tYkKmT3MIf8-JnJTkB3W4G)

4. 在弹框中填写包月或续费的时长，点击确定即可
