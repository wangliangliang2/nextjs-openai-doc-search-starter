# **审核流程**
1. 申请短信签名或内容模版后，正常情况下，2个小时左右会反馈审核结果（审核时间：周一至周日 10:00 - 20:00，法定节日顺延）。
2. 如果您急需使用短信功能或者审核超时，请[提交工单](https://support.qiniu.com/tickets/new/)，我们会加速审核进度；如果审核未通过，也可咨询联系技术支持详细沟通。
3. 对于审核通过的内容模版，不表示一定会下发成功（运营商侧也会有抽查审核机制），如遇到短信发送失败，可咨询技术支持详细沟通。

# **审核状态说明**

| 状态 | 说明 |  
| --- | --- | 
| 审核中 | 已经提交的签名或者模板，正在审核中，审核周期为2小时内。 |  
| 审核通过 |已通过了审核，短信签名和短信正文模版都通过审核时，便可以开始发送短信。 |  
| 审核未通过 | 提交的签名或者模板未通过审核，可登陆[短信控制台](https://portal.qiniu.com/sms)查看签名或者模板的“未通过原因”，并根据原因修改后再次提交审核。 |
