# **前提条件**
1. 已开通短信服务。
2. 已添加短信签名和短信模板并通过审核
3. 控制台批量发送**需审核开通**，可[提交工单](https://support.qiniu.com/tickets/new)或者联系对应商务经理。
4. 只支持**推广、通知短信模板**，如有其他需求也可以提交工单。


# **操作步骤**


1. 进入[短信控制台](https://portal.qiniu.com/sms/send)，选择“发送任务> 批量发送”。
2. 填写任务名称。
3. 选择短信模板，如果没有模板，可添加先添加短信签名。
4. 设置接收号码。可选择**导入接收号码文件**或**手动输入接收号码。**
5. 定时发送（可选）：选择群发短信任务发送时间。
