简单的列举了短信模板控制台和API的功能。
# **控制台介绍**
| 功能 | 描述 | 
| --- | --- | 
| 新建模板 | 控制台[创建模板](https://portal.qiniu.com/sms/template)，创建时可参考签名审核标准。 |
| 查询模板 | 可根据关联签名、模板审核状态和模板内容定位模板|
