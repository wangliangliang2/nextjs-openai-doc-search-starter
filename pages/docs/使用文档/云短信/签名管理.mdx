
简单的列举了短信签名控制台和API的功能。
# **控制台介绍**
| 功能 | 描述 | 
| --- | --- | 
| 新建签名 | 控制台[创建签名](https://portal.qiniu.com/sms/signature)，创建时可参考签名审核标准。 |
| 查询签名 | 根据签名审核状态和签名内容定位签名 |
