七牛云提供云短信服务SMS（Short Message Service，SMS）支持向国际/港澳台快速发送验证码、短信通知和营销推广短信。

# 国际/港澳台短信发送步骤
发送国内短信，请遵循以下步骤：
1. 添加签名。
短信签名必须遵循[短信签名审核标准](https://developer.qiniu.com/sms/manual/5816/signature-specification)，且经过审核。
2. 添加模板。
短信模板必须遵循[短信模板审核标准](https://developer.qiniu.io/sms/manual/5814/template-specification)，且经过审核。
3. 发送短信。
通过[短信服务API](https://developer.qiniu.com/sms/manual/5824/through-the-api-send-text-messages)发送短信。
通过[控制台批量发送](https://developer.qiniu.com/sms/manual/5823/the-console-send-sms)短信。

# 国际/港澳台短信模板说明
如您需要发送国际/港澳台短信，请注意国际短信语言支持说明：

|发送目的国家与地区|支持模板语言|
|---|---|
|中国港澳台|简体中文、繁体中文、英文|
|日本|日文、英文|
|韩国|韩文、繁体中文、英文|

</br> 其它国家只支持英文
