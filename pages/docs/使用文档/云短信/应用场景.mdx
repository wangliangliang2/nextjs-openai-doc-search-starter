# **验证码下发**
- 网站，APP 账号注册，向手机下发验证码； 
- 登录账户、异地登录时的安全提醒； 
- 找回密码时的安全验证； 
- 支付认证、身份校验、手机绑定等。

# **系统通知推送**
向注册用户或会员发送系统相关信息，包括：升级或维护、服务开通、价格调整、订单动态、订单确认、物流动态、消费确认、支付通知、账户余额等普通通知短信。

# **营销活动推广**
向注册用户、会员和潜在客户发送通知和营销推广信息，包括促销活动通知、业务推广、新产品宣讲、会员关怀等商品与活动的推广信息。增加企业产品曝光率、提高产品和企业的知名度，支持高并发批量发送，可以满足会员生日、客户关怀、节日祝福、线上线下打折促销、业务推广、新品上线、客户维系等场景。
