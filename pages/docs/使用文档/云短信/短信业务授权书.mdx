# **背景**
如果用户 A 创建自用签名，签名归属主体属于用户 A ，则不涉及第三方权益。
当申请的签名与公司主体或账号主体信息不一致时，需要额外提交短信业务授权书。比如用户 A 替用户 B 创建签名，签名归属主体属于用户 B ，则涉及第三方权益，需要 B 给予 A 的授权委托书。B 为授权方，A 为被授权方，并需要加盖 B 的公章。您可在这里下载授权书模板，进行填写盖章。
