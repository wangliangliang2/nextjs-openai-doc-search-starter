# **联系人设置**
余额提醒和发送阈值的通知提醒会以短信和邮件的方式通知，根据业务需要添加多个联系人，可确保重要信息的及时查阅。



# **操作步骤**
1. 打开短信控制台，在左侧导航栏中单击**联系人管理**。
2. 点击新建联系人。

![](https://dn-odum9helk.qbox.me/FvetKaKtIFGMmmoJCqWrwV30_PSj)

| 配置	 | 说明 |  
| --- | --- | 
|姓名  | 联系人的姓名 |  
| 手机号 |联系人的手机号码，11位数字 |  
| 邮箱 | 联系人的邮箱 |  
| 用户资源包余额提醒 | 该联系人是否接收套资源余量预警短信和邮件 |  
| 账户发送总量阈值 | 该联系人是否接收账户发送总量阈值预警短信和邮件 |  
| 模板发送总量阈值| 该联系人是否接收模板发送总量阈值预警短信和邮件 |  


说明：
1. 默认情况下，联系人中只有主账号联系人，主账号联系人默认接收余额提醒和发送阈值提醒，且不可变更。**账号管理**中可更换主账号联系人。
