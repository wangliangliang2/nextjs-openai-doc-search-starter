# 帐号体系

七牛 EVM 账户和七牛云账户是统一的，所有七牛云用户可以使用已有帐号通过 https://evm.qiniu.com 直接登录七牛 EVM 平台管理云资源。如果还没有注册为七牛云用户，您可以通过访问 https://portal.qiniu.com/signup/choice 页面进行注册。

# 访问权限

目前七牛 EVM 服务仅针对企业用户开放使用，七牛云用户可以通过申请方式开通使用。如果您已经有七牛云帐号，但仍然无法登录七牛 EVM 管理平台，请联系对应的销售人员开通相关权限即可。

![](https://dn-odum9helk.qbox.me/FjPcKGpmaUJmqxTRI7_Ez9MpPo2_) 

# 登录

- 在浏览器中访问七牛 EVM 平台，即 https://evm.qiniu.com
- 输入您的七牛云帐号和密码
- 点击登录

![](https://dn-odum9helk.qbox.me/Finw1nFa5Db_UbyJqHqfOMzzmuIA)

# 忘记密码

由于七牛 EVM 的用户体系从属于七牛云账户，所以“忘记密码”的找回功能即是七牛云账户密码找回功能。用户可以点击登录页面的忘记密码按钮跳到七牛云账户的忘记密码页面，然后输入您的七牛云帐号注册邮箱地址，最后点击找回密码。

![](https://dn-odum9helk.qbox.me/FnmZONLDOih6gK9pC3DuPSvw3br0)

# 注销

在用户登录的状态下，点击导航栏右上角的用户按钮，然后点击弹出菜单中的“注销”按钮即可注销登录。

![](https://dn-odum9helk.qbox.me/FkmJEeRJ8r_5LIyJxjdeuSvO1Jq5)
