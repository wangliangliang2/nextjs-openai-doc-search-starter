# 1. 产品概述
<br/>
七牛云智能播控系统，是七牛云以业界领先成熟产品：直播云、融合CDN、智能多媒体API、深度学习平台等为技术能力基础，为专业的直播生产、运营用户打造的一站式直播播控SaaS产品。在云播控系统中，在可控的低延时下，用户可以快速处理多路音视频直播源，切换内容，添加图文、抠像、视窗、切换等效果。云播控提供的垫片处理、延时控制、智能审核等功能，可以确保直播的安全可靠播出。最终将切换后的效果，输出至用户指定的推流地址。

云播控系统同样支持在用户环境中的私有化部署，对直播环境中的系统进行智能化自动化的播出控制。

本产品采用预付费的购买使用模式，根据用户输入源数目、输出分辨率和使用时长进行计费，购买后，即开即播。

# 2. 使用场景
<br/>
## 2.1 多机位全接入
充分挖掘多机位直播内容资源，对多输入源，进行同步音视频切换。
<br/>
## 2.2 复杂直播控制
针对复杂的、需要多工种专业协同配合的直播场景，减少人工依赖，避免播出协同失误，完成智能化自动化播出控制。
<br/>
## 2.3 二次内容加工
对接入的直播源内容，进行广电演播室级的二次效果加工，或使原直播内容更符合移动端用户的观看体验。
<br/>
## 2.4 端到端全整合
提供从直播采集端到直播播出端一站式直播平台。
<br/>

# 3.产品优势 
<br/>
## 3.1 专业直播效果 
* 多路音视频处理，支持推拉流直播，切换、特效添加、输出等延时可控。 
* 多视窗、虚拟抠像、图文包装、画面动态切换等直播内容效果。 
<br/>

## 3.2 自动播出控制 
* 基于脚本自动化播控，支持MOS、VDCP、GPI等文稿与播控协议。 
* 基于切换画面识别、音频节奏视频深度学习的智能化播控。 
<br/>

## 3.3 智能直播安全 
* 智能黄暴恐内容监控、质量监控、延时控制、垫片应急等安全播出保障。 
<br/>

## 3.4 融合行业体验 
* 融合广电和互联网媒体行业特点的用户界面和用户体验。 
<br/>

## 3.5 多样内容增值 
* 直播广告植入等内容增值应用场景。 
<br/>

## 3.6 多平台类型部署 
* 公有云SaaS服务，计费简单，即开即播。 
* 支持私有化部署，全直播环境整合。 
<br/>

## 3.7 跨终端互动 
* 多平台，多终端，多媒体形式之间的无缝互动能力。 
* 多种形式的同步、极低延时的信息交流。
