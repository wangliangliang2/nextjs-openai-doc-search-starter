# 审计日志简介
七牛 EVM 为了方便用户复审对云资源执行的操作，会自动记录各个资源的审计日志。用户可以通过对审计日志的评审，追溯可能引发故障的误操作，为定位故障原因提供一定的线索。

# 如何查看审计日志
用户可以在控制台界面的审计日志中查看指定时间段内的所有资源的审计日志，如下图： 

![](https://dn-odum9helk.qbox.me/FpAOKCFdP39a7dTclfauJpIWtbek)

# 审计日志的分类
审计日志的分类包括 所有、主机、磁盘、公网 IP、磁盘快照、监听器、主机快照、负载均衡器、资源池和安全组，用户可以选择对应类别的审计日志，方便用户更加快速的定位日志，如下图：
