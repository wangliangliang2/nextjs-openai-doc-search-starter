# 云主机简介
七牛 EVM 云主机可以为用户提供不同的计算和存储能力。用户在创建七牛云主机的时候可以根据实际的需要选择不同的规格，这些规格决定了云主机的计算和存储能力。

# 镜像配置

镜像是 EVM 云平台预置的操作系统和软件的模板。用户在创建云主机时需要为云主机选择一个镜像，用户可以选择七牛 EVM 云平台默认提供的公共镜像，或者选择自定义的私有镜像快照。公共镜像为用户提供了标准操作系统和必要的基础软件设施；私有镜像为用户自定义的软件和数据环境，方便反复多次使用。

# 规格配置

用户在创建云主机的时候需要为云主机选择特定的规格配置，这里主要包括 CPU 核数和内存大小；CPU 核数决定了云主机计算能力的强弱，内存大小决定了云主机暂存程序和数据的多少。七牛 EVM 云主机为用户提供了多种规格的 CPU 核数和内存大小，方便用户根据实际的业务需要进行选择。

![](https://dn-odum9helk.qbox.me/Fo5RUM0LpxiGKjisfovmP2arUsi9)

# 网络配置

七牛 EVM 云平台默认为所有用户提供一个经典网络，经典网络下所有云主机共享同一网络，不同云主机通过安全组 (security group）隔离访问。同时，七牛 EVM 云平台也支持为用户创建独立的私有网络，提供更安全稳定的独享网络。  

<font color=Red>注意：相同网络下的云主机可以通过内网地址相互访问，不同网络下的云主机（经典网络和私有网络）相互隔离，用户在经典网络下创建的云主机与私有网络下创建的云主机不能通过内网地址相互访问。</font>

# 磁盘配置

云磁盘是云主机存储数据和程序的载体，创建云主机会默认提供一个系统盘用于存放操作系统必要的数据。用户如需更多的存储空间，可以通过为云主机添加云磁盘的方式扩展，挂载的云磁盘可以在云主机详情中的磁盘选项中查看。
