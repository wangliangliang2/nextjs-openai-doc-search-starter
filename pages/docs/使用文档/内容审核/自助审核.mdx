自助审核适用于对机审结果有人工复核需求的客户。
使用自助审核，用户能灵活配置人工复核流程和进审数据，并且与 iam 子账号深度集成，支持多轮审核，支持多人同时审核，灵活的回调以及扩展关联资源配置。

# 基本概念

审核任务：自助审核中的基础单位，可以进行关联审核数据、分配人员等操作。

任务通道：对应审核任务中不同步骤的数据集合，人员分配的具体单位。

# 审核任务

![自助审核任务](https://dn-odum9helk.qbox.me/Fl15jUWOE-cI8vplFqd8Hs3ZNrFz)

审核任务是自助审核中的基础单位，新建任务后，可以进行关联资源和人员分配等操作。

## 新建任务

新建任务需要完成以下步骤

### 基本设置

![基本设置](https://dn-odum9helk.qbox.me/FppirDybHElKw2gFPGmKlMZy74-W)

审核任务的基本信息

| 字段 | 选项 | 说明 |
|------|-----|-------|
| 文件类型 | 图片/视频/文本/音频 | 审核任务所支持的文件类型，不同文件类型会有不同的复核操作，创建后不可修改 |
| 任务名称 |  | 任务名称，有意义的命名将有助于后续的运营管理 |

### 模式设置

![模式设置](https://dn-odum9helk.qbox.me/FqY8DDTWnjCN7D4iYC-j2trQSquW)

选择自助审核的模式，目前支持以下模式：

#### 快速审核模式

简单的流程配置，通过一轮人工复查，快速确认及修正机审结果。

支持根据机审结果过滤进审数据。

并且根据进审数据的设置，该模式包含以下任务通道：

* 高危审核
* 低危审核
* 正常审核

复核后，流程结束，复核结果作为自助审核的最终结果。
  
#### 严格审核模式

通过三轮人工审核，最大限度保证审核的准确性。

支持根据机审结果过滤进审数据。

并且根据进审数据的设置，该模式包含以下任务通道：

* 高危/低危/正常一审
* 高危/低危/正常二审
* 专家终审

当一审和二审复核结果一致时，则流程结束，二审结果作为自助审核的最终结果；

当一审和二审复核结果不一致时，则需要进行专家终审；

专家终审后，流程结束，专家终审作的结果为自助审核的最终结果。

#### 自定义审核模式

请联系技术支持人员，定制适合业务场景的审核模式。

| 字段 | 选项 | 说明 |
|------|-----|-------|
| 审核模式 | 快速审核模式/严格审核模式/自定义模式 | 参考上述 |
| 进审数据设置 | 机审结果违规/机审结果疑似/机审结果正常 | 选择审核模式后，可以根据机审结果过滤进审数据，可多选，默认为 机审结果违规 和 机审结果疑似 |
| 回调地址 | | 填写回调地址后，自助审核的最终结果将通过改地址回调给用户 |

### 其他设置

![其他设置](https://dn-odum9helk.qbox.me/FiWMryjCxxmmZFFRB2tS8oUPZTkV)

#### 对象存储

当进审的数据来源为对象存储时，可以设置根据审核的最终结果 解封/封禁/删除 资源。

| 字段 | 选项 | 说明 |
|------|-----|-------|
| 启用设置 | | 开启审核任务的对象存储设置 |
| 结果正常 | 无/解封文件 | 无：不进行任何操作 <br /> 解封文件：当审核的最终结果为正常时，解封已经封禁的文件 |
| 结果违规 | 无/解封文件/删除文件 | 无：不进行任何操作 <br /> 解封文件：当审核的最终结果为违规时，封禁文件，封禁后的文件不可直接访问 <br /> 删除文件：当审核的最终结果为违规时，删除文件，`删除后不可恢复，请谨慎操作` |

## 关闭任务

`关闭审核任务会导致资源漏检，增加运营风险，请谨慎操作。`

可以关闭确定不再使用的审核任务。关闭后，将不会有新的数据进入。

## 开启任务

已关闭的任务在确认配置后，可以直接保存重新开启。

# 关联资源

![关联资源](https://dn-odum9helk.qbox.me/FhWWqn6VX8HUEFRp8lMtgQKWJkh6)

新建任务后，需要关联资源后数据才能进入审核任务，进行复核。

目前支持与以下资源关联：

* [内容审核 API](/censor/api/5588/image-censor)
* [存储空间审核/增量任务](/censor/manual/8066/censor-portal-stream)

关联关系为一对一多，即一个审核任务可与多个资源关联，但每个资源只能与一个任务关联。当选择了已与其他任务关联的资源时，保存时会提示确认重新关联资源，确认后即可取消原有的关联，并与当前任务关联。

# 人员分配

![人员分配](https://dn-odum9helk.qbox.me/FlM1706pVAu_1H2h5-opw7RHisTB)

自助审核与 iam 子账号系统深度集成。已开通 iam 子账号的用户，可以对任务进行人员分配，将任务的各个通道分配给 iam 子账号，支持同一通道多人同时复核。

子账号权限设置见 [内容审核 iam](/censor/manual/10244/censor-iam)

# 复核

![复核](https://dn-odum9helk.qbox.me/FhJHGYTYoJpKLIuWv5P9uDHFjz5G)

用户可以从图片复核、视频复核、文本复核和音频复核分别进入不同文件类型的复核界面。

如果用户被分配了多个任务通道，则首先需要选择复核的任务通道。红点表示当前该任务/通道有待复核的数据。

用户可以通过 正常 和 违规 按钮设置当前数据的复核结果。

可以通过开关设置点击复核按钮后是否需要二次确认，关闭二次确认能减少点击次数提高复核效率，但无法防止误操作。

同时用户也可以通过快捷键复核。`快捷键无需二次确认`。具体快捷键可以点击快捷键按钮查看。

完成复核后会自动获取下一条待复核数据。若当前无数据，则会提示无更多数据，此时用户还可以通过点击下一条尝试刷新获取更多数据。

右下方会提示当前剩余待复核的数据数量。

## 视频复核

![视频复核](https://dn-odum9helk.qbox.me/FhAfvwdTveLwCPz77GAMMRrDFsgi)
