# 结果复核获取资源使用的域名和方式，是否产生 CDN 流量和费用？

对于增量和存量审核的结果，结果复核时会通过七牛内网的方式访问，和用户域名无关，不产生 CDN 的流量和费用；对于直播相关的结果，截帧和音频由内容审核服务保存和提供，不会产生额外流量和费用；对于 API 调用的结果，复核时将通过调用 API 时的原链接获取资源。

# 结果复核时资源无法加载、显示？

请先按如下步骤自行排查：

* 检查资源是否存在，是否可以正常访问，不论是对象存储中的资源还是用户的外部资源
* 如果用户资源有防盗链设置，则需要在白名单中加上 portal.qiniu.com 的域名

有更多问题可以提交工单/联系技术支持反馈给我们。

# 增量任务是否可以关闭或删除？

增量任务暂时不支持删除操作。如需关闭，可点击 增量任务列表 -> 任务卡片 -> 更多 -> 关闭任务 按钮，确认后即可关闭停止任务。`关闭审核任务会导致资源漏检，增加运营风险，请谨慎操作`。

# 存量任务是否可以关闭或删除？

存量任务开启后`不支持后续的用户修改、删除和关闭操作，请开启前仔细检查确认配置`。如需操作，请提交工单/联系技术支持。`关闭审核任务会导致资源漏检，增加运营风险，请谨慎操作`。

# 如何开通华东区域外的视频增量/存量审核？

控制台上暂时不支持用户主动开通华东区域外的视频增量/存量审核。如有需求，可填写配置信息，提交工单/联系支持，由我们内部为用户开通。配置信息参考：

* [增量审核配置](/censor/manual/8066/censor-portal-stream)
* [存量审核配置](/censor/manual/8067/censor-portal-batch)
