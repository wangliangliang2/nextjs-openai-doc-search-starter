# 市场需求
市场上越来越多的客户由于自己本身或者自己的客户在数据业务方面安全性和合规性的要求，或者整体项目综合成本的考虑，希望采取私有化的部署方案。我们推出了七牛智眼服务器（内容审核一体机）来满足客户的业务需求。
# 产品方案
七牛智眼服务器整合硬件计算存储资源、智能多媒体服务、人工智能算法模型为一体，最大限度的提升软硬件的综合能力，达到最佳的使用方式。

1 **标准化**。兼容支持公有云上所有已经标准化的内容审核功能，比如鉴黄、鉴暴恐、政治人物识别，支持图片及视频格式。

2 **硬件算法定制**。服务器及GPU资源授权定制化，提升整体效果，在客户业务场景下，充分发挥硬件资源的使用率，达到最佳配比。

3 **可扩展**。基于客户业务场景，处理能力弹性可扩展，算法模型后续可迭代升级维护。

4 **监控可视化**。提供监控服务，实时监控计算、存储等资源实时消耗情况，监控异常并报警。
