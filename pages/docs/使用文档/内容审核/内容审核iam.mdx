对于已经开通 IAM 子账号功能的用户，可以通过 `访问管理` 控制子账号使用内容审核产品的权限。

# 开通内容审核服务

在 `访问管理` > `策略管理` 中，新建策略 / 编辑已有策略时，将 `内容审核` 加入到服务列表中。

# 查看存储空间审核任务

开通 `查看存储空间审核任务` 功能后，子账号将可以查看存储空间审核任务的列表、数据统计、查看审核结果并进行复核

`注意`：为了能正确使用存 查看存储空间审核任务 功能，子账号需要同时开通以下 对象存储 的功能

|功能|说明|
|-|-|-|
| 读取文件| 子账号只能查看自身拥有读取文件功能的存储空间的审核任务，其他审核任务不可见 |
| 禁用/启用文件 | 子账号在复核 或 批量封禁存量任务结果 时，需要开启对应存储空间的 禁用/启用文件 功能，否则会失败 |
| 删除文件 | 子账号在复核删除文件 或 批量删除存量任务结果 时，需要开启对应存储空间的 删除文件 功能，否则会失败 |

# 管理存储空间审核任务

开通 `管理存储空间审核任务` 功能后，子账号将可以新建/修改/关闭审核任务

`注意`：为了能正确使用存 管理存储空间审核任务 功能，子账号需要同时开通以下 对象存储 的功能

|功能|说明|
|-|-|-|
| 读取文件| 子账号只能对自身拥有读取文件功能的存储空间开启审核任务，否则会失败 |
| 禁用/启用文件 | 开通增量审核任务并设置自动封禁时，子账号需要开启对应存储空间的 禁用/启用文件 功能，否则会失败 |
| 上传文件 | 开通视频审核任务或存量审核任务时，子账号需要开启截帧/结果保存对应存储空间的 上传文件 功能，否则会失败 |

# 调用内容审核 API

开通 `调用内容审核 API` 功能后，子账号可以使用自身的 AK/SK 调用内容审核 API。参考[调用内容审核 API](/censor/5588/image-censor)

# 查看内容审核 API 结果

开通 `查看内容审核 API 结果` 功能后，子账号可以在控制台上查看内容审核 API 调用的结果并进行复核。

# 自助审核

开通 `自助审核` 功能后，子账号可以在控制台上进行自助复核。

# 管理自助审核任务

开通 `管理自助审核任务` 功能后，子账号可以在控制台上管理自助审核任务，包括以下操作：
- 新建/修改/关闭/开启自助审核任务
- 对自助审核任务进行资源关联
- 对自助审核任务进行人员分配

`注意`：关联资源时，需要对应资源的读权限。参考：[查看存储空间审核任务](#1)
