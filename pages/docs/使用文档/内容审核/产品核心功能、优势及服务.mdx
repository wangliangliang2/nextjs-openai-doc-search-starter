# 产品核心功能
## 鉴黄

七牛云图像鉴黄利用超大规模数据所提供的知识来对深度神经网络进行引导，训练出多个极具泛化能力的网络模型，同时基于蒸馏学习的思想对模型复杂度和参数规模进行了大幅压缩，快速准确的识别出色情、低俗的图片和视频，解决直播、电商、社交等领域的违规内容问题。

## 鉴暴恐

七牛云鉴暴恐通过海量暴恐图片和视频数据源，依托分布式深度学习平台，能准确地对图片视频进行暴恐分类，具体支持血腥类，爆炸类，恐怖主义，战争军队，枪支刀具，各种旗帜等不同类型。

## 政治人物识别

七牛云政治人物识别基于海量人脸库和专业审核人员的审核标准，利用分布式深度学习平台，识别正常、漫画、负面涉政人物的违规信息，降低违规风险，覆盖涉政人物1700余个，具体包括国内外元首，副国级以上领导人，落马官员，反华势力和劣迹艺人等。

# 产品优势
** 性价比高：** 智能机器审核在节省97%以上的人力成本的同时，达到99%以上的准确率，支持秒级返回结果。同时，七牛云内容审核产品依托于 AVA 深度学习平台不断快速迭代，优化算法。

** 使用便捷：** 一次接入即可提供图片、视频等形式内容审核，覆盖鉴黄、鉴暴恐、涉政等风险防范。更支持其他公有云资源的API调用接入。

** 对风险理解更专业：** 与多地的监管部门合作，并成立联合实验室，能让七牛云的内容审核更加贴近真实使用场景，从而提供更加专业的算法和产品。

# 产品服务
七牛云内容审核服务提供标准审核流程，无论客户的资源在何处，都可以提供内容安全保障。如果客户资源在七牛云，更能形成审核业务闭环，减少处理流程和操作步骤。
