介绍欠费状态下使用各种产品/服务的限制。

**一、欠费保护期状态下限制功能**

| 模块 | 限制功能 | 
| --- | --- | 
| 财务功能 | 无法使用账户余额支付订单，只能使用支付宝/微信等第三方支付。 |
| CDN	 | 无法新增域名和修改域名。 |
| 对象存储	 | 详情见[账号欠费保护期功能限制](https://developer.qiniu.com/kodo/manual/6219/accounts-owe-protection) |
| 云主机	 |详情见[账户冻结后资源停用删除说明](https://developer.qiniu.com/qvm/manual/6649/disable-delete-resources-following-the-account-is-frozen) |
| IoT视频云	 |1.我的应用<br>（1）创建应用<br>（2）修改应用的基本信息<br>（3）修改云存配置<br>（4）创建/编辑Topic<br>（5）编辑日志配置<br>2.设备管理<br>（1）创建设备/批量创建设备<br>（2）批量获取密钥<br>（3）编辑设备信息<br>（4）更新设备影子 |

**二、如何解除欠费**
在财务中心-账户充值，进行充值，充值抹平欠费后将解除欠费状态。
