# **二、抵用券分类**
根据抵用券适用的系统/付费类型的不同，分为：

-账单券：适用于按量付费的后付费产品费用。
-订单券：适用于流量包、SSL证书、包年包月的云主机等一次性结算的产品费用。


# **三、抵用券优惠规则**
## **1、满减类型**
根据有无金额门槛，分为：无门槛券和满减券。

## **2、抵用券面额**
抵用券能抵扣的最大金额。


## **3、抵用券门槛**
使用抵用券需要满足的费用门槛。


## **4、生效时间**
抵用券的有效时间范围。


# **四、抵用券使用规则**

## **1、使用次数**
根据抵用券能够使用的次数，分为：仅一次使用和可多次使用。

## **2、付费类型**
-预付费（即订单券）
-后付费（即账单券）

## **3、适用产品/商品**
抵用券能适用在哪些产品/商品上。


## **4、抵扣规则**
账单券：账单出账扣费后，系统将自动抵扣多张有效状态、符合时间范围、符合适用范围的账单抵用券，优先抵扣快过期的抵用券。
订单券：订单券不会自动抵扣，需要您选购好商品下单时，主动选择需要使用的抵用券，再进行支付。

# **五、抵用券使用注意事项**
## **1、订单抵用券**
（1）每个订单最多只能使用一张抵用券，抵用券不可叠加使用。
（2）订单退款时，抵用券不支持退回；并且抵用券无法转让、提现，抵用券支付的金额无法开票。
（3）抵用券无法抵扣欠费金额。

## **2、账单抵用券**
（1）抵用券无法抵扣欠费金额。
（2）若通过活动获得抵用券，请于激活绑定抵用券后再使用，已使用或绑定成功的抵用券将显示在已激活抵用券列表。
