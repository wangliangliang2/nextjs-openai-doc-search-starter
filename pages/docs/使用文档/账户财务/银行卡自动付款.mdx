# 功能介绍
### 概述
适用于海外后付费用户，添加银行卡，月初出账后（1～4日）系统从添加的银行卡中自动扣取账户未支付金额，无需担心忘记还款影响使用。
### 支持卡类型
支持带有 Visa、Mastercard 标志的信用卡/借记卡。
### 特别说明
- 添加银行卡后会自动获得 $ 5 欠费额度，超额仍会触发账号冻结，若需提高欠费额度请联系商务或提交工单；
- 目前该功能支持2023年2月28日之后（含当日）注册的海外用户使用哦，在此之前注册的用户若有需求，请联系商务或提交工单。

# 操作流程
### 1、添加银行卡
前往【[支付方式管理](https://portal.qiniu.com/financial/payments)】页面，点击【添加支付方式】按钮，填写银行卡信息。
- 添加银行卡过程中，会进行 3D 验证，若认证不通过请联系您的发卡行核对哦。
- 3D 校验后会冻结 $ 5 以验证您的身份，验证通过后会立即取消冻结。（此操作并不会真正扣费）

![image.png](https://dn-odum9helk.qbox.me/FsLOoZWaB7srgGEA_NgfKiA2_Cq3)
### 2、设置为默认支付方式
- 添加首张银行卡会自动设置为默认支付方式，非首张银行卡可添加时设置也可添加后自助设置；
- 更改默认支付方式会自动取消已有信用额度（若有），您可以联系商务重新设置。
![image.png](https://dn-odum9helk.qbox.me/FlZsXYsAHuyl5GuhbU4j1fiN5vFY)
### 3、删除银行卡
- 非默认支付方式您可自助删除；
- 默认支付方式暂时不支持删除，您可更换默认支付方式再进行删除。如有任何需求请联系商务哦～
![image.png](https://dn-odum9helk.qbox.me/FhvKbuIHfEuTS6Mc5-TRxgKt2Gze)
### 4、出账扣费
每月出账后（1～4日）系统从添加的银行卡中自动扣取账户未支付金额，您可以在【[财务中心/账单管理](https://portal.qiniu.com/financial/bills/settlement)】核对账单金额。
