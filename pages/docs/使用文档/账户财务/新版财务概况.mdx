# 二、账户金额说明
* 可用额度：可以用来购买云服务的余额。
 * 可用额度=现金余额+赠送金余额-欠费金额-实时消费金额
 * 支付费用时，七牛云将优先扣除抵用券、赠送金余额，再扣除现金余额。
* 现金余额：通过线上充值或线下银行转账充值的现金余额。
* 赠送金余额：只能支付按量付费产品的费用，不能提现和开发票。
* 欠费金额：用户使用七牛云服务产生的欠费金额，但不包括已消费未出账金额和已出账未扣费金额。
* 实时消费金额：七牛云会在账户内暂时冻结实时消费金额，冻结的金额并未实际消耗，更多说明请参考 [实时消费明细](https://developer.qiniu.com/af/manual/6167/real-spending-detail)
