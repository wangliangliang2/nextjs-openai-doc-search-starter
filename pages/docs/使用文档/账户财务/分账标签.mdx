# 介绍
### 概述
分账标签作为分账账单的辅助功能，方便您将分拆后的账单按分账标签汇总。目前仅支持将存储标签设置为分账标签。

### 应用场景
用户域名 / 存储空间数量较多，并且希望按不同层级分摊账单金额。如按照“项目 - 部门 - 分公司”分别分摊账单金额。

### 使用方式
1、【按标签汇总】页面可查看 / 下载按分账标签汇总后的金额。
![image.png](https://dn-odum9helk.qbox.me/FmD5R07QxzGIH0UukW8oLaG0vDvY)
2、分账标签会作为单独一列展示在【分账明细】、【按资源汇总】页面中，可下载后按分账标签筛选汇总。
![image.png](https://dn-odum9helk.qbox.me/FrckHxesl91kaqRBPC5xdaOSPmle)
# 操作流程
### 1、开启分账功能
联系商务或提交工单，开启分账功能。开启分账功能后才能看到分账标签页面。
### 2、查看详情
点击【详情】按钮，可查看标签与资源绑定情况。
![image.png](https://dn-odum9helk.qbox.me/FucB8k5ZrSVkXD19Do10PCmIM08I)
![image.png](https://dn-odum9helk.qbox.me/FlUT6LqdsWT56KTX4AHkzqnln1ok)
### 3、设置分账标签
设置时间：每月月末 24:00 前设置的分账标签才能在当月分账账单中展示，例如 6月30日 24:00 前设置的分账标签才能在 6 月分账账单中展示。
设置数量：最多可设置 15 个分账标签。

![image.png](https://dn-odum9helk.qbox.me/FmiYMk_4bhv6LoruwGEzppsgCKDa)
### 4、新增 / 修改 / 删除标签
新增 / 修改标签请前往相应的产品线页面设置，设置后会自动同步到分账标签页面。
若在产品线页面删除标签，则在分账标签页面也会自动删除该标签，无论该标签是否设置为分账标签。

### 5、查看/下载分账账单
【按标签汇总】页面可查看 / 下载按分账标签汇总后的金额。
分账标签也会作为单独一列展示在【分账明细】、【按资源汇总】页面中，可下载后按分账标签筛选汇总。
![image.png](https://dn-odum9helk.qbox.me/Fo1llftVtut-93UyGpyA9ZB0XbEk)
