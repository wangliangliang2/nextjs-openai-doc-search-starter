# 介绍
### 概述
分账账单将账单金额分拆至资源粒度，目前支持将存储、CDN 账单金额分别分拆至存储空间、域名粒度。

### 应用场景
可用于财务成本核算，项目 / 部门单独计算成本，为您企业内部成本分摊提供参考（不作为对账依据）。

### 分账账单三种展示形式
1、分账明细
将账单中计费项金额按照组成该计费项的存储空间 / 域名用量分摊。
![image.png](https://dn-odum9helk.qbox.me/Foitrag8OcO4poKmgkOnagncP7vv)

2、按资源汇总
将分账明细分别按照存储空间、域名汇总。
![image.png](https://dn-odum9helk.qbox.me/FnZ3_CZphmE-KfSNFos9PxDUCJzk)

3、按标签汇总
将分账明细按预先设置好的分账标签汇总。
![image.png](https://dn-odum9helk.qbox.me/FncCtLGh_qjMTFVj2H0eJzsXFPdI)

# 操作流程
### 1、开启分账账单
联系商务或提交工单，开启分账功能。开启分账功能后才能看到分账账单页面。
### 2、生成分账账单
每月出账后系统自动生成分账账单，推荐每月第 5 日起查看。
### 3、查看 / 下载分账账单
您可在分账账单页面自助查看 / 下载相应的分账账单。
