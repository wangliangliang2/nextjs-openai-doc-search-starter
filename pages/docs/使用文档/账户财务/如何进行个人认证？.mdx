个人认证有三种认证方式：个人扫码认证；个人银行卡转账认证；个人银行卡认证
# **扫码认证**
扫码实名认证是通过现场对着真实本人录制视频。
视频录制成功后，系统进行活体检测及人证核身校验，校验通过立即完成认证。

# 第一步

使用手机微信扫描二维码。

![image.png](https://dn-odum9helk.qbox.me/FoFwV6B8ms_lq2tR7hKnHyHDvkHZ)


# 第二步

手机端根据页面提示，手动输入个人身份信息。

# 第三步

根据界面引导进行人脸核验。人脸识别成功，系统会提示您认证成功，人脸识别失败，系统会提示您认证失败。

PC端会同步您在手机端的认证结果。

<br>



# **银行转账验证**

**该认证方式需填写个人信息及财务信息**

若个人信息审核无误后，七牛云会向您所填写的银行卡里面打一笔0.01～0.99 元之间的小额款项，请在收到款项后，回填打款金额完成认证。
# 第一步

**填写个人信息和财务信息**

- 请确保持证件照清晰。
- 请填写开户银行及支行信息。
- 银行开户名需要与认证填写的身份证信息一致。

![image.png](https://dn-odum9helk.qbox.me/FnFcQq83mD7lRAwwzV1uRWjnWthb)


# 第二步

**基本信息审核**

七牛云将在 3 个工作日内完成基本信息的审核，审核结果会以短信的方式通知。


![](https://dn-odum9helk.qbox.me/FiH4sAq2CvsESwc8m2YAxqWyyo2I)


# 第三步

**待汇款**

七牛云将在 3 个工作日内向您所填写的银行卡里面打一笔0.01～0.99 元之间的小额款项，请在收到款项后，回填打款金额完成认证。

![](https://dn-odum9helk.qbox.me/Fg3rKQpsh2zDdOw5-iol1w7GlC9J)

# 第四步

**待验证收款**

请输入附言对应的打款金额进行认证。

<font color="red">注意：输错 3 次后会自动认证失败，需发起重新认证。</font>

![](https://dn-odum9helk.qbox.me/FqMGO2o4-3SXiMRH5PfO2dpvNuLs)

# 第五步
**认证完成**
<br>

# **银行卡认证**
银行卡认证是以个人客户的银行卡进行实名认证，支持个人借记卡和信用卡。个人客户可以输入银行卡信息进行实名认证。

# 第一步

使用手机微信扫描二维码。
![image.png](https://dn-odum9helk.qbox.me/Ftoe3u9Qpu9AqeUsZ6SLDr93bqqq)

# 第二步

手机端根据页面提示，拍摄完整面部照片，完成上传。

# 第三步

返回PC端继续操作。填写完整相关信息，点击“立即提交”按钮完成认证。
