# 简介
实名认证即认证使用主体身份。据《网络安全法》规定，用户必须进行实名认证，否则不得使用相关服务。

个人客户可进行个人认证或企业认证，企业客户只可进行企业认证。个人认证成功后也可升级成企业认证。企业认证成功后不可降级为个人认证。

如账号为企业使用，请务必进行企业认证。因为个人认证与企业认证在账号归属上有极大不同，同时遇到人员变更或账号财务纠纷，错误的账号属性可能对个人/企业产生不良影响或带来经济损失。
