# 一、设置入口
登录账号后，进入管理控制台>财务中心>[财务概况](https://portal.qiniu.com/financial/overview)，点击开启「可用额度预警」。

![](https://dn-odum9helk.qbox.me/FrE7D5SdcwqPMVtafAX4ydzBY8ws)

 

# 二、设置阈值
请输入大于0的数值，最多精确到小数点后2位。

![](https://dn-odum9helk.qbox.me/FiGGBgGKp82TZRwCRB9IuChDYkZL)



# 三、预警规则
**1.通知条件。**

当您账户的可用额度低于预警阈值时，系统会每天为您发送短信和邮件预警通知。

**2.通知对象。**

七牛云会通知到您账户绑定的邮箱和手机号。如需修改可以前往 个人中心>[安全设置](https://portal.qiniu.com/user/security)，更换邮箱和手机号。

**3.通知频率。**

每天发送一次。

# 四、关闭通知
1.点击开关，将预警设为关闭。
