# 产品定义
云存储网关是一款可在线下和云上部署的软网关，以七牛云对象存储的作为后端存储，可在企业的内部IT环境和基于云的存储基础设施之间提供无缝、安全的集成。支持行业标准的文件和块存储协议，并通过在本地缓存经常访问的数据来提供低延迟性能， 您可以安全地将数据存储在七牛云对象存储中。

文件网关将七牛空间的对象结构与文件系统的目录与文件建立映射关系。用户通过POSIX协议即可读写指定七牛空间里的对象。同时，网关还利用本地存储空间做为热数据缓存，使用户在享受海量空间的同时，保障数据访问的高性能。文件网关还高度兼容第三方备份软件。用户如果是小文件的备份和共享读写场景，推荐配置标准型和基础型的文件网关；如果对性能有一定要求的或者同时有多个客户端访问数据的用户，推荐性能型配置的文件网关。

### **高性能智能缓存策略**

提供本地缓存模式，灵活的缓存策略，异步多线程的方式将数据同步到云端，给您良好读写体验。

### **跨平台支持**

支持直接挂载到服务器上，也支持NFS/CIFS/FTP等协议。

### **高效的数据上传和校验机制**

异步多线程上传机制，自动数据一致性校验

### **完备的日志和监控**

提供从存储网关到云上存储空间全量操作日志，提供多维度的指标监控
