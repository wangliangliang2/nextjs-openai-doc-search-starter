# Windows 内置远程连接
1. 如果本地是 Windows 机器 点击【开始】- 【运行】，输入"mstsc" 命令，即可打开远程桌面连接对话框。在输入框输入 Windows 服务器的公网IP（在 EVM 控制台登录后的 [云主机列表](https://evm.qiniu.com/machines/) 可查看云服务器的公网IP），如下图所示：

	![](https://dn-odum9helk.qbox.me/FmCPiJ6PfDqkH6PFkh-4G29S7yBM)

- 点击 **连接** 按钮 在打开的Windows登录界面中输入账号Administrator 和主机的密码即可登录  
	![](https://dn-odum9helk.qbox.me/FoeqK9OXrImsoHTAF0kpRrdJ_EQa)

# Mircrosoft Remote Desktop 登录
1. 使用微软提供的 **Microsoft Remote Desktop** 软件进行登录  

	![](https://dn-odum9helk.qbox.me/Fj_vFkjqOdfBWGpeTv3ln6BG4Ijr)

- 打开应用 点击 **New** 新建按钮  

	![](https://dn-odum9helk.qbox.me/FsTsQsSrXU3CMv1nv1OML62qt6ip)

- 可以看到新建远程桌面的对话框  

	![](https://dn-odum9helk.qbox.me/FiuD961WvpfnnDndRCBieXzCWpFf)

- 在关键的 **PC name** 输入框中填上主机的IP 地址或主机名可以直接在 ** Credentials ** 下面的 **User name** 和 ** Password ** 中填入 ```Administrator``` 和主机的密码，软件会自动帮你做登录认证的过程。  

	![](https://dn-odum9helk.qbox.me/FiXA1N1L7Xme2vuOLX7yl151QvJy)

- 在桌面列表中点击我们刚刚创建的远程桌面就可以登录到云主机  

- 更多详细的关于微软远程桌面问题可以查看 
[Microsoft 远程桌面客户端](https://msdn.microsoft.com/zh-cn/library/dn473009)
