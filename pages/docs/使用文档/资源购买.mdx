七牛 EVM 中以下资源需要用户购买使用:

- 云主机
- 云主机快照
- 云磁盘
- 云磁盘快照
- 公网 IP (包括带宽)


<font color=Red>注意：负载均衡默认不收取任何费用，且负载均衡默认不包括公网 IP 服务。</font>
<font></font>
七牛 EVM 以上需要纳入计费的云资源都会在创建的时候为用户提供计费模式选项的选择，目前包含“按需”和“包月”两类。  

在购买的时候会根据用户选择的云资源类型和规格，给用户显示一个该资源的预估价格。

![](https://dn-odum9helk.qbox.me/FoYQQKldu7wiCJjULIL8p9hIQnhR)

七牛 EVM 针对特定的云资源，如磁盘快照和主机快照，只提供默认的按需付费模式。用户在创建该类型云资源时无需选择计费模式。
