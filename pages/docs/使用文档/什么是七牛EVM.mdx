# 简介
七牛 EVM 是七牛 Enterprise Virtual Machine 的简称，是七牛面向用户提供的企业级云主机、云硬盘和虚拟网络服务。
通过 EVM 云主机服务，企业只需要根据业务发展需要创建相应规模的虚拟机实例、云硬盘和网络服务，再也无需提前进行相关资源的采购，缩短了业务部署上线的周期，同时大大降低了企业的运维成本。

了解七牛 EVM 时，通常会涉及到以下概念：

- **区域 (region)**，即 EVM 服务部署的数据中心 [地域和可用区](/evm/manual/3983/area)
- **云主机**，即运行操作系统的 [实例](/evm/manual/4032/cloud-hosting-an-overview)
    - **主机镜像**，即创建云主机的基础操作系统模版，包含服务器的预配置环境（操作系统和其他已安装的软件）
    - **主机配置**，即创建云主机所需的物理资源规格，包括 CPU、内存和网络等
- **云硬盘**，即云主机可挂载并可用作持久化存储的块设备
- **云网络**，即云主机使用的网络服务，包括经典网络和私有网络
- **公网 IP**，即云主机可绑定并用以公网访问的 IP 地址，包括创建时指定的带宽
- **安全组**，即云主机访问权限控制
- **SSH 登录**，即用户通过更高安全性的 SSH 密钥对的方式登录云主机

# 优势

- 定制化服务：面向企业用户打造更安全稳定的云主机服务，为企业用户提供定制化方案（包括但不限于虚拟网络优化、私有网络配置和高带宽配置服务等）
- 存储网络互联：支持与同一区域七牛云存储内网互联互通，满足企业对存储在七牛云存储中的数据就近计算的需求
- 安全性：支持配置安全组规则、多用户隔离
