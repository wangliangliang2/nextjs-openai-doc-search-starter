# SSH 密钥简介
使用云服务器实例的第一步是登录云服务器实例，为保证实例的安全可靠，七牛云提供两种加密登录方式：密码登录和 SSH 密钥对登录。

七牛云允许使用公有密钥密码术加密和解密对于 Linux 实例的登录信息。公有密钥密码术使用公有密钥加密某个数据（如一个密码），然后收件人可以使用私有密钥解密数据。公有和私有密钥被称为密钥对。用户可以通过密钥对安全地与云服务器进行连接，是一种比常规密码更安全的登录云服务器的方式。

要使用 SSH 密钥登录您的 Linux 实例，您必须首先创建一个密钥对，并在启动实例时指定密钥对的名称，然后使用私有密钥连接实例。七牛云只会存储公有密钥，您需要存储私有密钥。拥有您的私有密钥的任何人都可以解密您的登录信息，因此将您的私有密钥保存在一个安全的位置非常重要。

- 目前只有华东区域开启了 SSH 密钥在线管理功能
- Windows 实例不支持 SSH 密钥登录。

# 使用说明
### **创建 SSH 密钥**
1. 打开终端

2. 输入以下命令
```bash
ssh-keygen -t rsa -b 4096 -C "your_comment"
```
之后会有三次信息输入提示，可以直接按回车使用默认信息，或者根据需求填写

3. 生成好后，SSH 密钥会默认存放在 **~/.ssh** 目录，您可以使用以下命令查看您的公钥
```bash
cat ~/.ssh/id_rsa.pub
```

### **导入 SSH 密钥**
1. 登录 EVM管理控制台

2. 点击左侧导航栏的 **SSH 密钥** 进入 SSH 密钥 管理界面

	![](https://dn-odum9helk.qbox.me/FneBXthc45WVDtu_kUWjBsWHAQh3)

3. 点击 **导入 SSH 密钥** 按钮

	![](https://dn-odum9helk.qbox.me/FnFNvPbEb9qiiBTEnWAVE_-c4kom)

4. 输入名称和您生成好的 SSH 公钥，点击 **确认** 即可

### **绑定 / 解绑 SSH 密钥到主机**
1. 进入 SSH 密钥 管理界面

2. 选择一个 SSH 密钥

3. 点击列表上方的 **绑定 / 解绑** 按钮，会弹出一个主机选择列表

	![](https://dn-odum9helk.qbox.me/FuFCaaSPbcMTwPA2-O73deN_BThL)

4. 选择需要绑定的主机点击确定即可

### **删除**
1. 进入 SSH 密钥 管理界面

2. 点击列表上方的 **删除** 按钮，在弹出框输入密钥名称点击 **确定** 按钮，然后输入用户登录密码，防止用户的 SSH 密钥 被恶意删除

	![](https://dn-odum9helk.qbox.me/FmvumaFu1ItNwzhKqO2JElD862WH)
