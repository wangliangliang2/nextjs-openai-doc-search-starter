# 退款条件

免费域名型(DV) SSL 证书无法申请退款；
已经由CA机构成功颁发的 SSL 证书无法申请退款；
已完成订单支付、审核流程中非客户缘故所导致的未成功颁发的 SSL 证书可申请退款。
