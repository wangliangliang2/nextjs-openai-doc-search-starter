胶囊柱图是一种特殊的柱状图，通过柱子上的进度条表示指标的占比情况。一般用来展示指标为占比值的统计信息，如在线率、合格率等。

### **样式配置说明**
胶囊柱形图的配置项由数据、图表样式、颜色3部分组成。

#### **数据**
数据用于选择胶囊图数据展示相关信息

|配置项|说明|
|-|-|
|分组字段|用于分组的字段，每一个柱子表示一个组结果|
|统计指标|用于展示一个组数值的字段，可以设置其单位和精度|
|数值单位|统计指标的单位|
|数值精度|统计指标的数值精度|

#### **图表样式**
数据用于配置胶囊图的样式

|配置项|说明|
|-|-|
|指标显示|配置显示的指标值，支持显示指标和占比情况|
|汇总方式|计算每一组占比的分母选择方式|
|柱子形状|胶囊图柱子的样式|
|标签位置|每一组名称标签的展示位置|
|数值样式|可以配置展示数值的字体大小和颜色|
|标签样式|可以配置标签的字体大小和颜色|



#### **颜色**
颜色用于配置胶囊柱子的颜色

|配置项|说明|
|-|-|
|柱子颜色|表示每个柱子的颜色|


### **场景示例**
查询过去7天不同操作类型的使用情况
```
repo="_internal"
| stats count() by action.name
| sort by count
| rename count as "操作次数",action.name as "操作类型"
```

- X轴选择操作类型，统计指标选择操作次数绘制胶囊图
- 标签样式选择上方，更好地展示标签过长的情况
