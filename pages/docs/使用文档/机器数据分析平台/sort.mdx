sort 命令按指定字段对所有结果进行排序（按字典顺序）。

**语法**

sort `[<N>]` by `<sort-field>`\[, `<sort-field>`\]

**参数说明：**

* `<N>`：可选参数，指定要从排好序的结果中返回的结果数。如果未指定计数，则使⽤默认限制值 10000。如果指定为 0，则返回全部结果。

* `<sort-field>`：field [desc|asc]，排序所依据的字段的列表及其排序顺序。desc 表⽰降序，asc 表⽰升序，默认按 desc 排序。

**用法示例**

```
sort 10 by _time asc, sourcetype
```
