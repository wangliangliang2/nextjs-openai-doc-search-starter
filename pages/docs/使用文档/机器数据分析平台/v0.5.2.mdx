**新功能**

* SPL 新增支持 [Append](/express/manual/6685/append)、[IPLocation](/express/manual/6689/iplocation) 算子。

* 支持[胶囊柱图](/express/manual/6690/capsule_histogram)图表类型。

* 支持将搜索分析结果导出至日志仓库。

* 接收数据支持对数据源编码方式解析(utf-8除外)。

**Bug 修复**

* 字段提取自动进行字段类型推断，将字符串字段推断为整型字段。

* 仪表盘字段返回值错误。

* 搜索任务导出停止失败。

* SPL 命令相关：

 * stats 命令的 distinct 函数返回不准确。

 * bin 命令对除 _time 以外的字段进行时序分桶失败。

 * bin 命令对时序数据以月（M），年（y），季度（q）， 周（w）为单位分桶返回结果不准确。

* 调用 api 创建仓库，存储天数为负数不报错。
