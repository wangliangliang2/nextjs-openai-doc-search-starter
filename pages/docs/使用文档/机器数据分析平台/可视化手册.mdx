**可视化功能**是您探索数据的有力工具，您可以看到原始数据经过配置之后呈现出了多种多样的数据报表，这些报表为您展示数据分析结果的同时使得数据探索过程生动有趣。同时，将报表保存到**仪表盘**，您可以在任意时刻通过仪表盘数据报表观察原始数据，如通过单值占比图查看服务器 CPU 占比情况，及时调整运维策略，又或者，通过饼图查看来自不同 IP 的访问数量，及时识别恶意访问 IP。Pandora 2.0 支持您使用 spl 或者通过交互式的方式配置报表，如果您是spl使用专家，您可以使用 spl 进行任意的数据探索，配置出相应的分析报表。如果您没有那么擅长spl，平台也支持通过交互式的方式配置报表。本文档将为您介绍：

* 如何使用 spl 配置可视化报表
* 如何使用交互式方式配置可视化报表
* 图表类型详解
* 如何使用仪表盘
