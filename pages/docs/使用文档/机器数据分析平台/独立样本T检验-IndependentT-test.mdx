独立样本 T 检验用来检测输入的两组样本数据之间是否存在显著性差异。

**语法：**

```
...|score ttest_ind equal_var=<true(default)|false> alpha=<float> <a_field_1> ... <a_field_n> against <b_field_1>... <b_field_n>
```

**参数说明：**

- equal_var，用来指定给定的两个数据之间的方差是否相同，默认为 true。当 equal_var=true 的时候，使用标准的双独立样本 T 检验；当 equal_var=false 的时候，使用 Welch's T-test 检验。
- alpha，用来指定显著性水平，即在零假设为真时拒绝零假设的概率，默认是 0.05。

**使用场景：**

- 支持 1-1，1-n，n-n 的两两比较，
- 在 1-n 的时候支持模糊搜索（ \*）。
- 独立样本 T 检验的 H0 假设：两组样本数据之间不存在显著性差异；H1 假设：两组样本数据之间存在显著性差异
