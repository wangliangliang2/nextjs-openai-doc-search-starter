# 新功能
- 权限
  - 支持内置对象授权
  - app导入逻辑调优
- 查找（lookup）
  - 新增inputlookup算子，可在线查看lookup的csv文件
  - 新增outpitlookup算子，可将搜索结果增加到csv文件中
- 监控控制台
  - 新增权限控制
- 系统账户
  - 新增双因子认证，支持邮箱、验证器两种方式进行双因子认证
- 低代码平台
  - 新增组件：预建片段、表单项组、swith
- matrix 引擎
  - 新增rex算子
  - 新增算子函数：分位数、mode 
- 帮助文档
  - 站内文档全新升级，提供图文并茂的产品使用文档和应用开发文档
  
# 功能优化
- 任务管理
  - 所有任务增加搜索来源，方便用户管理任务
- 首页首次加载性能优化
- 仪表盘打开性能优化
