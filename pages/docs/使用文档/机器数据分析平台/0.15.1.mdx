# 新功能
- 外部数据库：能够在PDR平台远程连接并查询数据库数据
  - 新增数据库连接，支持mysql、oracle数据库连接
  - 新增sql查询，通过sql即可查询已连接的数据库数据，可快捷保存为告警
- 搜索动作：将搜索结果与与其他环节进行串联、交互，形成完整的分析场景闭环
  - 指定分析结果点击可跳转新搜索、报表、仪表盘、URL、发送post请求、关联知识库
  - 搜索动作管理：支持启用/禁用、编辑、复制、导入、导出配置、授权、删除、重新分配所有者
