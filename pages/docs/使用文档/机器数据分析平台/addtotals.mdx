`addtotals` 命令对每一行指定的 N 个字段计算算术和.

**语法：**

```
addtotals [fieldname=<field>] [field]...
```

**参数说明：**

* `fieldname`: 可选，用于指定显示算数总和的字段名称。
* `field...`: 可选，指定对哪些字段进行求和；如果未指定，则对所有数字类型字段求和。

**用法示例：**

有如下数据：

products|Q1|Q2|Q3|Q4
--------|--|--|--|--
ProductA|1000|1400|1300|1500
ProductB|1100|1175|1250|1700
ProductC|2000|1550|1375|1625

对每一行求和：

```
... | addtotals fieldname=total
```

products|Q1|Q2|Q3|Q4|total
--------|--|--|--|--|-----
ProductA|1000|1400|1300|1500|5200
ProductB|1100|1100|1250|1700|5150
ProductC|2000|1550|1300|1600|6450
