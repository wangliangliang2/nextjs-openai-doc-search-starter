使用summary命令来展示之前保存过的机器学习模型的总结概览信息。summary命令不需要依赖任何数据源，只需要提供模型名称，不同模型summary展示的信息不一定一致。

部分算法产生的模型不支持使用summary算子，是否支持请参考[算法及支持的SPL算子](/express/7422/the-algorithm-provided-by-the-app-and-spl-operator)。

**语法**
```
| summary <model_name>
```

**参数说明**

***必选参数***
<model_name\>：必须提供一个通过fit命令保存过的机器学习模型的名称。



**示例用法**


1. 返回某线性回归算法的summary结果。线性回归算法的summary会返回模型中所有特征及其系数和截距。
```
|summary LinearRegression_test_1
```

2. 返回某随机森林分类器的summary结果。随机森林模型的summary会返回模型中所有特征及其特征重要性。
```
| summary RandomForestClassifier_test_1
```
