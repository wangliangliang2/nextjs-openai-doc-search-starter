## 概览
可以对服务、实体添加标签，添加后有利于服务、实体的管理和过滤查找。

## 新建标签
单击标签Tab，选择新建标签，配置以下配置项以定义一个新的标签。或者当不存在想要的标签时，可以直接点击新建进行快捷创建。。

| 配置项 | 是否必填 | 配置项解释 |
|-|-|-|
| 名称       | 是      | 标签名称，全局唯一 | 
| 描述   | 否       | 标签描述      | 
| 实体    | 否       | 当前标签下面的实体，支持添加多个。 |   
| 服务    | 是       | 当前标签下面的服务，支持添加多个                       | 

## 标签管理
标签管理界面对所有的标签进行统一管理，支持对标签进行编辑、删除（单个、多个）、查看详情、给某个标签批量添加实体/服务，同时支持对标签的模糊搜索。

## 查看标签详情
点击标签名称，可以查看标签详情。包括当前标签下面的所有实体和服务信息。
