**使用场景及限制：**

- 统计函数帮助计算和提供数据相关的统计信息。
- 支持 1-1，1-n，n-n 的比较。
- 数据必须为 int 或者 float，不支持 str。
- 需要提前将空值行去除，不然会影响计算的结果。
- 有一些函数支持两组字段，有一些函数只支持一组字段。
- 在一组字段的情况下，支持使用模糊搜索（\*)

**通用语法：**

```
 ..| score <scoring-method-name> [options] a_field_1 a_field_2 ... a_field_n [against b_field_1 b_field_2 ... b_field_m]

```
