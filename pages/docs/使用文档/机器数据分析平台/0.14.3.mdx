# 新功能

- 数据管理
  - 仓库支持全文索引和分词器设置
  
- 搜索
  - 搜索限制支持设置系统用户变量，如user.name,user.nickname，可以实现根据不同登录用户控制搜索限制
  - 支持搜索框的高亮，格式化，补全，错误提示
  - 搜索框支持添加注释，注释内容以灰色字体呈现

- 告警
  - 告警呈现原始事件、聚合事件概览统计信息，方便用户实时监控系统发送的告警信息

- 查找（lookup）
  - 支持时序查找：如果查找表中有时间字段，可以创建基于时间的字段查找
  - 支持自动查找：自动查找将字段查找针对相应的数据范围自动运行，在这种情况下，字段查找在每次搜索时自动运行
  - 支持字段查找高级配置：支持最小匹配、最大匹配、默认匹配等高级配置
  
- 新增低代码组件
  - 新增低代码组件：SPL编辑器、text area、panel、radio
  
- 平台监控控制台
平台监控控制台提供对平台节点、查询任务、流量、机器的统一监控：
  - 支持分析任意节点任意线程池的stack信息和cpu、iowait等信息
  - 支持针对任意节点绘制火焰图
  - 支持展示系统和节点Settings
  - 节点-分片详情新增disk信息
  - 支持数据的生命周期管理展示
  - 支持根据水位配置来判断磁盘健康
  - 增加profiling分析，可分析查询效率与性能

- 系统设置
  - 平台支持外观设置：可自定义平台的系统名称、登录页图片、系统logo等外观设置，满足自定义外观需求
  - 平台账户密码系统安全策略增强：新增可配置密码复杂度、密码过期提醒、密码过期强制修改安全策略，保障系统账户密码安全

- Pandora智能服务分析APP（PISA）
  - PISA的服务分析器、服务、服务/KPI告警、实体、标签模块支持功能权限，可配置功能的查看、编辑、删除等权限
  - PISA的服务分析器、服务/实体导入、标签、KPI/服务告警支持知识对象配置授权
