Pandora 2.0 正式对外发布 v0.2.3 版本。

功能如下：

* 数据搜索和可视化
	* 发布动基于态Schema（Schema-On-Read）的数据分析平台
		* 灵活处理非预定的非结构化、半结构化数据分析工作，帮助数据分析更加实时和便捷
	* 正式发布搜索处理语言（Search Process Language, SPL）
		* 提供灵活的类SQL查询语言，支持管道计算，能够持续迭代计算，完成高级分析任务
		* 支持上百种计算命令，适应各种复杂场景
	* 支持用户使用交互的方式对数据进行分析和可是户，也支持用户使用搜索语言对数据进行可视化配置。
		* 通过仪表盘统一管理报表。

* 支持多种形式的数据采集
	* Web界面通过文件上传
	* 监听 TCP/UDP 端口
	* 监听 HTTP 端口
	* 通过第三方采集工具上报数据

* 数据索引和解析
	* 自动识别事件时间、断行的方法。将日志文件分解成单个独立的事件。
	* 通过使用多种方式可以对原始数据进行解析和字段提取：
		* 支持数据中KV 字段自动提取发现
		* 支持JSON、XML字段自动提取
		* 支持划词辅助+正则表达式提取
		* 支持固定分隔符数据提取
		* 搜索语言解析，支持在搜索结果中使用正则表达式、Eval计算等多种方式对数据进行二次解析运算
	* 字段映射：将原始字段映射为新的字段
