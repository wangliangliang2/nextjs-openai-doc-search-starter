您可以从 Pandora 2.0 中导出搜索分析结果至第三方系统，目前支持通过使用[export](/express/manual/6362/export)搜索命令导出和交互式导出。
本文档介绍如何通过交互式操作导出搜索分析结果至第三方系统。

点击`搜索分析->搜索导出任务`进入搜索导出任务管理界面。

### 创建搜索导出任务

**1.在搜索导出任务管理界面点击创建导出任务。**
* 输入任务名称；
* 输入导出SPL语句，形如`……|export ……`，如
 
 ```
sourcetype="json" AND repo="default" AND origin="jingwei.json"| export type="json" sink="es" host="10.200.20.39:9999"
```
* 选择导出频率，支持的频率包括：`每分、每小时、每日、每周、每月、每季度、每年`，选择频率之后，您也可以自定义具体的发送时间。如：选择每天导出一次搜索结果，您可以指定发送时间为每天的 16:00 发送。您也可以使用Crontab表达式设置导出频率。

![](http://pw7x36qrk.bkt.clouddn.com/export_es.png)


**2.在搜索界面输入搜索分析语句，并另存为导出任务。**

![](http://pw7x36qrk.bkt.clouddn.com/save_as_export.png)

您输入搜索分析语句之后，点击`另存为->导出任务`，在弹窗中搜索语句部分补足export命令即可。

参数说明与`在搜索导出任务管理界面点击创建导出任务`一致，这里不再赘述。

>注意：默认情况下，export命令导出1个小时内的搜索分析结果。如果您想指定时间范围，可以通过搜索分析界面的时间选择器指定时间范围或者通过search命令的start，end参数指定时间区间。形如：
```start="-3h" end="-2h"……|……|export type="json" sink="kafka" host="10.200.20.39:9092" topic="test_dump"
```
导出从前3小时到前2小时的搜索分析结果。关于search命令的start，end参数，[点此](/express/manual/6021/search)了解详情。


### 导出任务管理

Pandora 2.0 平台支持对搜索导出任务进行如下操作：

1.编辑任务：可修改任务的搜索分析语句、名称和导出频率。
2.停止、启动任务。
3.查看运行历史，包括任务开始事件、持续时间、导出状态（是否成功）。
4.删除任务。
5.导入导出任务。支持通过导入JSON文件创建导出任务，支持导出任务配置为JSON文件。
