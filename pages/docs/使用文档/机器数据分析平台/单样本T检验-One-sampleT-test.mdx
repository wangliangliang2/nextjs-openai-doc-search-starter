单样本 T 检验用来检测输入的单样本数据与总体数据是否有显著性差异。

**语法：**

```
...| score ttest_1samp popmean=<float> alpha=<float> <field_1> <field_2> ... <field_n>
```

**参数说明：**

- popmean，用来指定总体数据的平均数，必填。
- alpha，用来指定显著性水平，即在零假设为真时拒绝零假设的概率，默认是 0.05。

**使用场景：**

- 支持模糊搜索（ \*）。
- 单样本 T 检验的 H0 假设：样本数据和总体数据之间不存在显著性差异；H1 假设：样本数据和总体数据之间存在显著性差异
