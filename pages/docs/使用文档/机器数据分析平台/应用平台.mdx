应用平台的 app 为您的各种使用场景提供解决方案，通过预先配置好的仪表盘、搜索、数据来源等，帮助您快速对数据进行探索分析。
![](http://pw7x36qrk.bkt.clouddn.com/app.png)

Pandora 团队为您提供了以下 app 安装包：

* DNS Insight - `v0.1.0` [点击链接下载](http://q3d80qvam.bkt.clouddn.com/dns-insight-0.1.0.tar.gz)
	* 提供 DNS 流量、性能、以及其它 DNS 相关统计信息的可视化，帮助查明 dns 服务器中的错误和异常。

* Apache Tomcat - `v0.1.0` [点击链接下载](http://q3d80qvam.bkt.clouddn.com/apache-tomcat-0.1.0.tar.gz)
	*  监控 Apache Tomcat 服务器使用情况，帮助您深入了解整个 Web 服务器中网站访问者的行为模式。同时还提供了对 Tomcat Servlet 容器，Catalina 和垃圾回收器的高维度的分析。

* Nginx 日志分析 - `v0.1.0` [点击链接下载](http://q3d80qvam.bkt.clouddn.com/Nginx-0.1.0.tar.gz)
	* 分析 nginx 服务器的性能，并帮助您了解整个 Web 服务器中网站访问者的行为模式。
	
* 机器监控 - `v0.1.0` [点击链接下载](http://q3d80qvam.bkt.clouddn.com/metrics-0.1.0.tar.gz)
	* 提供对 CPU、磁盘、内存、网络、TCP 等机器指标的分析。

* MySQL - `v0.1.0` [点击链接下载](http://q3d80qvam.bkt.clouddn.com/mysql-0.1.0.tar.gz)
	* 帮助您深入了解 MySQL 服务器的运行状况，跟踪缓慢的查询，有助于在问题变成严重问题之前对其进行故障排除。

* Github - `v0.1.0` [点击链接下载](http://q3d80qvam.bkt.clouddn.com/github-0.1.0.tar.gz)
	* 在组织或存储库级别连接到您的 GitHub 存储库，并通过 Webhook 接收 GitHub 事件。这些事件将填充到预配置的仪表板中，从而为您提供 GitHub 分支，问题，拉取请求，用户活动和安全事件的完整概述。
