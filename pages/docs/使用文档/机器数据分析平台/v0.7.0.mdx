**新功能**

* 支持字段加速。

* 系统支持搜索任务管理，管理正在运行或运行完成的搜索、仪表盘、数据集、报表任务。

* 应用平台支持通过交互式操作创建应用对象（视图、来源类型、提取规则）。

* 支持通知中心，系统磁盘用量等指标告急时发送站内通知。

* 应用平台新增集群监控app，呈现集群的节点、仓库、分片的分布以及数据量等指标信息.

* 账号系统支持对接SSO。

* 支持仓库高级配置，可设置仓库副本数量、并发度、自动刷新间隔等。详情阅读[仓库管理](/express/manual/6886/repo)。

**性能优化**

* 优化字段自动发现性能。

**功能优化**

* 企业微信告警内容模板优化。

* export 命令增加导出前参数验证，不合法的导出语句运行后系统提示export语句格式错误。

**Bug 修复**

* sort 命令对搜索结果排序不稳定。

* 编辑字段内容类型的模板变量，取值字段未正常显示。

* 用 json 解析方式提取字段时，无法识别数组中的 bool 类型字段。

* export 命令导出搜索结果至仓库，数据中的时间戳与指定的 sourcetype 中时间戳不一致。

* convert 命令在仓库包含多分片的情况下对数据转换失败。

* append 命令在仓库包含多分片的情况下失效。

* 仪表盘中图表下载内容为明细日志。

* 仪表盘被权限更高者（非仪表盘创建者）编辑后所有者变为权限更高者。

* transaction 命令默认参数反序列化后无法解析。

* 下载搜索分析结果需获取任务管理权限。
