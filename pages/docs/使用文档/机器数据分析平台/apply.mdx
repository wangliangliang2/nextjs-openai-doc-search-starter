使用apply命令来把通过fit命令计算出来的模型应用到搜索出来的数据上，用来进行结果的预测。Apply命令不但可以用在原先的训练数据上，也可以用在验证数据或测试数据上，输出的结果形式也是相同的。
**语法**
```
apply <model_name> [as <output_field>] 
```
**参数说明**
***必选参数***
<model_name>：“模型名称”选项是必不可少的，它一般来源于之前用fit命令得到的输出结果。
***可选参数***
[as <output_field>]：有些算法会需要一个“输出字段”的选项，用来将基于模型预测的输出字段名称修改为一个指定的名称。它以关键词“as”为引导，接下来指定一个可以自定义的字段名称。
**示例用法**
1、使用之前构建的house_model预测结果
```
... | apply house_model
```
2、使用之前构建的house_model预测结果，并将保存预测结果的字段命名为“predicted_value”。
```
... | apply house_model as predicted_value
```
