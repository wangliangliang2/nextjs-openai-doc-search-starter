如果您的数据有如下特点：

* 通过⼀个共同的分隔符（例如空格、逗号或管道符号）清晰地分隔开。

* 每条日志格式⼀致（每条日志的每个值都是在相同的位置）。

这通常是使⽤结构化、基于表格的数据的情况，例如 .csv ⽂件或从数据库索引的事件。

例如：
```
2018-01-02T04:00:00Z,20,15,1,sensor1
```
这样一条日志明显符合上述数据特点，使用固定分隔符方法可以快速对其进行解析。

![](http://pw7x36qrk.bkt.clouddn.com/csv_parser.png)

选择合适的分隔符对原始数据进行解析，一键生成字段名与值一一对应的表格，字段名称与类型可编辑。

如果您的数据中有以逗号、空格、换⾏符和制表符等分隔符进⾏分隔的字段/值对（或只是字段值），此时选择分隔符提取字段将会很有⽤。
