KPSS 检验用来检测序列是否为平稳型数据，与 ADF 检测的目的类似。

**语法：**

```
...| score kpss regression=<c(default) | ct> lags=<int> alpha=<float> <field>
```

**参数说明：**

- regression，用来指定需要检测的平稳型类型。
  - regression=c（默认值）有常数均值
  - regression=ct，有趋势
- lags，用来指定延迟阶数，默认为 None。当 lags=None 的时候，设延迟阶数为 int (12 \* (n / 100)\*\*(1 / 4)),其中 n 为样本数量。
- alpha，用来指定显著性水平，即在零假设为真时拒绝零假设的概率，默认是 0.05。

**使用场景：**

- 只能对单个字段做检验。
- 不支持模糊搜索（ \*）。
- KPSS 检验的 H0 假设：序列为平稳型数据；H1 假设：序列为非平稳型数据。
