# 新功能
- 数据脱敏：Pandora支持对数据进行脱敏，以更精细地管理数据访问控制、保护数据安全。
  - 脱敏配置：具有脱敏功能权限的用户可新建、编辑、删除脱敏规则，内置中文姓名、手机号码、银行卡号、电子邮件、企业单位名称等脱敏规则；也支持自定义规则，如保留n至m，输入起始位置n和结束位置m，从n至m保留，其余部分脱敏处理，例如SK***155。
  - 针对角色配置脱敏限制：管理员在分配数据权限时，可以制定某个仓库的数据对角色展示脱敏前数据或脱敏后数据，如果展示脱敏后数据，则搜索时结果展示脱敏后的数据，以达到更细粒度数据安全控制。

- 交互式搜索
  - 新增交互式搜索，用户可以在搜索界面通过交互式添加过滤器的方式进行快速数据查询，无需手动输入SPL。降低搜索门槛，提升搜索可用性。
  - 支持多种过滤器类型，支持同时配置多个过滤器。对于过滤器，支持快速反选、启用/禁用、编辑、删除单个、多个过滤器。
  - 交互式搜索支持切换搜索引擎，快速查看生成的SPL
  - 支持在搜索界面快速保存、打开报表，对于常用的搜索，通过报表形式保存能更方便地使用

- 仪表盘分享
  - 内部分享：支持仪表盘内部分享，可选择是否分享仪表盘的模板变量、时间选择配置
  - 外部分享：支持仪表盘外部分享，即用户无需登录系统即可查看仪表盘结果，可配置分享链接有效期。注意此操作会让用户无需登录查看仪表盘结果，请谨慎操作。

- 仓库管理
  - 仓库支持配置容量上限配置，用户可制定仓库数据的容量上限和超过上限的处理规则

- 账户体系
  - 新增新建用户时限制用户首次登录强制修改密码，且新建用户/重置密码时可通过企业微信、邮件、钉钉方式通知用户账户信息
  - 用户密码支持随机生成密码

- 低代码平台
  - 支持页面新增应用：可在页面快速创建、编辑应用。
  - 低代码新增p-for、p-computed语法
  
- matrix 引擎
  - _raw 字段的存储优化
  - sort 下推到 Lucene 执行
  - timeline 性能优化

- 监控控制台
  - 新增告警预览：包括运行告警个数、错误数、告警运行耗时等指标
  - 新增默认告警规则：包括license相关告警，节点load、disk相关告警，集群健康状态告警，collector 错误告警，用户可自定义配置通知信息进行告警通知
