# 创建磁盘快照
- 点击左侧导航栏，进入 **磁盘管理**。选中磁盘，点击 **操作** 下拉框，选择 **创建快照**,随后可在弹窗中为快照进行命名操作。

	![](https://dn-odum9helk.qbox.me/FvCVD0QyCXhsg9Ts4n-EgpHZwkaY)

# 通过磁盘快照创建新磁盘

- 点击左侧导航栏，进入 **磁盘快照**。选中某个磁盘快照，点击上方的 **恢复磁盘** 按钮。

	![](https://dn-odum9helk.qbox.me/FmY6efEnFSFvqk0nJFffQqoVIK34)

- 进入 **恢复磁盘** 界面后，便可通过该快照进行新建磁盘。如下图：

	![](https://dn-odum9helk.qbox.me/FpdVxAaWbHMKV69EZvzJiJQlj5hh)

**【注意事项】**

1. 磁盘快照的大小和原云磁盘大小一致

2. 磁盘快照的磁盘类型为 SATA 

3. 磁盘快照的计费类型只能为按需模式
