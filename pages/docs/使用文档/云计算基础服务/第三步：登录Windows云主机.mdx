## 1. 

对于购买了公网带宽/流量的云主机，可以从您的本地 Windows 机器登录 Windows 云主机（ 远程连接 MSTSC ）：

- 在本地 Windows 机器上，点击【开始】- 【运行】，输入"mstsc"命令，即可打开远程桌面连接对话框。
- 在输入框输入 Windows 服务器的公网IP（登录 [云主机控制台](https://portal.qiniu.com/kylin/) 可查看云主机的公网IP），如下图所示：

![](https://dn-odum9helk.qbox.me/FvuJ9lQRIYvf9LCDDo8UseQmg-pI)

- 点击【连接】，在新打开的界面中输入帐号（Administrator）和您设置的初始密码即可登录。

![](https://dn-odum9helk.qbox.me/FsmUd35pBCaHKtlarmDfMzmNBCzB)
