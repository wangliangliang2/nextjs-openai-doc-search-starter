# 云主机的地域选择

首先，您需要确定云主机所在地域。

> 地域选择小贴士：
> - 应根据您的用户所在地理位置选择地域。云主机与访问客户端距离越近，越能获得较小的访问时延和较高的访问速度。比如您的用户大部分位于长江三角洲附近时，上海地域是较好的选择。
> - 请注意，不同地域之间的云主机不能通过内网互相通信（通信需经过公网，收费），相同地域下的云主机可以通过内网相互通信（内网通信不收费）。

# 云主机的配置选择
  
七牛云提供如下推荐配置：

- 入门型：1核1GB内存，带宽选择1Mbps或使用流量。适合个人博客等小型网站。
- 基础型：1核2GB内存，带宽选择1Mbps。适合企业官网等简单的Web应用。

推荐配置只是七牛云提供的配置中很小一部分，仅供您参考。七牛云提供了更多灵活的配置方案。当然您也可以在购买云主机之后，根据您的需求随时进行 [调整实例配置](/fec/manual/adjust-the-instance-configuration)。
