七牛云主机系统提供以下网络和安全功能

- **安全组**  
  
  您可以使用安全组来控制您的实例的访问权限。这些安全组类似于一个网络防火墙，使您可以指定允许访问的协议、端口和源/目的 IP 范围。您可以创建多个安全组，并给每个安全组指定不同的规则。然后您可以给每个实例分配一个或多个安全组，我们将按照这些规则确定允许哪些流量可访问实例以及实例可以访问哪些资源。您可以配置一个安全组，以便只有特定的 IP 地址或特定的安全组可以访问实例。
- **加密登录方式**  
  
  七牛云主机提供两种加密登录方式：密码登录 和 SSH 密钥对登录。用户可以自由选择两种方式安全的与云服务器进行连接。
- **公网 IP**  
  
  您可以按需创建不同运营商、不同带宽大小的公网 IP，并动态的将其绑定到您有静态公网 IP 地址需求的主机上。我们的 Internet 链路接入了国内主流网络运营商，保证您的客户不管使用哪家 ISP，都能享受相同的高速访问；内网链路均经由底层的万兆/千兆网络互通，保证高速互访、高可靠性和低时延。  
  
  用户的网络环境可以大体分为 基础网络 和 私有网络 两种。在基础网络下您的云产品实例位于七牛云预设的大资源池中；私有网络下您的云产品实例可被启动在您预设的、自定义的网段下，与其他用户相互隔离。

以上的各种网络与安全服务保障您的实例安全、高效、自由地对外对内提供服务。
