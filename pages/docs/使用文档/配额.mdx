# 查看配额
七牛 EVM 默认所有资源都包含了配额限制。用户登录 EVM 控制台后系统会跳转到“用户概览”页面，该页面展示了用户当前各类虚拟资源的默认配额和使用情况。

![](https://dn-odum9helk.qbox.me/FkUZynXERUb1TPJg0SIPj8zA4PZ3)

由于各个区域 (region) 和节点 (node) 之间的资源是相互独立的，用户需要通过切换区域来查看各个区域的资源配额和使用情况。

这些配额包括：
- 云主机：云主机数量、总 CPU 核数、总内存大小等
- 云硬盘：云硬盘数量，总容量、性能型云硬盘总容量和总个数、容量型云硬盘总容量和总个数等
- 安全组：安全组数量、安全组规则数量、SSH 密钥数量
- 网络：公网 IP 数量
- 快照：硬盘快照数量
- 负载均衡：负载均衡器数量

# 提升配额

如果用户在某一区域节点的配额不足，在创建新资源时系统会提示配额不足相关的错误信息，这时需要联系我们为您提升对应区域节点的资源配额。创建超出配额范围的云资源是不被允许的！
