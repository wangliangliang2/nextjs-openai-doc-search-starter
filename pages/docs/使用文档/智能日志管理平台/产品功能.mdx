 # **数据采集**

* 通过 logkit-pro 实时采集日志数据，支持上百种数据源（包括文件、MySQL、MSSQL、ElasticSearch、MongoDB、AWS CloudTrail 等）。

* 支持在 logkit-pro 采集端完成对数据格式的解析与字段转换。

* logkit-pro 采集的数据支持自动导出到智能日志分析平台。

* 支持七牛云产品（对象存储、CDN）、第三方云平台产品、本地 IDC、IOT 设备及各种上层服务应用的日志数据采集。

# **数据处理**

* 数据处理包括数据接收、计算、投递三个步骤。

* 支持直接将数据投递到日志分析平台。

* 支持通过流式计算或批量计算对数据计算后再投递到日志分析平台。

* 支持将数据投递到对象存储做低成本的持久化存储。

* 支持将数据投递到时序数据库。

* 支持将数据投递到 HTTP 地址。

# **实时数据接收与输出**

* 通过本地文件上传功能接入实时日志数据，帮助用户对已有存量数据快速接入。

* 支持对实时上传数据进行解析，抽取日志中的字段，支持包括 JSON 日志解析、CSV 日志解析、划词解析、键值对解析以及直接传入原始数据这几种解析方式。

* 与工作流管理服务（ pipeline） 对接，进行实时数据计算。

* 对实时日志进行滚动输出，毫秒级延迟支持用户第一时间获得信息。

# **日志查询与分析**

对日志内容进行查询分析，支持 Lucene、类 SQL 两种语法来进行日志搜索。

* 日志统计视图：支持日志搜索结果时序统计图。

* 划词过滤：支持对搜索结果划词过滤日志搜索结果，无需手动输入过滤条件。

* 字段统计分析：通过字段列表快速查看某字段的时序统计信息。

* 上下文查询：将某日志在原始文件中的前若干条（上文）或后若干条日志（下文）查找出来。

* 可视化：支持报表和仪表盘功能。

* 告警：配置告警指标来监控日志数据。

* 异常检测/预测：通过机器学习进行异常检测、预测和智能告警。

# **监控审计**

* 用户在智能日志管理平台的所有操作（创建、更新、删除）可查。

# **权限管理**

* 支持对日志服务和工作流服务授予其他账号部分操作权限。

# **对接第三方可视化工具**

* 支持 Grafana、Kibana 等第三方可视化工具。
