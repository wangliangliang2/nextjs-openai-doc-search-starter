Pandora v2.1 版本隆重推出[应用商店功能](https://portal.qiniu.com/insight/apps)，用户可根据私有化需求开通应用。

![](https://pandora-kibana.qiniu.com/appstore.png)


应用市场新增全链路追踪应用：追踪服务在执行过程中的具体信息，如查询的 sql，请求的 HTTP 地址，RPC 调用，开始、结束、间隔时间等，帮助拆分业务之间的调用关系，并提供可视化界面查看调用链详情。【[文档](/insight/manual/5054/all-link-track-profile)】

![](https://pandora-kibana.qiniu.com/trace1.png)

![](https://pandora-kibana.qiniu.com/trace2.png)

适用的场景包括但不限于以下几种：

* 监测 API 调用的耗时；

* 分析 url 调用过程中性能消耗原因，抓取超过阈值 url 的详细数据；

* 分析服务之间的依赖关系；

* 数据库场景下可以检查分析执行缓慢的 SQL 语句，辅助性能优化。
