**新功能**

* 仪表盘支持发送邮件报告。【[文档](/insight/manual/4867/the-dashboard#6)】

![](https://pandora-kibana.qiniu.com/dashboard_report.png)

* 仪表盘支持选择字段对数据进行下钻分析。【[文档](/insight/manual/4867/the-dashboard#5)】

![](https://pandora-kibana.qiniu.com/report/drill.png)

* 日志仓库告警规则支持条件对比告警：以第一条条件为基准，第二条条件与第一条条件进行对比，指标变化率告警阈值则触发告警。【[文档](/insight/manual/4631/warning#1)】

![](https://pandora-kibana.qiniu.com/alert/comparison1.png)

* 告警服务支持`钉钉`通知方式，可发送通知给钉钉群组、部门、个人。(仅在私有云支持)

![](https://pandora-kibana.qiniu.com/dingding.png)

* 告警事件支持删除。

* 日志告警规则支持导入 json 文件创建，支持将告警规则导出成 json 文件以便二次复用。

**功能优化**

* 告警服务删除需二次确认。

* 丰富告警事件详情。
