智能日志平台通过以下三种方式帮助用户快速进行日志搜索：

# **1.极速搜索**

进入[日志搜索](https://portal.qiniu.com/insight/logdb/overview)页面，找到`我的仓库列表`，在操作里选择`极速搜索`或者`详细搜索`即可快速查看默认时间范围下该仓库的全部日志信息。同时支持快速查看授权给您的仓库下的日志信息，点击`授权的仓库`即可。
![](https://pandora-kibana.qiniu.com/search/quick_search.png)

其中，点击`详细搜索`可以看到字段统计信息以及日志统计直方图。

![](https://pandora-kibana.qiniu.com/search/search_mode.png)

# **2.搜索历史**
在日志搜索界面找到`搜索历史`，通过搜索历史快速定位之前的搜索记录，点击搜索历史记录右侧的搜索按钮即可开始搜索，避免重复输入操作。

![](https://pandora-kibana.qiniu.com/search/history.png)

# **3.搜索结果分享**

您可以将已有搜索页面链接分享给其他人，对方可以基于您的页面直接查看搜索结果或者再搜索，不需要重复输入相关搜索条件，方便团队内部进行沟通，提高效率。
