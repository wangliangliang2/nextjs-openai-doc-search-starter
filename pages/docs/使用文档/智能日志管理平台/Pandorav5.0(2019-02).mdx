**新功能**

* 应用平台支持业务链路关联查询：通过配置业务链路中各系统的关联字段，串联业务链中各环节日志数据进行业务关联分析，以达到提升业务运维管理效能，驱动业务健康增长的目的。【[文档](/insight/manual/5696/business-link-related-query-is-introduced)】

![](https://pandora-kibana.qiniu.com/business_analysis/business_analysis_1.png)

![](https://pandora-kibana.qiniu.com/business_analysis/business_analysis_visual.png)


产品特性：

* 通过起始系统的搜索条件可查出业务链中全部系统的关联数据，直观显示故障模块。

* 查询业务链中全部系统的详细日志。

* 可视化呈现业务链路节点，还原业务办理路径。

* 跳转查看告警事件。
