迁徙图是在地图中动态、实时地展示数据的迁徙轨迹和迁徙量级，直观查看数据的来源和去向。常用于流量监控、安全攻击来源监控等场景。

**使用步骤**

1.在搜索栏输入搜索条件，点击右侧的查询按钮；

2.选择图表类型为`迁徙图`；

3.在`指标与分组`配置里配置分组字段与指标；

* 选择流出节点分组字段；
* 选择流入节点分组字段；
* 选择指标。

4.设置图表样式。

**气泡地图样式设置**

|配置项|说明|
|:-:|:-:|
|地图类型|支持中国地图迁徙图/世界地图迁徙图|
|展示指标|若在数据配置添加了多个计算指标，可以筛选展示在图中的指标|
|流入节点|可编辑流入节点名称及节点颜色|
|流出节点|可编辑流出节点名称及节点颜色|
|迁徙线颜色|选择迁徙线的配色方案|
|数值单位|切换数值显示单位|
|气泡最小直径|限制气泡显示的最小直径|
|气泡最大直径|限制气泡显示的最大直径|

示例：

在安全日志分析场景里，经常需要对全国范围内的web攻击、访问控制攻击等安全威胁进行展示。
例如，分析 waf 安全日志展示全国范围内 web 安全攻击地图，选择分组字段为：srcIp_City 和 destIp_City，指标为计数。
