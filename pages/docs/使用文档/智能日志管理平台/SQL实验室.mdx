**SQL 实验室**

通过 SQL 语句搜索数据立方实例里的实时数据，并以表格形式呈现搜索结果。

![](https://pandora-kibana.qiniu.com/datacube/cube_sql.png)

* 表格数据可下载。

![](https://pandora-kibana.qiniu.com/datacube/cube_sql_download.png)

* 左侧数据立方实例可展开查看立方里的字段，并通过右箭头直接插入 SQL 语句，方便查询。

![](https://pandora-kibana.qiniu.com/datacube/sqlField.png)

* 告警：对 SQL 查询结果配置告警，有且仅支持一个数字指标，如果返回多个，默认使用第一个字段值。

点击另存为告警，配置告警规则步骤请阅读：[数据立方 SQL 告警规则配置](/insight/manual/5454/configure-the-alarm-rules)

后续操作流程（告警服务管理、事件合并规则、通知策略管理、告警事件管理等）请阅读[告警平台相关文档](/insight/manual/5449/intelligent-alarm-system-is-introduced)。
