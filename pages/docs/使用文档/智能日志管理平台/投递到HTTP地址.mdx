工作流支持将数据投递到 HTTP 地址。

**填写参数说明**

|参数|必填|说明|
|:---|:---|:---|
| 名称 |是|导出节点名称|
| 服务器地址 |是|ip 或域名，例如:https://pipeline.qiniu.com 或 https://127.0.0.1:7758|
| 请求资源路径 |是|具体地址，例如: /test/repos|
| 导出类型 |是|导出的文件格式|
| 数据起始位置 |是|从最早还是最新数据开始导出，默认从最新数据导出|
