数据透视表（仅在私有云支持）是智能日志平台提供对日志数据通过可视化交互进行交叉分析及结果呈现的工具，通俗地说，数据透视表是一个基于日志仓库的 excel，通过您指定的各种筛选、关联、统计条件，自动为您呈现分析结果。
![](https://pandora-kibana.qiniu.com/search/pivot_1.png)

您只需要根据自己的分析需求对原始数据进行挑选、过滤、聚合、关联，定义数据透视表需要返回那些字段，数据透视表通过表格的形式显示您所关心的数据。这样即使没有技术背景您也可以直接在智能日志平台上定制自己的报表。

![](https://pandora-kibana.qiniu.com/pivot/pivot_2.png)

**产品特性：**

* 支持过滤、排序、关联、统计等操作对原始数据进行交叉分析。

* 进行交叉分析的每一步操作均可查、可改，操作极其灵活。

* 通过表格形式直观呈现数据透视表分析结果，一目了然。


**典型使用场景：**

* 电商网站业务数据分析：电商网站一次订单成交会有订单日志、买方信息日志，通过数据透视表对原始日志进行关联分析，可迅速找出促成订单的影响因素，以便平台调整运营策略促进营业额。

* 企业安全信息监控：企业通过各类系统来保证自身业务的健康运行，每个系统都记录了自身日志，如主机访问日志（记录了访问 IP 和用户名信息）、DHCP （Dynamic Host Configuration Protocol）日志（记录了访问 IP 和主机信息）、SCCM （System Center Configuration Manager）日志（记录了主机和访问用户信息），通过数据透视表对各系统的日志关联分析可迅速找出用户异常登录信息（主机访问日志中的 username 信息和 SCCM 日志中的 username 信息不一致），及时告警。
