智能日志平台全面升级，独立站点 https://portal.qiniu.com/insight ，使用全新设计的 UI 框架，功能模块更清晰，交互更易上手。

![](https://pandora-kibana.qiniu.com/dashboard/landing_page.png)

**新特性：**

1.增加落地页，帮助新用户了解智能日志平台，引导新用户使用产品功能。

2.使用全新设计的 UI 框架，统一各个功能的视觉呈现，界面更清爽简单，尊重用户体验，重新规划交互信息层级，保证产品的可用性及易用性。

![](https://pandora-kibana.qiniu.com/search/log_search.png)

3.功能模块划分更清晰，功能入口列在侧边栏。日志搜索模块下包括搜索、日志仓库管理、告警管理三个模块，实时日志模块下包括实时日志查看、实时仓库管理、解析规则管理三个模块，解决用户关于实时日志查看与日志搜索子功能关系不清晰的疑惑。

![](https://pandora-kibana.qiniu.com/search/sidebar.png)

侧边栏功能入口可收起，方便用户在功能界面查看更多内容。

![](https://pandora-kibana.qiniu.com/sidebar2.png)

4.增加使用教程入口，帮助新手快速了解智能日志平台重要功能的用法。

![](https://pandora-kibana.qiniu.com/release/tutorial.png)

![](https://pandora-kibana.qiniu.com/release/tutorial1.png)

5.`数据上传`独立入口，通过`数据上传`上传文件到实时仓库，同时导出到日志仓库，帮助用户快速体验智能日志管理平台的各种功能。【[文档](/insight/manual/4882/file-upload)】

![](http://p5ktjx46s.bkt.clouddn.com/release/upload.png)

6.支持到专业日志收集工具 logkit-pro 的入口，帮助用户一站式完成日志采集与日志分析。

![](http://p5ktjx46s.bkt.clouddn.com/release/tologkit.png)

7.日志搜索结果支持自定义每页展示条数：

![](https://pandora-kibana.qiniu.com/search/records.png)

8.支持在日志仓库管理直接对其他用户授予日志仓库的查看权限。权限管理详情请阅读[权限管理](/insight/manual/4736/authorization)。

![](https://pandora-kibana.qiniu.com/repo/authorize.png)

![](https://pandora-kibana.qiniu.com/repo/authorize1.png)

9.仪表盘的图表支持下钻。

在仪表盘点击任意感兴趣的数据，即可下钻进入日志详情页面。

![](https://pandora-kibana.qiniu.com/dashboard/xiazuan.png)

点击下钻以后：

![](https://pandora-kibana.qiniu.com/dashboard/xiazuan1.png)
