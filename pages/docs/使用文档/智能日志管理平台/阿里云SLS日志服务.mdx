**配置阿里云SLS的认证参数**
![](https://dn-odum9helk.qbox.me/FmHDmEYIrWdyPHDzlK8I52jAc8UO)

**具体参数说明如下**：
***ProjectName***: SLS服务的项目名称
***AK***: SLS账号的access key
***SK***: SLS账号的secret key
***LogStoreName***: SLS服务的store name
***Endpoint***: 阿里云服务节点, 可参考[阿里云endpoint列表](https://help.aliyun.com/document_detail/31837.html)

采集器会以单点消费的模式从SLS拉取日志
