`常用功能`方便您将常用的功能集中管理，并自定义您的专属日志平台首页。通过`常用功能`模块，您可以快速访问您常用的功能，提高工作效率。

![](https://pandora-kibana.qiniu.com/common_function.png)

支持常用功能的功能模块包括：

* 日志搜索
* 实时查看
* 数据透视
* 数据立方
* 仪表盘
* 应用商店的全部应用

常用功能使用步骤：

1.进入功能页面，以日志搜索为例，选择仓库、输入搜索条件，点击搜索按钮搜索日志结果。

2.点击`添加至常用功能`按钮，将当前页面添加至常用功能。

![](https://pandora-kibana.qiniu.com/add_to_common.png)

填写当前页面在常用功能模块的标题与描述，可设置是否显示为首页。

![](https://pandora-kibana.qiniu.com/add_to_common2.png)

3.在常用功能模块管理您添加的常用功能，
   * 可编辑、删除常用功能。
   * 修改常用功能基本信息。
   * 通过拖拽对常用功能进行排序。
   * 通过 url 直接添加页面至常用功能。

![](https://pandora-kibana.qiniu.com/common_fun_edit.png)
