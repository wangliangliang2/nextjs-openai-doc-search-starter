AWS 日志是一种自动解析 AWS 日志格式的解析方式,目前支持 s3 access log, elb access log, cloudfront access log
![](https://dn-odum9helk.qbox.me/FpexmYLOCgIstdjhF8uMrhFpwF-K)

* aws日志类型(`category`): 支持 s3 access log, elb access log, cloudfront access log

* 禁止记录解析失败数据(`disable_record_errdata`)：默认为 false，解析失败的数据会默认出现在`pandora_stash`字段，该选项可以禁止记录解析失败的数据。

* 保留原始数据（`keep_raw_data`):默认为`false`，配置此选项之后会保留原始数据在`raw_data`字段下。
