### **基础配置信息**

  * 数据库地址(`mysql_datasource`):
      - 明文填写：由 `username`: 用户名, `password`: 用户密码, `hostname`: mysql 地址, `port`: mysql 端口, `dbname`: 数据库名称组成， 一个完整 mysql_datasource 示例:`"root:123456@tcp(127.0.0.1:3306)/db1"`。 `！注意：请使用内网方式访问，以免出现安全隐患！` 
      - 环境变量填写：为了不在配置中明文存储，我们支持使用环境变量的方式填写该字段，填写方式为：`${YOUR_ENV}`，其中 `YOUR_ENV` 为环境变量，需要在机器上将该环境变量设置为原先要填写的值，`YOUR_ENV` 可以根据您的实际使用需求进行修改。
  * 数据库表名称(`mysql_table`)：必填项，表示目的数据库表名称。注意，发送之前数据表必须已存在。
  * 发送最大速率(`max_send_rate`)，可选项，默认值为`-1`，表示不限速，单位为条/秒，比如填写1000，则表示每秒限制发送1000条。
