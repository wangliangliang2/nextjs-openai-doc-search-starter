# 产品功能模块
**智能审核中心**
高效学习敏感内容，快速响应宣传提示、智能预警违规信息。相比全人工处理节省70%人力成本和90%的时间成本。

**智能编目平台**
图像识别添加人物、物体、场景标签。音频识别、自然语义识别提取音频中的重要信息。关键帧识别，实现镜头层精切。智能脚本，辅助片段层拆条。
-人物、物体、场景、智能识别，海量样本库，可靠准确率。
-音频智能识别，智能兼容多种语言、方言。
-自然语义识别，长文本分词并提取词频生成关键信息。
-关键帧识别，从镜头层标记片段出入点，可自动完成镜头层精切、编目。
-打点、精切应用，交互友好，操作便捷。
