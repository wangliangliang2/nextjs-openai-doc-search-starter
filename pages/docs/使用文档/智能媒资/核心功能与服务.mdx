# 核心功能与服务
**深度学习平台**
基于人工智能神经网络的学习和延展能力，提供一套完整的媒体信息数据模型和计算机视觉工具集。
-模型训练资源按需调度，优化成本。
-识别算法精准调优。
-数据集全网抓取。
-模型发布快速迭代。

**音视频处理平台**
针对海量多源多格式音视数据，提供零运维、高可用、高性能的编码、转码、识别、提取、标记等数据处理能力，日处理数超百亿次。
-支持人脸识别，集人脸检测、比对、聚类于一体。
-支持大量场景、物体识别，Top-5准确率达 90.07%。
-高效精准识别图片及视频中的违规内容，准确率高达 99.95%。
-转码支持所有主流音视频格式，同时支持截图、图片压缩等多种处理方式。

**微服务工作流**
业务主流程微服务化，服务之间松耦合，各类服务部署、运行、升级全独立；功能模块高度统一，界面风格、权限管理、安全策略、调度方式、日志管理及审计方法全统一。
-可定制的UI组件，确立视觉规范。
-基于运行数据的应用质量检测，防患于未然。
-智能日志管理，自动分析，降低用户心智负担。

**容器调度平台**
资源按需分配，快速调度。问题自动修复，确保服务不中断。内置应用市场，中间件一键部署。自动化构建与测试，秒级发布上线。开放第三方接口，平滑融入企业已有用户管理体系。
-容器调度，保证业务高并发时可用性。
-镜像仓库，提供高可用、高可靠、高吞吐的镜像仓库服务。
-应用编排，根据模版快速一个应用的所有服务。
