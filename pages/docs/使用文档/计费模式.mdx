# 计费类型
七牛云提供两种类型的云服务器购买方式：
包月和按需计费，分别适用于不同场景下的用户计算需求。

# 包月计费
	
包月计费是一种后付费模式，即先使用后付费。用户可以一次性选择购买一个月或者多个月的云计算资源。

** 包月周期：** 每一次进行包月操作会产生一个包月周期，这个周期的跨度时长就是用户的包月时长。如果当前时间资源不在一个包月周期内，则包月周期的开始时间是包月操作执行的时间；如果当前时间资源处于一个包月周期内，则包月周期的开始时间是当前包月周期的结束时间。用户可以对同一个资源进行多次包月操作，包月周期的开始时间和结束时间按包月操作的先后顺序依次往后推。

**扣费：** 包月模式会在包月周期内的第一个账单日一次性结算当前包月周期的所有费用。包月的资源到期后，计费模式会由包月计费方式转为按需计费。

** 优惠: ** 用户一次性对云主机包月超过12个月时，会额外赠送两个月的包月时长

**适用场景：** 包月模式适用于在中长期范围内对云资源有稳定需求的场景，如成熟业务、作业系统等。包月购买的云资源在到期前不支持调整其规格 (虚拟主机) 或大小 (云硬盘和带宽)，在价格方面包月计费的价格相比按需购买要更加优惠。

# 按需计费

按需计费是一种后付费模式，即先使用后付费。用户可以动态的创建和释放资源，计费时只计算资源实际使用的时长。用户也可以弹性的增加和缩小主机的规格和网络的带宽（云磁盘只能扩容不能缩小），计费时根据具体规格的使用时长进行收费。

** 扣费：** 按需计费会在每月的账单日结算上月按需使用资源的费用。按需计费也可以随时变更为包月计费。

** 适用场景：**适用于对云资源需求不确定的场景，比如测试，新的业务线，电商等等。资源可以动态的创建和释放，也可以根据具体的业务量来调整规格，从而降低用户的支出，并且可以应对更多的不确定性场景。
    
# 具体对比

| **模式** | **包月模式** | **按量计费** |
| --- | --- | --- |
| 付费方式 | 后付费 | 后付费 |
| 计费单位 | 元/月  账单日一次性结算 | 元/秒  按秒计费，按月结算，随时购买随时释放 |
| 计费方式变更 | 到期自动变为按量计费 | 随时可以修改成包月计费 |
| 产品价格 | 相对便宜 | 相对包月模式略贵 |
| 是否可删除 | 资源到期前，不支持 | 随时可释放 |
| 调整资源规格 | 不支持动态调整，到期后可调整 | 随时弹性调整配置 |
| 使用场景 | 适用于对资源需求量长期稳定的成熟业务 | 适用于新的业务线，电商等对具体资源使用需求不明确的场景 |


**【注意事项】**

1. 包月模式的资源不可修改规格和释放

2. 包月的资源到期后计费方式自动转为按需计费

3. 已经包月计费的资源可以在其基础上再进行多次包月操作，后续的包月操作会产生一个额外包月周期；这个包月周期的开始时间是上一个包月周期的结束时间。

4. 调整按需使用资源的规格会产生多段按需计费的时间段，这些时间段内分别按其具体的规格配置进行按需计费

5. 包月周期内的首个账单日会对当前的包月周期进行结算，后续月份的账单里会体现包月的周期但是用户的费用为零。例：用户在8月25号以包月的方式购买了6个月的云主机；假设账单日为每月5号，那么用户会在9月5号收到云主机包月6个月的账单;接下来10月份的出账日，用户的账单里会体现该云主机的包月周期但没有费用
