# **功能丰富**
* 实时直播：以稳定且具成本优势的P2P技术快速连接联提供直播服务，同时还支持rtmp协议来实现实时直播。
* 视频回放：查看过去一段时间内的历史视频，不遗失任何精彩的片段或线索。
* 视频检索：检索重要的视频片段。
* 视频收藏：将收藏的重要片段，保存在云存储。
* 视频画面分享：随时随地分享生活中的精彩画面。
* 视频管理：有效合理的管理视频文件的增删改。
* 云存储：按需把视频上传至云存储。
* 云台控制：通过移动端控制智能摄像头的画面选取角度。
* 画面切换：通过移动端随时切换视频监控画面。
* 事件消息提醒：异常事件监测，可以将家和企业里的异常事件第一时间发送到用户的手机上，免除牵挂。

# **集中管理**
支持应用管理、设备管理、权限设置、存储管理、视频管理，大大提高了对于产品的管理、调度、协调能力。
* 应用管理：设备的集合，针对不同的产品业务可以创建多个应用进行业务隔离。
* 设备管理：提供设备级的配置管理及数据统计。
* 权限设置：基于IAM的权限控制，灵活、自由的配置权限规则，控制子账号对资源的操作权限；有效分组，还可以对设备权限进行控制以及客户端访问设定边界的管理。
* 存储管理：丰富的存储配置可以高效、合理使用云存储。
* 视频管理：对视频文件进行有效管理，按需进行检索查询回放。

# **高效、稳定、安全的云存储服务**
七牛云存储系统（QOS）是七牛团队完全自主研发的一套高性能海量数据存储系统，解决了传统存储方案和其他分布式文件系统所存在的大多数技术问题。七牛云存储系统采用了基于EC进行大幅优化的QEC技术，设计为运行于普通服务器设备上，最大化系统的读写性能，同时具备接近于无限的扩容能力，通信加密，及时脱敏，边缘存储数据具备网银级安全，更可靠，更安全。
