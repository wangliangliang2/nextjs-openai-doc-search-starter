本文汇总了各类型网站的部署方式，便于您自助搭建网站和应用方式汇总。

| 网站类型 | 部署方式 | 说明 |
|-|-|-|
| WordPress | [手动搭建WordPress（CentOS 7）](http://) | WordPress是一款常用的搭建个人博客网站的软件。您可以使用WordPress架设自己的网站，也可以搭建内容管理系统CMS（Content Management System）。您也可以手动搭建WordPress。 |
| LNMP环境 | [手动部署（CentOS7）](https://developer.qiniu.com/qvm/manual/7073/manual-deployment-lnmp-environment-centos-7) | LNMP分别代表Linux、Nginx、MySQL和PHP。使用镜像和资源编排服务ROS模板可以帮您省时省力地部署LNMP环境。如果您熟悉Linux操作系统，希望满足个性化部署的要求，您也可以选择手动部署。 |
| Java Web环境 | 手动部署Java Web环境（CentOS 7） | 如果您熟悉Linux命令，可以在QVM上个性化地部署Java web项目。 |
| Node.js | [部署Node.js环境（CentOS 7）](https://developer.qiniu.com/qvm/manual/7075/the-deployment-of-the-node-js-environment-centos-7) | Node.js是一个基于Chrome V8引擎的JavaScript运行环境，用来方便、快速地搭建易于扩展的网络应用。<br>Node.js使用了一个事件驱动、非阻塞式I/O的模型，轻量又高效，非常适合在分布式设备运行的数据密集型实时应用。Node.js的包管理器npm，是全球最大的开源库生态系统。 |
| GitLab | [手动部署](https://developer.qiniu.com/qvm/manual/7104/install-and-use-gitlab) | GitLab通过Ruby on Rails，实现自托管的Git项目仓库，可通过Web界面轻松访问公开或者私人项目。 |
| Drupal内容管理框架 | [手动部署](https://developer.qiniu.com/qvm/manual/7101/manually-set-the-drupal-site) | Drupal是一款采用PHP语言编写的开源内容管理框架（CMF），由内容管理系统（CMS）和PHP开发框架（Framework）共同构成。<br>如果您熟悉QVM、 Linux系统，刚开始使用QVM实例建站，可以使用镜像快速搭建Drupal环境。 |
| LAMP环境 | [手动部署](https://developer.qiniu.com/qvm/manual/7074/manual-deployment-lamp-environment-centos-7) | LAMP指Linux、Apache、MySQL和PHP，是一组常用来搭建动态网站或者服务器的开源软件。它们本身都是各自独立的程序，但是因为常被放在一起使用，拥有了越来越高的兼容度，共同组成了一个强大的Web应用程序平台。 |
| 手动部署 MySQL 数据库 | [手动部署](https://developer.qiniu.com/qvm/manual/7121/manual-deployment-mysql-database-centos-7) | 手动部署 MySQL 数据库。 |
| RabbitMQ | [手动部署 ](https://developer.qiniu.com/qvm/manual/7123/manual-deployment-of-the-rabbitmq-centos-7)| RabbitMQ是AMQP的一个开源实现，支持：Python、Ruby、.NET、Java、JMS、C、PHP、ActionScript、XMPP、STOMP、AJAX等多种客户端。<br>用于在分布式系统中存储转发消息，具有较好的易用性、扩展性和高可用性。 |
| PostgreSQL主从架构 | [手动部署](https://developer.qiniu.com/qvm/manual/7122/manually-set-up-postgresql-master-slave-architecture-centos-7) | 七牛云版数据库PostgreSQL具有NoSQL兼容、高效查询、插件化管理、安全稳定的特性。<br>如果您熟悉QVM、Linux系统、PostgreSQL，可以手动部署PostgreSQL主从架构。 |
| Ghost博客（CentOS 7） | [手动部署](https://developer.qiniu.com/qvm/manual/7103/manually-set-up-ghost-blog-centos-7) | Ghost是一个免费的开源博客平台，使用JavaScript编写，基于Node.js，旨在简化个人博客和在线出版物的在线发布过程。此外，将来随着业务的扩展，您可以利用七牛云强大的产品平台，平滑地横向和纵向扩展服务容量。 |
| FTP站点 | [手动部署（Linux）](https://developer.qiniu.com/qvm/manual/7105/manually-set-up-ftp-site-centos-7) | 在QVM实例上安装vsftpd，它是Linux下的一款小巧轻快、安全易用的FTP服务器软件。 |
