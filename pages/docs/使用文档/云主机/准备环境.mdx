# 操作步骤

1. 通过下面两种方式可以引入依赖（任选一种）：
 - Maven 方式引入依赖：
 ```
<dependency>
   <groupId>com.aliyun.openservices</groupId>
   <artifactId>ons-client</artifactId>
   <version>1.8.0.Final</version>
	  //设置为 Java SDK 的最新版本号
</dependency>
```
 - 下载依赖 JAR 包：
Java SDK 下载链接：请参见 Java SDK 的版本说明。

2. 代码里涉及到的资源信息，例如 Topic 和 Group ID 等，需要到控制台上创建。
