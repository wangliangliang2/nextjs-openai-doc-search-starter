云数据库MongoDB支持申请公网连接地址，用于通过公网连接实例。
<a id="step05"></a>
## **背景信息**
<a id="step05"></a>
您可以通过控制台申请公网连接地址。通过公网连接存在一定安全风险，为保障数据安全性，若您不需要公网连接地址，可以释放公网连接地址。
<a id="step05"></a>
## **操作步骤**
<a id="step05"></a>
1. 登录[MongoDB管理控制台](https://portal.qiniu.com/qvm/db/mongo)。

2. 在控制台左上方选择地域。

3. 单击目标实例ID或目标实例右侧的管理。

4. 在基本信息页面。

5. 单击公网连接栏目右侧的申请公网连接地址。

6. 在弹出的申请公网地址对话框，单击「确定」。

<table><tr><td bgcolor=#F0F8FF>**说明**：<font color='gray'> 申请步骤操作完成后，需要通过申请的公网连接地址来访问该实例时，请将您连接实例终端所在的公网IP地址加入到白名单中。 </font></td></tr></table>
<a id="step05"></a>
## **执行结果**
<a id="step05"></a>
申请操作完成后，MongoDB副本集会同时为用户提供两个公网地址：Primary节点和Secondary节点。
