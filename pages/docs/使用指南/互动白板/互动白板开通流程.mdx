# 1. 注册账号
七牛云官网，前往[注册页面](https://portal.qiniu.com/signup)

# 2. 实名认证
前往七牛实名认证页面，进行[实名认证](https://portal.qiniu.com/identity/choice)
![1.png](https://dn-odum9helk.qbox.me/FmKvGHbYNHxfklUQe1ctCAC-Xg88)

# 3. 创建RTC应用
在实时音视频业务板块中[创建应用](https://portal.qiniu.com/rtn/app/create)
![2.png](https://dn-odum9helk.qbox.me/Fta6jTuzVrOC18f7QAyC4Lc15TMf)

# 4. 开通互动白板服务
创建RTC应用完成，查看应用详情页面，找到互动白板服务模块，点击开启，如下图所示
![3.png](https://dn-odum9helk.qbox.me/FkZ4j3CjMaz0dzoJs0XBMasb1x9E)

# 5. 接入使用
互动白板服务开启后即可接入使用，更多详情可咨询七牛商务人员。
