#  <span id="tutorial">创建直播流</span>
1. 登录直播云控制台
2. 点击需要查询的**直播空间名称**
3. 点击直播流管理 > **添加流**，进入创建直播流界面

<img src="https://dn-odum9helk.qbox.me/FpQIooZfYCNM-VI2UdO6D1nFL44A" width = "90%" height = "90%" align=center />

4. 输入你要创建的**直播流名称**，点击确定创建

<img src="https://dn-odum9helk.qbox.me/FhLr0D4i56sTAXjcDjR2kVWzlpdh" width = "90%" height = "90%" align=center />

5. 点击直播流 > **流属性**，查看当前直播流的详细信息，获取直播推流地址

<img src="https://dn-odum9helk.qbox.me/FpVLSVwOi1CmQtrV8XAh6g5cbNfp" width = "90%" height = "90%" align=center />

6. 点击直播流 > **实时直播**，查看当前直播流的详细信息，获取直播播放地址

<img src="https://dn-odum9helk.qbox.me/FrrjKJf4fB_wLlVmsXahRpI-jZqm" width = "90%" height = "90%" align=center />
#  <span id="tutorial">禁播直播流</span>
1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击直播流管理 > 活跃流
4. 选择需要禁播的流名称，点击流属性
5. 在流属性中可以对直播流进行禁播处理，选择**永久禁播**或**限时禁播**及禁播时长，点击保存设置后，推流中的直播将被断开，该地址无法进行推流

<img src="https://dn-odum9helk.qbox.me/FsKg8dm0oVLABUu5ZB925XB4d9-m" width = "90%" height = "90%" align=center />

6. 如果需要继续进行推流操作，则点击 关闭禁播，保存设置后需要重新进行推流操作


#  <span id="tutorial">查看直播实时信息</span>
实时直播中主要显示的是活跃流的信息和状态

1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击 直播流管理 > 活跃流
4. 选择需要查看的流名称
5. 在**实时直播**可以查询实时推流信息，包括推流市场，主播IP地址，上线时间，实时观看人数，

<img src="https://dn-odum9helk.qbox.me/Fn6Fi8LBPnvzdWLW89-wWhGd9EyS" width = "90%" height = "90%" align=center />

4. 提供**帧率**和**码率**的实时报表的查询，便于及时查看直播流的上行传输状态，保证直播观看的体验，帮助排查直播卡顿的原因。当数据显示状态为平滑，且波峰、波谷较稳定时，表示上行传输较稳定。如果出现较大的抖动，则需要及时排查上行传输出现的问题

**帧率实时报表**
帧率是每秒钟播放的画面帧数，帧率越高越稳定画面越流畅

<img src="https://dn-odum9helk.qbox.me/FtD09FjDad9PbJ08eH0JR398IwDm" width = "90%" height = "90%" align=center />


**码率实时报表**
码率是码率决定的是画面的质量。相同帧率下，码率越高换面越清晰

<img src="https://dn-odum9helk.qbox.me/FpSUNhyFE1qHoLNEzafMrZGxe9nW" width = "90%" height = "90%" align=center />


#  <span id="tutorial">直播记录</span>
直播记录统计了这段直播流所有的推流开始的时间，断流的时间以及推流时间长度，可以查看每次推流的帧率及预览画面
1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击 直播流管理 > 所有流
4. 选择需要查看的流名称

<img src="https://dn-odum9helk.qbox.me/FmBIRnN5MAzVsKAuWbQhVtGa5EE5" width = "90%" height = "90%" align=center />

#  <span id="tutorial">查看直播流帧率码率</span>
推流历史信息主要是方便用户查询一定时间段内的帧率和码率的变化曲线。点击红框内的日期，可以查询指定时间段中的流信息变化
查询当前推流中的直播流可直接在**直播实时信息**中查询
1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击 直播流管理 > 所有流
4. 选择需要查看的流名称
