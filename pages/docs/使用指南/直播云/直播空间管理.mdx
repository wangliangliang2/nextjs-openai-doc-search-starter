# <span id="tutorial">创建直播空间</span>
1. 在控制台首页定位到七牛官方产品 > 直播云服务 卡片，点击**立即添加**，可以创建直播空间页面；或进入直播云服务页面后，点击左上角新建直播空间

<img src="https://dn-odum9helk.qbox.me/Fugtq8wcp3Meqn2IGOfznJ3Zm0eF" width = "90%" height = "90%" align=center />

2. 填写直播空间基本信息，分别输入**直播空间名称**和**直播域名**，系统会自动为您生成直播服务使用的推流域名及播放域名

<img src="https://dn-odum9helk.qbox.me/FuLyP417Oh476qp8W1wQUXZqRNuQ" width = "90%" height = "90%" align=center />

3. 点击下一步，进入**存储空间**配置页，进行存储空间及存储过期时间配置；若没有存储空间，点击“新建存储空间”进行存储空间的设置
  * 存储空间是用于存放直播的数据，价格独立计算。
  * 被存在对象存储空间中的数据可以用于点播和回放。
  * 国家规定存储周期不得少于60天，过期内容会被自动清空。
  
<img src="https://dn-odum9helk.qbox.me/FhNtgN4vi0p2Nnnz8N_PiemI2X9v" width = "90%" height = "90%" align=center />

4. 点击下一步，进入**高级设置**，配置直播封面、直播状态通知、直播鉴权等，完成后即可创建直播空间

<img src="https://dn-odum9helk.qbox.me/Fhuym3wDld4FZka_TqbzvEcsLJ2u" width = "90%" height = "90%" align=center />

# <span id="tutorial1">直播空间设置</span>

1. 点击左侧列表的**直播空间名称**，进入直播空间的控制面板。
2. 点击右下角直播空间设置按钮或切换导航栏至直播空间设置，进入直播空间设置页面

<img src="https://dn-odum9helk.qbox.me/FjpMXiqrJL80iZrxez-WnHcHXNuB" width = "90%" height = "90%" align=center />


3. 直播空间的可设置字段包括：
  * **存储过期时间**：国家规定存储周期不得少于60天，超过过期时间的直播内容将被自动清空
  * **自动创建流**：开启后，当推一路之前没有创建过的流时，这路流会被自动创建。
  * **自动删除过期流**：开启后，在一定时间（同 存储过期时间 ）内没有任何活动的部分流，将被自动删除（仅针对开启该功能后新创建的流生效）。
  * **HLS低延迟**：开启 HLS 低延迟，可以极大降低首开时间和延迟，适合对延迟敏感的用户
  * **直播封面**：开启直播封面，系统会根据间隔时间对直播内容自动截图
  * **直播中断超时**：直播推流中断时，服务端将会重试连接，超过此时间后返回推流超时状态。若一条流断流后进行重推，中间断流时间若小于直播中断超时则不认为发生断流，也不会触发断流回调
  * **直播鉴权**：设置直播鉴权方式，对推流地址进行授权认证，防止非法推流，避免直播内容的盗用。
        * 限时鉴权是指：客户通过限时鉴权获取的推流地址存在一个有效期，过了这个有效期该地址就不可以推流。
        * 无校验鉴权：客户通过无校验鉴权获取的推流地址不会过期，只要还有流量就可以一直推流
  * **直播鉴黄**：直播鉴黄会定期对直播画面截帧进行鉴黄，若鉴黄结果超过阈值，将通过回调接口进行通知

# <span id="tutorial2">删除直播空间</span>
删除直播空间的动作对正在进行直播业务影响较大且不可逆，请谨慎操作

1. 点击左侧列表的**直播空间名称**，进入直播空间的控制面板。
2. 点击右下角直播空间**设置**按钮或切换导航栏至直播空间设置，进入直播空间设置页面。最底部有删除直播空间选项
3. 点击**删除**，在新页面的输入框中填写你要删除空间的名字。
4. 点击确认删除后，输入登录密码，完成彻底删除。
