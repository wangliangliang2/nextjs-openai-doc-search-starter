数据统计包括对计费项带宽、流量的统计，转码以及鉴黄用量的统计以及对直播的运营分析，可根据域名、区域、时间、省份、运营商等不同的维度进行查询，支持相关数据的导出以便进行核对
**操作路径：**
1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击 数据统计 > 所有流
4. 选择需要进行录制的流名称

# 带宽和流量统计
带宽和流量统计，分别提供主播推流占用的上行带宽/流量，用户播放内容占用的下行带宽/流量，上下行总 带宽/流量，提供按域名、区域、时间进行查询
提供**日带宽峰值表**，按日统计上行带宽、下行带宽及总带宽的峰值时间（以分钟为单位）及峰值数值
提供**日总流量表**，按日统计上行总流量、下行总流量及上下行总流量

# 转码用量统计
开启直播转码后，可以在数据统计 > 转码中查询不同转码格式（标清、高清、超清）的转码时长，以日为统计粒度。 

<img src="https://dn-odum9helk.qbox.me/Fr5glh7bu7H553qLwywPORbG27rw" width = "90%" height = "90%" align=center />

# 鉴黄用量统计
开启直播鉴黄后，可以在数据统计 > 鉴黄中查询**确认鉴黄次数**以及**非确认鉴黄次数**，以日为统计粒度。

# 运营分析
