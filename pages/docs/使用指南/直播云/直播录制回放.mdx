录制回放功能可以将直播的内容录制下来存到七牛的对象存储空间里，便于配合政府部门的**监管**和**点播回放**的使用。
支持录制为M3u8，MP4，FLV 三种格式，录制为 MP4/FLV 格式需要使用多媒体处理转码服务。多媒体处理服务独立计费，使用会额外产生费用。

# 实时直播录制
实时直播录制可以在直播的同时录制内容，录制的视频将永久存储于**对象存储**中。 使用直播录制功能，需要先在直播空间设置中配置好**点播域名**。
1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击 直播流管理 > 活跃流
4. 选择需要进行录制的流名称
5. 在**实时直播**中，选择需要录制的格式

# 历史直播录制
历史直播录制可将该直播流一段时间内的直播内容**生成回放**，方便用户对热点内容进行点播
1. 登录直播云控制台
2. 点击 需要查询的直播空间名称
3. 点击 直播流管理 > 所有流
4. 选择需要进行录制的流名称
5. 在直播记录中，筛选时间选择需要录制的时间，点击生成
6. 输入录制的文件名称，时间范围以及生成格式
