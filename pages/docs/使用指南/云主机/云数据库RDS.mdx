# 简介
七牛云关系型数据库RDS（Relational Database Service）是一种稳定可靠、可弹性伸缩的在线数据库服务。RDS支持MySQL、SQL Server、PostgreSQL，并且提供了容灾、备份、恢复、监控、迁移等方面的全套解决方案，彻底解决数据库运维的烦恼。

# 云数据库 RDS 的优点
选择云数据库RDS，您可以快速搭建稳定可靠的数据库服务，相比自建数据库有如下优势：
- 便宜易用，具有灵活计费、按需变配、即开即用等优点。
- 高性能，包括参数优化、SQL优化建议等。
- 高可用架构和多种容灾方案。
- 高安全性，提供多种安全措施保障数据安全。
- 在性价比、可用性、可靠性、易用性、性能等方面，云数据库RDS都有很大优势，价格相比 QVM 自建数据库，仅需约1/3，相比自购服务器搭建数据库，仅需约1/10。
