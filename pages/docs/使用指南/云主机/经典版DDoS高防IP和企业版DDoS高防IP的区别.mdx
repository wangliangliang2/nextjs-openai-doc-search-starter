# 经典版 DDoS 高防 IP 和 企业版 DDoS 高防 IP 的区别
## **防护攻击类型**
|防护分类|描述|经典版 DDoS 高防 IP| 企业版 DDoS 高防 IP |
|-|-|-|-|
|畸形报文过滤|过滤 frag flood，smurf，stream flood，land flood 攻击，过滤 IP 畸形包、TCP 畸形包、UDP 畸形包|✓|✓|
|网络层 DDoS 攻击防护|过滤 UDP Flood、SYN Flood、TCP Flood、ICMP Flood、ACK Flood、FIN Flood、RST Flood、DNS/NTP/SSDP 等反射攻击、空连接|✓|✓|
|应用层 DDoS 攻击防护|过滤 CC 攻击和 HTTP 慢速攻击，支持 HTTP 自定义特征过滤如 host 过滤、user-agent 过滤、referer 过滤|✓|✓|

## **参数对比**
|参数|经典版 DDoS 高防 IP|企业版 DDoS 高防 IP|
|-|-|-|
|保底防护带宽|20 Gb～300 Gb|30 Gb～600 Gb|
|弹性防护带宽|0 ～600 Gb|30 Gb～600 Gb|
|业务带宽|100Mbps～2000Mbps|100Mbps～5000Mbps|
|防护域名数|60个 - 500个|50~200个|
|业务QPS|10,000～200,000|3000~100,000|
|端口数|60个 - 500个|50~400个|
|源站 IP 数|20个/单个实例|20个/单个实例|
|每秒新建连接数|50000个(单一源IP）|1~50000个(单一源IP）|
|并发连接数|200,000个(单一源IP）|1~50000个(单一源IP）|
|默认黑洞自动解除时间|2个小时|2.5小时|
|每天可手动解除黑洞次数|3次|5次|

## **收费模式对比**
|产品|经典版 DDoS 高防 IP|企业版 DDoS 高防 IP|
|-|-|-|
|保底防护费用|包年包月|包年包月|
|弹性防护带宽|按天按量计费|按天按量计费|
|转发规则数（端口/域名）|包年包月按个数计费|包年包月按个数计费|
|业务流量费用|包年包月按带宽计费|包年包月按带宽计费|
|业务 QPS|无单独扩展|扩展按 QPS 数量收费|
