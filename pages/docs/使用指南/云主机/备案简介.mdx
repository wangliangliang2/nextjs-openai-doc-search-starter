# **为什么要备案** 

根据工信部《互联网信息服务管理办法》（国务院 292 号令），网站在未完成备案之前，不能指向大陆境内服务器开通访问。如果你的网站托管在中国大陆节点服务器，就必须申请互联网内容提供商 (ICP) 备案。若网站服务器为非中国大陆节点，则不用申请备案。

对于单纯的资讯性并且不涉及直销的网站，需要申请 ICP 备案。ICP 备案可通过网站托管服务器提供商或网络空间服务商（如电信、移动、联通等）进行申请。七牛云 ICP 备案操作可分为：首次备案、接入备案、变更备案、新增网站备案（原备案不在七牛云） 和 增加网站备案（原备案在七牛云）。

你可以在服务器提供商的备案系统提交 ICP 备案申请。如果你使用的是[七牛云主机](https://portal.qiniu.com/qvm/)，请登录[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)提交备案申请。具体步骤，请参考[首次备案指南](/qvm/manual/qvm-first-guide)。

注意：根据 2016 年最新要求，网站在工信部备案成功后，需在网站开通之日起 30 日内提交[公安备案](/qvm/manual/qvm-public-security)申请。

对于商品或服务的在线平台或第三方卖方，需获得商用许可证。要申请经营性 ICP 许可证，你必须先有一个有效的 ICP 备案号。经营性备案需用户自己到当地管理局相关部门进行办理，服务商可协助提供资质以及接入协议。具体办理要求需你登录当地经营性备案网站进行查看。

# **备案准备**

为节约备案时间，建议你提前做好备案准备。因各省管局要求不同，准备资料也有不同，请你访问[工信部备案管理系统](https://beian.miit.gov.cn)了解备案省市的备案规则准备备案资料。

**备案必需：**

- **域名**

 备案域名需为**有效期内、工信部已批复的且已完成实名认证**的域名。你可以在域名注册商处购买域名。 
自 2018 年 1 月 1 日起，从事互联网信息使用的域名应为其依法依规注册所有，个人性质备案域名注册者应为本人，单位性质备案域名注册者应为单位（含公司股东）、单位主要负责人或高级管理人员。具体要求，请查看[工业和信息化部关于规范互联网信息服务使用域名的通知](http://www.miit.gov.cn/newweb/n1146285/n1146352/n3054355/n3057709/n3057714/c5930543/content.html?spm=a2c4g.11186623.2.3.mFjsgs)。备案时会验证域名所有者和备案主办单位名称，网站域名所有者与网站备案主办者需一致。如果你的域名所有者与主办单位不一致，请先更改域名所有者信息。

- **云主机**

	通过七牛云备案需要有七牛云中国大陆节点云主机。**目前可备案的服务器必须是预付费包月 3 个月及以上（含续费）的云主机，且必须有公网带宽（如果是资源池一的 IP 那么付费类型为按量付费或者包年包月都可以，如果是资源池二的 IP ，那么付费类型必须为包年包月）**。你可根据你的网站需求选择购买服务器，服务器地域需选择中国大陆地域。如果您备案时，您的 ECS 实例和弹性 Web 托管实例的包年包月剩余时长不满足备案要求，你可以续费。如果你有七牛云主机与备案相关问题，请联系我们的技术支持。

- **证件资料**

 - 个人备案：需要个人身份证正反面原件扫描件或照片。
 - 企业备案：需要企业注册证件原件扫描件、备案负责人身份证正反面原件扫描件或照片。
 - 其他资料：当地管局备案规则中要求的其他证明材料，如域名证书、法人委托授权书等。

# **备案导航**

你可以在[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)进行以下非经营性备案操作：

- **[首次备案](/qvm/manual/qvm-first-guide)**

 主办单位或个人证件（即备案主体）及网站域名都未进行过工信部备案，且使用中国大陆境内节点七牛云主机托管网站，需通过七牛云合作备案系统进行工信部备案。

- **[变更备案](/qvm/manual/qvm-change-record)**

 你通过七牛云合作备案系统成功备案后，如需变更备案主体信息，如：主办单位名称、证件号码等或网站信息，如：增加、删除域名等，需通过变更主体或变更网站信息来实现。

- **[接入备案](/qvm/manual/qvm-access-registration)**

 如果你的网站域名已通过其它服务商成功备案，现将网站域名解析设置指向中国大陆境内节点七牛云主机，则需要你在七牛云合作备案系统进行接入备案。接入备案分为：[主体不在七牛云接入备案](/qvm/manual/qvm-access-registration#access1)和[主体在七牛云接入备案](/qvm/manual/qvm-access-registration#access2)

- **[增加备案(已是七牛云备案用户)](/qvm/manual/qvm-record-users)**

 主办单位或个人证件（即备案主体）已通过七牛云合作备案系统有过成功的备案，现有新的网站托管到中国大陆境内节点七牛云主机上，需要新增网站备案。


- **[新增备案(原备案不在七牛云)](/qvm/manual/qvm-new-site)**

 主办单位或个人证件（即备案主体）已通过其他服务商进行过工信部备案，现有中国大陆境内节点七牛云主机托管的新网站还未备案，需在七牛云合作备案系统提交新的网站域名备案申请，可按照原备案不在七牛云进行新增网站备案。

- **[注销备案](/qvm/manual/qvm-delete-registration)**

 如你通过七牛云备案的网站已不再使用，你需注销备案（主体/网站）或取消接入。如你是要注销某个网站的多个域名中的一个域名，请勿注销网站，而是通过变更网站信息删除该域名，管局审核通过后，该域名备案删除。

此外，[公安备案](/qvm/manual/qvm-public-security)，依据《计算机信息网络国际联网安全保护管理办法》相关规定，各网站在工信部备案成功后，需在网站开通之日起 30 日内登录[全国公安机关互联网站安全管理服务平台](http://www.beian.gov.cn/portal/index)提交公安备案申请。
