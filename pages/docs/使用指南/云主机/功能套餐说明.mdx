#### 当前企业版 DDoS 的版本只有专业版，其中套餐类型包括标准套餐和增强套餐，增强套餐在标准套餐的基础上，额外提供非标准业务端口等增强功能，增强企业版 DDoS 高防 IP 的业务接入能力。
#### 购买套餐时默认选择标准套餐，用户可以根据更高的业务需求选择增强套餐，也可以在购买标准套餐后再升级为增强套餐，但不支持在购买增强套餐后降级为标准套餐。

| 功能项                                     | 功能描述                                                             | 标准套餐 | 增强套餐 |
|-----------------------------------------|------------------------------------------------------------------|------|------|
| **流量型攻击防护**                                 | 支持常见的流量型DDoS攻击防护，包括畸形报文攻击防护和各类流量型Flood攻击防护。                      | √    | √    |
| **资源耗尽型防攻击防护**                              | 支持常见的网络四层/七层资源耗尽型CC攻击防护，例如HTTP GET Flood、HTTP POST Flood攻击等。     | √    | √    |
| **黑白名单**                                    | 针对每个接入防护的域名业务支持最多200条访问IP白名单和200条访问IP黑名单规则配置。                    | √    | √    |
| **CC策略**                                    | 提供四种防护模式，还可以针对具体页面设置自定义CC规则                                      | √    | √    |
| **DDoS防护策略**                                | 针对IP的制定端口设置虚假源、空连接和连接限速等防护规则                                     | √    | √    |
| **健康检查**                                    | 包括四层检查和七层检查，它们可以通过设定健康阈值进行定期检查服务状态，七层检查还可以设定具体的页面发起HTTP HEAD请求检查 | √    | √    |
| **HTTP\(80/8080\)、HTTPS\(443/8443\)标准端口转发** | 支持HTTP（80/8080）、HTTPS（443/8443）业务的DDoS攻击防护。                      | √    | √    |
| **HTTP、HTTPS非标准端口转发**                       | 支持HTTP、HTTPS非标准端口（不限于80、8080、443、8443端口）业务的DDoS攻击防护。             | x    | √    |
