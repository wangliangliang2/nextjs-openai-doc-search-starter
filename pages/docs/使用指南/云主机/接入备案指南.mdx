如果你的网站域名已在其他服务商备案，现要将域名及二级域名或其他子域名解析到七牛云主机，使用七牛云主机来托管你的网站，则需要你在七牛云进行接入备案。如果你将网站迁移至其他提供商的服务器后，你需[取消接入备案](#access3)。

接入备案分为：
- [主体不在七牛云接入备案](#access1)

- [主体在七牛云接入备案](#access2)

<br>

### **接入备案准备**

- 请访问[工信部备案管理系统](http://beian.miit.gov.cn)详细了解相关省市的备案规则。

- **注册域名** 备案域名需是**有效期内、工信部已批复的、且已完成实名认证**的域名。
    
- **购买七牛云主机** 通过七牛云备案需要有七牛云中国大陆节点云主机。你可根据你的网站需求选择购买云主机，云主机地域需选择中国大陆地域，包月 3 个月以上（含续费）且有公网带宽（如果是资源池一的 IP 那么付费类型为按量付费或者包年包月都可以，如果是资源池二的 IP ，那么付费类型必须为包年包月）。如果你备案时，你的云主机 QVM 实例的包年包月剩余时长不满足备案要求，你可以续费。
    
- **电子版本证件资料** 

 个人备案：请提前准备好个人证件（例如身份证、台胞证、护照等）正反面扫描件或照片

 企业备案：请提前准备好企业证件（例如营业执照、组织机构代码证等）以及备案负责人证件正反面扫描件或照片
 
 其他资料：当地管局备案规则中要求的其他证明材料（例如域名证书、法人委托授权书等）

<a id="access1"></a>
### **主体不在七牛云接入备案** 
**1.** **激活 [QVM 云主机服务](https://portal.qiniu.com/qvm/)**
**2. **登录[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)，根据原备案信息填写备案域名以及主体信息，然后单击**验证备案类型**。

**3. 产品验证** 通过**备案服务号**进行验证，备案服务号可通过七牛云主机平台中**备案**申请。备案服务号申请规则，请参考文档[申请备案服务号](/qvm/manual/qvm-registration-number)。

**4. 备案密码验证** 提交接入备案申请时会提示输入工信部的备案密码，每个备案号都有一个专属密码。若忘记备案密码，请访问[工信部备案管理系统](http://www.miitbeian.gov.cn)进入你备案省市通信管理局**找回备案密码**。

**5. 根据提示输入基本信息。**

**6. 上传资料**

- 上传个人证件扫描件或照片

- 上传网站备案信息真实性核验单需单击右侧**下载**按钮，下载、打印、用黑色签字笔填写，再扫描或拍照后上传

- 所有资料上传完成后，单击**提交备案**

- 在弹出的提示框中，单击**确定**

提交备案至初审需 1 个工作日，请保持备案信息中的联系电话畅通以便工作人员与你核实信息。

**7.初审通过**后，登录[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)，办理网站负责人拍照核验。

- **上海地区用户** 上海市通信管理局要求，上海地区所有备案用户**必须**使用上海通信管理局 app 人像核验方式进行网站负责人核验。具体操作步骤，请参考合作方阿里云文档[上海管局 app 核验网站负责人](https://help.aliyun.com/document_detail/42692.html?spm=a2c4g.11186623.2.21.c9vTZQ)。

- **非上海地区用户** 阿里云可提供幕布帮助你完成网站负责人核验过程

	- 如**我已有幕布**并已拍照，请单击**上传照片**。
    
	- 如**我没有幕布**，请**填写邮寄地址**。阿里云会将幕布邮寄至此地址（此地址必须为中国大陆境内地址）。收到幕布后，自行拍照并上传照片进行审核。
    
    拍照要求：
    
    - 拍照人必须与网站负责人为同一人。
    
    - 负责人需站在幕布中间位置，身着当季服装拍摄上半身即可。请你避免身着红色或蓝色上衣进行拍照。
    
    - 整个拍照背景必须是阿里云幕布，且为蓝色，幕布字迹清晰显示。
    
照片通过审核后，我们会在 1 个工作日内将你的备案信息提交省通信管理局审核。

**8.手机号码短信核验**。**天津**、**甘肃**、**西藏**、**宁夏**、**海南**、**新疆**、**青海**为工信部要求的手机号码短信核验试点省份，用户需完成手机号码短信验证后，备案申请才能成功提交管局审核。其他省份暂无短信核验要求。

有关如何短信核验具体操作步骤，请参考合作方阿里云文档[备案短信核验](https://help.aliyun.com/document_detail/63826.html?spm=a2c4g.11186623.2.22.c9vTZQ)。

备案申请信息成功提交管局系统后，管局审核一般为 3-20 个工作日。管局审核结果将以短信及邮件形式通知，你也可以登录七牛云备案系统主页查看你的备案信息情况。

### **主体在七牛云接入备案** 

登录[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)，在你的备案信息页面中，单击**继续接入备案**按钮，然后根据提示完成网站接入备案。

**注意：**如果页面上没有显示**继续接入备案**按钮，可能是你的备案主体下有其他进行中的备案订单。你可以放弃当前进行中的备案订单或待备案订单通过管局审核后，再进行增加接入备案。

按系统流程提交资料，并完成拍照核验后，**天津**、**甘肃**、**西藏**、**宁夏**、**海南**、**新疆**、**青海**用户，请注意查收工信部发出的短信验证码。该 7 省为工信部要求的手机号码短信核验试点省份，用户需完成手机号码短信验证后，备案申请才能成功提交管局审核。其他省份暂无短信核验要求。

有关如何进行短信核验操作的具体步骤，请参考合作方阿里云文档[备案短信核验](https://help.aliyun.com/document_detail/63826.html?spm=a2c4g.11186623.2.22.c9vTZQ)。

### **取消接入备案** 
**1.登录**[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)

**2. **找到你想取消的网站备案号，单击右侧的**取消接入**

**3. **填写你的备案好密码，确认取消接入即可。若忘记备案密码，请访问[工信部备案管理系统](http://www.miitbeian.gov.cn)进入你备案省市通信管理局**找回备案密码**。

注意：取消接入备案只是将你的备案信息与七牛云备案关联取消，备案号在工信部仍存在。请需尽快前往正在使用的服务器提供商备案系统完成备案接入申请，否则备案信息有被管局注销的风险。
