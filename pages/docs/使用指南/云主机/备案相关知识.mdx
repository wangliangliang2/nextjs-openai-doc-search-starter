<a id="step01"></a>
### **ICP 备案** 

由中国工业和信息化部 (MIIT) 发布的许可证。如果你需要在中国大陆地区托管网站，就必须申请 ICP（互联网内容提供商）备案。ICP 备案包含两种类型：

- ICP 备案号： 对于单纯的资讯性并且不涉及直销的网站，需要申请 ICP 备案号。七牛云主机用户需通过[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)提交备案申请。

- 经营性 ICP 许可证： 对于商品或服务的在线平台或第三方卖方，需获得商用许可证。要申请经营性 ICP 许可证，需先申请 ICP 备案号。经营性备案需用户自行到当地相关部门进行办理。

<br>

<a id="step02"></a>
### **接入备案** 

域名已通过其他服务商取得备案号，现要将域名指向七牛云主机，由七牛云主机托管网站，则需要将该网站域名的备案信息接入七牛云。

<br>

<a id="step03"></a>
### **变更备案** 

修改和变更备案的信息（主体信息或网站信息）

<br>

<a id="step04"></a>
### **增加网站备案** 

备案主体在七牛云存在备案，现有使用七牛云主机的新网站需备案，需通过增加网站备案。

<br>

<a id="step05"></a>
### **无主体增加网站备案** 

备案主体已通过其他服务商有过成功的网站域名备案，现有使用七牛云主机托管的新网站需备案，需在七牛云进行新网站域名备案。

<br>

<a id="step06"></a>
### **备案服务号** 

备案服务号是用于在[七牛云合作备案系统](https://portal.qiniu.com/qvm/beian)填写备案信息时关联七牛云主机的验证码，例如`32a43232-a888-88e8-b9ec-00888be8888f`。七牛云主机 QVM，需通过备案服务号验证产品。备案服务号通过七牛云主机控制台进行申请。

<br>

<a id="step07"></a>
### **备案密码** 

提交接入备案申请或注销备案时，会提示输入工信部的备案密码，每个备案号都有一个专属密码。

<br>

<a id="step08"></a>
### **真实性核验** 

需备案网站负责人到核验点拍照或在七牛云合作备案系统申请邮寄幕布，收到幕布后自行拍照上传照片。

<br>

<a id="step09"></a>
### **注销备案** 

如果不再继续使用已备案的主体或域名，需注销备案信息。注销分为以下两种：

- 注销主体：将备案主体的全部信息注销。注销后备案的主体和域名可重新进行备案申请。

- 注销网站：将备案主体下某个网站备案信息注销。注销后主体依然存在，主体下其他未注销网站不受影响。

<br>

<a id="step10"></a>
### **取消接入** 

将备案信息与七牛云备案关联删除，七牛云将不再是你的接入服务商，域名也无法指向七牛云境内服务器实现访问。但你在工信部的备案号仍然存在，你需尽快前往正在使用的服务提供商提交接入备案申请，否则备案信息有被管局注销的风险。
