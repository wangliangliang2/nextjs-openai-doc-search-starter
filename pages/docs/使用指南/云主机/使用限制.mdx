# 使用限制
- 每个迁移源仅能关联一个未完成状态的迁移任务。未完成状态包括Ready（未开始）、Running（运行中）、Stopped（已暂停）、InError（出错）和Expired（已过期）。
- 每个中转实例会创建一个专有网络VPC，每个账号在同一个七牛云地域下的VPC限额为10（包括您自行创建的VPC和中转实例创建的VPC）。
- 迁移源为Linux系统时，系统引导程序GRUB需为2.02及以上版本。
