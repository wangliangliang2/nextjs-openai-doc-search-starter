### 域名要求
- 支持使用精确域名（例如，www.qiniu.com）和泛域名（例如，\*.qiniu.com）格式。具体说明如下：
使用泛域名后，WAF将自动匹配该泛域名对应的所有子域名（例如，\*.qiniu.com能够匹配www.qiniu.com、test.qiniu.com等）。
- 如果同时存在泛域名和精确域名配置，则精确域名的转发规则和防护策略优先生效。
- 暂不支持添加.edu域名。

### 协议及端口说明
- 协议支持 HTTP 和 HTTPS
- 支持多个非标端口，可配置端口支持模糊搜索查询

## 2. 编辑域名
和添加域名的操作基本一致，但无法修改当前域名。

## 3. 删除域名
删除域名将会导致所有关于该域名的防护配置失效，且防护配置无法备份还原。

## 4. 切换WAF开关
可以通过切换WAF开关调节单个域名的防护状态，一旦域名的WAF状态处于关闭，则所有该域名的防护配置都无法生效。
![截屏2021-05-11 下午3.21.05.png](https://dn-odum9helk.qbox.me/FlNMwKYTFhZyHe7AP1g0L8ZQXL7D)

## 5. 上传证书
对于开启HTTPS协议的域名，需要上传有效证书及私钥信息。
