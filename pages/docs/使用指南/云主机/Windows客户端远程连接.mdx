# 开始之前
在部署VPN网关前，确保您的环境满足以下条件：
- 本地设备和VPC的私网IP地址段不能相同，否则无法通信。
- 客户端必须能访问Internet。

# 步骤一：创建VPN网关
完成以下操作，创建用户网关。

1. 登录[控制台](https://portal.qiniu.com/qvm/network/vpnGateway)。
2. 在左侧导航栏，单击VPN网关 。
3. 在VPN网关页面，单击创建VPN网关。
4. 在购买页面，配置VPN网关，然后单击立即购买完成支付。
注：选择选择VPN网关的地域时，确保VPC的地域和VPN网关的地域相同。

# 步骤四：客户端配置
完成以下操作，配置Windows客户端。
注：需要以管理员身份运行客户端。
1. 下载并安装OpenVPN客户端。
2. 将步骤三中下载的证书解压后复制到OpenVPN安装目录中的config文件夹中。
3. 单击Connect发起连接。
