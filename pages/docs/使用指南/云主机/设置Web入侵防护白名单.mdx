# 设置Web入侵防护白名单

网站接入Web应用防火墙防护后，您可以通过设置Web入侵防护白名单，让满足指定特征的请求不经过规则防护引擎、深度学习引擎的检测。Web入侵防护白名单一般用于放行因触发Web入侵防护相关规则被误拦截的特殊业务请求。

![截屏2021-05-11 下午3.49.51.png](https://dn-odum9helk.qbox.me/FqvsGmvag3RHJ2XWfaRDE_b_JC3E)

Web入侵防护为网站提供针对Web通用攻击的防护能力和对0day漏洞的快速响应能力，保证网站安全性，具体包含以下检测模块：
- 规则防护引擎
- 深度学习引擎

![截屏2021-05-11 下午3.51.21.png](https://dn-odum9helk.qbox.me/Fgc-IJMU5PP2Tl-iOTraukovMF8K)

需要注意的是白名单要求选择**不检查模块**，选中规则防护引擎后，默认不检测规则防护引擎中包含的全部规则。
