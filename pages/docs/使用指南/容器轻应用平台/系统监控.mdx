 为了方便用户了解自己启用实例的运行情况和健康状态等信息，QAPP会为每个用户启用的实例进行状态监控，您可以根据这些信息了解实例池的负载情况，并根据需要调解实例的数量。

**监控服务有以下特性：**

- 监控服务延时较低，您可以看到最近5分钟之前的数据（不包括实例运行时间）。

- 监控的数据保留7天。

  


# **操作步骤**

1. 登录 [容器轻应用平台](https://portal.qiniu.com/dora/qapp)。

2. 在左侧导航栏单击 **具体应用名称**，在顶部菜单栏选择 **监控中心**。

   ![img](https://dora-doc.qiniu.com/WX20221103-202445%402x.png)

3. 在监控中心页面，顶部tab栏选择 **监控项** 类型。

   | 监控项       | 单位      | 说明                                                 |
   | ------------ | --------- | ---------------------------------------------------- |
   | CPU使用率    | %         | 展示CPU使用率。                                      |
   | 内存占用     | %         | 展示CPU内存占用情况。                                |
   | GPU使用率    | %         | 展示GPU使用率。                                      |
   | 显存占用     | %         | 展示GPU显存占用情况。                                |
   | 请求次数     | 次/秒     | 按任务调用成功、失败展示某个时间内的请求次数。       |
   | 实例运行时间 | h（小时） | 按实例规格类型的维度来展示某个时间段内总的运行时间。 |

4. 选择查询条件，**时间范围**、**区域**。

   | 查询字段 | 说明                                                         |
   | -------- | ------------------------------------------------------------ |
   | **区域** | 必填，下拉选择区域                                           |
   | **时间** | 必填，可以单击 **刷新** 到最新的时间，默认为最近1周；<br/>1）固定时间范围：6小时、1天、1周；<br/>2）自定义时间范围，最大时间跨度为7天（除实例运行时间无限制）。 |

5. 查看监控数据，以动态图的方式展示。

   | 监控项       | 示例图                                                       |
   | ------------ | ------------------------------------------------------------ |
   | CPU使用率    | ![img](https://dora-doc.qiniu.com/WX20221107-165223%402x.png) |
   | 内存占用     | ![img](https://dora-doc.qiniu.com/WX20221104-105537%402x.png) |
   | GPU使用率    | ![img](https://dora-doc.qiniu.com/WX20221104-142402%402x.png) |
   | 显存占用     | ![img](https://dora-doc.qiniu.com/WX20221104-142421%402x.png) |
   | 请求次数     | ![img](https://dora-doc.qiniu.com/WX20221107-165326%402x.png) |
   | 实例运行时间 | ![img](https://dora-doc.qiniu.com/WX20221104-105312%402x.png) |
