当应用出现异常情况时，您可以通过控制台查看日志，以便定位问题。



# **前提条件**

1. 发布新版本时，需要配置的日志输出路径，来采集应用日志，方便业务日志对接。

# **操作步骤**

1. 登录 [容器轻应用平台](https://portal.qiniu.com/dora/qapp)。

2. 在左侧导航栏单击 **具体应用名称**，在顶部菜单栏选择 **日志查询**。

   ![img](https://dora-doc.qiniu.com/WX20221103-201539%402x.png)

3. 在查询日志页面，选择 **查询条件**。

   | 查询字段 | 说明                                                         |
   | -------- | ------------------------------------------------------------ |
   | 区域     | 必填，下拉选择区域                                           |
   | 发布版本 | 必填，下拉选择应用版本                                       |
   | 日志类型 | 必填，默认为全部；<br>1) 固定类型：标准输出、标准错误输出；<br/>2) 自定义类型：显示您在设置日志的的采集路径。 |
   | 时间     | 必填，默认为最近1小时；<br/>1) 固定范围：1小时、1天、1周；<br/>2) 自定义时间范围，最大时间跨度为7天。 |
   | 更多筛选 | 默认收起，点击展开；<br/>1) 实例 ID<br/>2) 关键字<br/>3) 日志条数 |

4. 点击 **立即查询** ，可勾选 **显示详细日志**，展示查询的日志内容。

   ![img](https://dora-doc.qiniu.com/WX20221109-161016%402x.png)
