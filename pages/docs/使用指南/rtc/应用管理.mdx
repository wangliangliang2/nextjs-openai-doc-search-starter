<div class="rtc-doc">
    <div><p>这一部分服务是为了让您的业务后台能够自动地完成一些在控制台上需要完成的操作，比如创建应用，编辑应用等。
对于这些服务，我们统一封装了 Restful 的接口方便给您调用，在调用各个接口之前，您还需要在 http headers 中添加鉴权头以识别您的权限。</p>
<blockquote>
<p>这些服务都可以通过服务端 SDK 来完成，以下都是介绍手动请求的情况。</p>
</blockquote>
<h2 id="http-请求鉴权"><a href="#http-%E8%AF%B7%E6%B1%82%E9%89%B4%E6%9D%83" ariaHidden="true" tabIndex="-1"><span className="icon,icon-link"></span></a>HTTP 请求鉴权</h2>
<p>Qiniu RTC Server API 通过 Qiniu Authorization 方式进行鉴权，每个房间管理 HTTP 请求头部需增加一个 Authorization 字段：</p>
<pre><code className="language-bash"><span class="hljs-keyword">Authorization</span>: &quot;&lt;QiniuToken&gt;&quot;
<span></span></code></pre>
<p>QiniuToken 是管理凭证，用于鉴权。
<br></br>
使用七牛颁发的AccessKey和SecretKey，对本次 http 请求的信息进行签名，生成管理凭证。签名的原始数据包括 http 请求的 Method, Path, RawQuery, Content-Type 及 Body 等信息，这些信息的获取方法取决于具体所用的编程语言，建议参照七牛提供的 SDK 代码。 计算过程及伪代码如下：</p>
<pre><code className="language-bash"><span class="hljs-comment">// 1.构造待签名的 Data</span>
<span></span><span class="hljs-comment">// 添加 Method 和 Path</span>
<span></span><span class="hljs-keyword">data</span> = <span class="hljs-string">&quot;&lt;Method&gt; &lt;Path&gt;&quot;</span>
<span></span><span class="hljs-comment">// 添加 Query，如果Query不存在或者为空，则跳过此步</span>
<span></span><span class="hljs-keyword">if</span> <span class="hljs-string">&quot;&lt;RawQuery&gt;&quot;</span> != <span class="hljs-string">&quot;&quot;</span> {
<span></span>        <span class="hljs-keyword">data</span> += <span class="hljs-string">&quot;?&lt;RawQuery&gt;&quot;</span>
<span></span>}
<span></span><span class="hljs-comment">// 添加 Host</span>
<span></span><span class="hljs-keyword">data</span> += <span class="hljs-string">&quot;\nHost: &lt;Host&gt;&quot;</span>
<span></span><span class="hljs-comment">// 添加 Content-Type，如果Content-Type不存在或者为空，则跳过此步</span>
<span></span><span class="hljs-keyword">if</span> <span class="hljs-string">&quot;&lt;Content-Type&gt;&quot;</span> != <span class="hljs-string">&quot;&quot;</span> {
<span></span>        <span class="hljs-keyword">data</span> += <span class="hljs-string">&quot;\nContent-Type: &lt;Content-Type&gt;&quot;</span>
<span></span>}
<span></span><span class="hljs-comment">// 添加回车</span>
<span></span><span class="hljs-keyword">data</span> += <span class="hljs-string">&quot;\n\n&quot;</span>
<span></span><span class="hljs-comment">// 添加 Body， 如果Content-Length, Content-Type和Body任意一个不存在或者为空，则跳过此步；如果Content-Type为application/octet-stream，也跳过此步</span>
<span></span>bodyOK := <span class="hljs-string">&quot;&lt;Content-Length&gt;&quot;</span> != <span class="hljs-string">&quot;&quot;</span> &amp;&amp; <span class="hljs-string">&quot;&lt;Body&gt;&quot;</span> != <span class="hljs-string">&quot;&quot;</span>
<span></span>contentTypeOK := <span class="hljs-string">&quot;&lt;Content-Type&gt;&quot;</span> != <span class="hljs-string">&quot;&quot;</span> &amp;&amp; <span class="hljs-string">&quot;&lt;Content-Type&gt;&quot;</span> != <span class="hljs-string">&quot;application/octet-stream&quot;</span>
<span></span><span class="hljs-keyword">if</span> bodyOK &amp;&amp; contentTypeOK {
<span></span>        <span class="hljs-keyword">data</span> += <span class="hljs-string">&quot;&lt;Body&gt;&quot;</span>
<span></span>}
<span></span><span class="hljs-comment">// 2. 计算 HMAC-SHA1 签名，并对签名结果做 URL 安全的 Base64 编码</span>
<span></span>sign = hmac_sha1(<span class="hljs-keyword">data</span>, <span class="hljs-string">&quot;Your_Secret_Key&quot;</span>)
<span></span>encodedSign = urlsafe_base64_encode(sign)  
<span></span><span class="hljs-comment">// 3. 将 Qiniu 标识与 AccessKey、encodedSign 拼接得到管理凭证</span>
<span></span>&lt;QiniuToken&gt; = <span class="hljs-string">&quot;Qiniu &quot;</span> + <span class="hljs-string">&quot;Your_Access_Key&quot;</span> + <span class="hljs-string">&quot;:&quot;</span> + encodedSign
<span></span></code></pre>
<h2 id="createapp-创建一个应用"><a href="#createapp-%E5%88%9B%E5%BB%BA%E4%B8%80%E4%B8%AA%E5%BA%94%E7%94%A8" ariaHidden="true" tabIndex="-1"><span className="icon,icon-link"></span></a>CreateApp 创建一个应用</h2>
<p>在七牛云控制台页面打开实时音视频云可以看到 连麦应用列表，这里每个应用都应该和您的实际业务上的应用相对应，也就是我们这里说的 app。app 是您在七牛实时音视频云的业务配置集合。 一个 app 帐号下的连麦房间（频道）拥有独立的命名空间，并且这些房间的可配置功能选项遵从 app 的配置，例如人数限制、是否允许抢流、合流、旁路直播等。您可以创建多个不同的 app（最 多 10 个） ，以实现不同的配置需求。注意，不同 app 下面所属的同名房间并不互通（因为 app 相互独立）</p>
<br></br>
下面列举了操作和管理这些 app 的接口，如果您希望通过服务端 SDK 来调用，请具体参照各个 SDK 自己的文档。

<span></span>POST /v3/apps
<span></span><span class="hljs-section">Authorization: qiniu mac</span>
<span></span><span class="hljs-section">Content-Type: application/json</span>
<span></span>{
<span></span>    <span class="hljs-string">&quot;hub&quot;</span>: <span class="hljs-string">&quot;&lt;Hub&gt;&quot;</span>,
<span></span>    <span class="hljs-string">&quot;title&quot;</span>: <span class="hljs-string">&quot;&lt;Title&gt;&quot;</span>,
<span></span>    <span class="hljs-string">&quot;maxUsers&quot;</span>: &lt;MaxUsers&gt;,
