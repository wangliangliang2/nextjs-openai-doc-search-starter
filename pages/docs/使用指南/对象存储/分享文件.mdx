您可以通过文件 URL 将存储在空间（Bucket）中的文件（Object），分享给第三方预览或下载。

Kodo 控制台支持您复制分享单个文件外链或批量导出文件外链。
<br>


# **说明**
- 仅支持对公开空间中的文件进行外链分享。
- 当开启原图保护时，不支持将文件外链分享，详情请见[设置原图保护](/kodo/development_guidelines/8560/set-the-original-protection)。
- 对于归档、深度归档存储类型的文件，需要确认文件已“解冻”完成才能成功对其进行分享，具详情请见[解冻文件](/kodo/development_guidelines/8573/thaw-file-storage)。
<br>


# **操作步骤**
1. 登录 Kodo 控制台进入[空间管理](https://portal.qiniu.com/kodo/bucket)界面，在空间列表中
2. 单击**目标空间名称**进入空间概览，后单击**文件管理**进入文件管理界面；或单击目标空间栏**文件**，快速链接至文件管理界面
3. 文件管理界面下方显示了当前空间的文件列表，如图1:文件列表

## **单个文件分享**
1. 鼠标移至目标文件栏的**更多**如图1，以获取下拉菜单
2. 单击**复制外链**
3. 将文件 URL 分享给第三方预览或下载
<br>

## **批量文件分享**
1. 单击**文件名前的选择框**，勾选或全选文件列表中所需的目标文件 
2. 鼠标移至**批量操作**以获取下拉菜单如图1
3. 单击**导出 URL**，完成外链批量导出至 Excel 文件 
   *注：该文件包含 Object 及对应的 URL，并将保存至浏览器默认路径*
4. 将文件 URL 分享给第三方预览或下载。
