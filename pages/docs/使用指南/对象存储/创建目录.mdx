对象存储 Kodo 支持通过目录来管理空间（Bucket）中的文件（Object），以便于您合理归类您的文件。您可以通过 Kodo 控制台创建目录。

<br>

# **说明**
- Kodo 默认将以 `/` 结尾的对象作为文件目录的形式展现。
- 支持创建单、多层级目录。

<br>

# **操作步骤**
1. 登录 Kodo 控制台进入[空间管理](https://portal.qiniu.com/kodo/bucket)界面，在空间列表中
2. 单击**目标空间名称**进入空间概览，后单击**文件管理**进入文件管理界面；或单击目标空间栏**文件**，快速链接至文件管理界面
3. 单击**创建目录**，显示如图1的弹窗
图1:
![image.png](https://dn-odum9helk.qbox.me/FjvWGfy8hyvyT2aF7vzjF2a8WqBl)
4. 在图1，按如下说明输入参数
   |操作名称|说明|规则描述|
   |-|-|-|
   |目录名|自定义目录名|- 输入目录名，不支持仅由英文句号 `.` 组成的名称<br> - 不支持以 `/` 开头， 不能包含连续的 `/` <br>  - 支持以 `/` 分割路径，快速创建子目录|
5. 单击**确定**，完成创建目录
