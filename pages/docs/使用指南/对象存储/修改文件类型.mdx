对象存储 Kodo 支持对已上传的文件（Object）类型作出修改，方便您在上传文件后仍能快捷编辑文件类型。
<br>


# **说明**
- 对于归档、深度归档存储类型的文件，需要确认文件已“解冻”完成才能修改文件类型，详情请见[解冻文件](/kodo/development_guidelines/8573/thaw-file-storage)。

<br>

# **操作步骤**
1. 登录 Kodo 控制台进入[空间管理](https://portal.qiniu.com/kodo/bucket)界面，在空间列表中
2. 单击**目标空间名称**进入空间概览，后单击**文件管理**进入文件管理界面；或单击目标空间栏**文件**，快速链接至文件管理界面
3. 在如图1的文件列表中，单击目标文件栏文件类型旁的**修改图标**
图1:
![截屏2021-08-09 下午2.47.15.png](https://dn-odum9helk.qbox.me/Fka06O037GT-8CQv_GT750EpxfYH)
4. 在将出现如图2界面，输入新的文件类型后单击**确定**完成修改
