<a id="path-style"></a>
七牛对象存储兼容 AWS S3 的 path-style 和 bucket virtual hosting 两种访问方式，以 GetObject 为例

| 风格 | 示例 |
| ---- | ---- |
| Path Style | `http://s3-cn-east-1.qiniucs.com/<s3空间名>/objectname` |
| Virtual-host Style | `http://<s3空间名>.s3-cn-east-1.qiniucs.com/objectname` |

s3 空间名：全局唯一（遵循 s3 协议的要求）
* 空间名称全局唯一时，s3 空间名=空间名称；空间名称全局不唯一时，系统会为空间自动生成一个 s3 空间名
* 七牛对象存储空间的 s3 空间名，可以通过 [Get Service](/kodo/manual/4087/compatible-s3-api#service-operation) 接口获取，也可在 [开发者平台-对象存储-空间概览](https://portal.qiniu.com/bucket) 中查看
