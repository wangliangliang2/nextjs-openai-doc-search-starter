# 一、填写域名并点击找回

由于域名在其他账号下创建，域名归属权不在当前账号，需要进行验证操作才可以获得该域名

![](https://dn-odum9helk.qbox.me/FrlOz7pZBpZtDlaSdEeJT4dzq5Lg1)
<br>
# 二、获取验证文件并使其可以公网访问

域名如果解析到之前的存储空间，下载该文件并上传到之前账号的存储空间
域名如果解析到自己的服务器，下载该文件并上上传到自己的服务器网站根目录

**注意**：需要保持 `http://<您要找回的域名>/verifyingfile.html` 这个链接可以正常访问

![](https://dn-odum9helk.qbox.me/FoeTFqrReu5J5Atcq_5yqZu0mQcSshqn3)
<br>

达到如下效果即可

![](https://dn-odum9helk.qbox.me/FpmKTchI4wSCr_aYaDRY-n9qP9D8)
