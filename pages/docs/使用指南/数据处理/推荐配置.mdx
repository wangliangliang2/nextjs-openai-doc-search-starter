
工作流满足对多媒体文件的多任务处理需求，可以灵活组建场景化处理流程，推荐配置主要介绍一些媒体处理相关的配置经验，提供了不同场景的音视频处理工作流，为您提供参考。同时，可以在推荐配置的

# 使用推荐配置

1. 登录【智能多媒体控制台】，进入【导航栏->工作流】页面。

2. 点击右上角的 **点击适配** ，进入推荐配置页面。

   ![img](https://dora-doc.qiniu.com/WX20201217-112441.png)

3. 下拉选择 **推荐配置方案**，提供了**多种**不同多媒体处理业务场景。

   ![img](https://dora-doc.qiniu.com/WX20201217-141557.png)

4.  **demo 效果演示**

    > 点击【播放】按钮，可以查看对比【源视频】和【处理后视频】，来判断是否符合您的处理预期。
   ![img](https://dora-doc.qiniu.com/WX20210223-154756%402x.png)

5. **配置详情**

    > 允许编辑工作流配置，方便用户修改调整参数，以产出一个符合自己业务场景的处理流程。

   ![img](https://dora-doc.qiniu.com/WX20201217-143001.png)
   ​

6.  **去创建任务触发器**
    > 需要 kodo 上传自动触发任务处理时，点击  **去创建任务触发器**  按钮，跳转到【创建任务触发器】页面，快速引用工作流配置，如何创建，请查看 [任务触发器](https://developer.qiniu.com/dora/manual/6489/task-workflow)

7.  **去创建任务** 

    > 需要对指定对象发起任务处理时，点击 **去创建任务** 按钮，跳转到【创建任务】页面，快速引用工作流配置，如何创建，请查看 [任务](https://developer.qiniu.com/dora/manual/6488/task-01)




# **推荐配置列表**

[查看推荐配置详情](https://portal.qiniu.com/dora/media-gate/template/recommend)

| 顺序 | 推荐配置| 使用场景                                                                                             |
| ---- | ------------------------------------------------- | ------------------------------------------------------------ |
| 1    | 转封装                                            | 将视频转换成不同的封装格式的视频流推送给观众，例如直播。     |
| 2    | 普通转码                                          | 对音视频进行转码，为用户提供流畅清晰的观看体验。             |
| 3    | 普通转码 + 视频水印                               | 对音视频进行转码，并增加品牌水印。                           |
| 4    | 普通转码+设置多个水印                             | 对音视频进行转码，并在视频的不同时段设置多个水印，曝光品牌保护版权 |
| 5    | 普通转码 + 音视频编辑（音频替换）                 | 对音视频转码，并支持音频混音、音频替换、视频马赛克、添加字幕等音视频编辑操作。 |
| 6    | 普通转码 + 视频片段截取                           | 抽取视频的某一片段并进行转码，可作为视频动态封面。           |
| 7    | 锐智转码                                          | 根据视频场景决定最佳编码方式，对视频体积瘦身，并保证用户观看质量，帮助企业降低存储成本和带宽流量。 |
| 8    | 锐智转码+视频水印                                 | 对视频进行转码瘦身、增加水印，保证观看体验并节省存储和带宽成本，曝光品牌保护版权，适用于制作视频教程。 |
| 9    | M3U8 多码率转码+视频水印                          | 在云端转码成不同规格的视频，适配不同终端设备播放，加入水印保护视频版权。 |
| 10   | M3U8 多码率转码+视频水印+音视频编辑辑（添加字幕） | 在云端转码成不同规格的视频，适配不同终端设备播放，加入水印保护视频版权，支持对音视频进行编辑，例如视频马赛克、音频混音、音频替换、添加字幕等 |
| 11   | 视频拼接                                          | 在源视频的头部或者尾部拼接视频片段，常用于短视频剪辑。       |
| 12   | 视频拼接-->锐智转码                               | 对原视频进行片头片尾拼接，再对拼接后的视频进行锐智转码，用于制作教程视频，可以帮助用户节省成本。 |
| 13   | 音视频元信息-->执行条件-->普通转码+截图           | 获取音视频元信息后，判断原视频分辨率范围，对某范围内的视频做转码+截图处理，减少不必要的资源处理，帮助用户节省成本。 |
| 14   | 视频审核->执行条件—>封禁+普通转码          | 对原视频进行鉴黄、鉴政、鉴暴恐审核，对审核通过的视频进行转码，并封禁审核违规的源视频，帮助用户规避风险、节省转码成本。 |
