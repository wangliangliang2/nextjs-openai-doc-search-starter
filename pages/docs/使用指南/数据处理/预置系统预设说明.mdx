# **截图系统预设**

| 截图类型 | 预设名称     | 截图格式(format) | 开始时间(offset) | 宽*高(w×h)       |
| -------- | ------------ | ---------------- | ---------------- | ---------------- |
| 单张截图 | 预置单张截图 | Jpg              | 00时00分01秒     | 和源视频保持一致 |

**备注：**

1. 预设未注明的参数，同源视频保持一致。
2. 表中 ”-“ ，表示同源视频保持一致。

# **自定义预设**

当系统预设不能满足需求时，用户可以自定定义转码参数，即创建“自定义预设”。

可通过控制台，创建对应类型的自定义预设：

- [创建普通转码预设](https://developer.qiniu.com/dora/manual/6492/transcoding-preset)

- [创建锐智转码预设](https://developer.qiniu.com/dora/manual/6850/perceptive-transcoding-preset)

- [创建截图预设](https://developer.qiniu.com/dora/manual/6494/screenshot-of-the-preset)

- [创建水印预设](https://developer.qiniu.com/dora/manual/6493/the-watermark-default)

- [创建视频拼接预设](https://developer.qiniu.com/dora/manual/6851/video-stitching-preset)
