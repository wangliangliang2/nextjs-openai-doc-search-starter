智能多媒体平台提供了多样化的媒体处理能力，包括音视频普通转码、锐智转码、视频截图、视频水印、音视频元信息等功能，后续将会开放更多计算能力。本文为您介绍多媒体处理的概述，帮助您对多媒体处理有大致了解。
​
# **简介**

七牛云的多样化的多媒体处理能力，以经济、弹性和高可扩展的音视频处理，帮助您将音视频转码成适合在PC、TV以及移动终端上播放的格式。用户可根据需求来编排丰富的多媒体处理流程，来满足不同的业务场景。

您可以通过控制台自定义处理流程、或者调用API来处理。更多信息，请参见 [工作流](https://developer.qiniu.com/dora/manual/6490/the-workflow-template)。

​
# **处理能力**

| 媒体处理     | 功能描述                                                     | 参考文档                                                     |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 普通转码     | 提供丰富的转码处理功能，将一个音视频文件转换成另一个或多个音视频文件，以适应不同网络带宽、不同终端设备和不同的用户需求。 | [普通转码](/dora/manual/7507/ordinary-transcoding01) |
| 锐智转码     | 锐智转码在编码过程中可以更加高效地分配比特，在保持总体码率不变情况下改善编码视频的视觉质量。通过加入这种场景内容自适应技术，原有的分辨率和码率组合会更加精简，从而有效降低内容提供商的存储和分发成本。 | [锐智转码](/dora/manual/7508/perceptive-transcoding01) |
| 视频截图     | 在指定视频特定的位置进行图像截取，并生成图片文件。           | [视频截图](/dora/manual/7509/video-capture01) |
| 视频水印     | 在视频的编解码过程中将图片、文字等信息压制到视频流中合并输出一个带图文水印的新的视频文件。 | [视频水印](/dora/manual/7511/video-watermarking01) |
| 视频拼接     | 用于将指定的数个音频片段拼接成一段音频，或者将数个视频片段拼接成一段新视频。 | [视频拼接](/dora/manual/7510/video-stitching01) |
| 音视频元信息 | 用于获取指定音频、视频资源的元信息。                         | [音视频元信息](/dora/manual/7515/audio-and-video-metadata-information) |
| 音视频元信息 | 用于获取指定音频、视频资源的元信息。                         | [音视频元信息](/dora/manual/7515/audio-and-video-metadata-information) |
| 视频审核 | 支持视频审核，通过自动化审核多媒体资源，极大降低色情、暴恐、涉政等内容违规风险，降低人工审核成本。  | [视频审核](/dora/manual/7771/video-review) |
| 封禁 | 指对审核结果为违规或疑似的文件，进行自动封禁处理。这些自动封禁的文件，终端用户不能进行访问。  | [视频审核](/dora/manual/7776/封禁) |

# **使用场景**

通过智能多媒体平台，可以解决如下问题：

- **多终端设备适配**：适配PC、TV以及移动终端等多平台播放。
- **降低存储&分发成本**： 保证在相同画质质量的前提下，调整视频码率、提高视频压缩率、减小文件体积，从而减少播放卡顿并节省存储和流量费用。
- **多任务组合编排**： 通过配置工作流，自定义编排成一个完整的任务处理流程。
- **事件驱动**：可以选择 kodo 自动触发 ，丰富的触发规则（当前仅支持对增量数据做处理）
