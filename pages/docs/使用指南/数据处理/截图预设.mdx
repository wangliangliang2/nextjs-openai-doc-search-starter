# **新建截图预设**

1.登录【智能多媒体控制台】，进入【预设集->截图预设集】页面。

2.点击右上角的 **新建预设** 按钮。

3.进入 **添加预设详情页**，配置预设参数，配置完成，点击 **保存** 即可完成。

![img](https://dora-doc.qiniu.com/perset-29.png)



> **截图预设参数：**
>
> - 名称：仅支持中文、英文、数字、_、-、. 六种格式，长度不能超过32个字符。
> - 截图类型：
>   - 单张截图
>   - 多张截图
> - 截图格式：仅支持 jpg、png 两种格式。
> - 图片大小(宽*高)
> - 单张截图参数
>   - 开始时间
> - 多张截图参数
>   - 截图间隔时间
>   - 开始时间
>   - 结束时间
>
> **备注：** 参数含义及取值范围说明，点击查看 [预设参数说明](/dora/manual/6501/the-preset-parameters-that)。

# **参数说明**

## 单张截图(vframe)

| 参数             | 必填 | 说明                                                         |
| ---------------- | ---- | ------------------------------------------------------------ |
| 截图格式(format) | 是   | 输出的目标截图格式，支持jpg、png                             |
| 开始时间(offset) | 是   | 指定截取视频的时刻，单位：秒，精确到毫秒。                   |
| 图片大小(w×h)    | 否   | • 0. 指定截图分辨率，格式为宽 x 高 ，长边取值范围[1,3840]的整数，短边取值范围[1,2160]的整数。<br />• 宽、高都为空时，则和原视频保持一致<br />• 宽为空，高不为空，则按高的值对原视频等比例缩放<br />• 宽不为空，高为空时，则按宽的值对原视频等比例缩放<br />• 宽、高都不为空时，则根据宽、高来缩放 |

## 多张截图(vsample)

| 参数                   | 必填 | 说明                                                         |
| ---------------------- | ---- | ------------------------------------------------------------ |
| 截图格式(format)       | 是   | 输出的目标截图格式，支持 jpg、png 两种格式                   |
| 截图间隔时间(interval) | 否   | 默认值为5秒，指定采样间隔，单元：秒<br />• 请输入正整数，值不超过100 |
| 开始时间(ss)           | 是   | 采样总时，单位：秒，精确到毫秒<br />• 时：不超过50；分：不超过59；秒：不超过59 |
| 结束时间(t)            | 是   | 指定截取视频的时刻，单位：秒，精确到毫秒<br />• 时：不超过50；分：不超过59；秒：不超过59 |
| 图片大小(w×h)          | 否   | • 指定截图分辨率，格式为宽 x 高 ，长边取值范围[1,3840]的整数，短边取值范围[1,2160]的整数。<br />• 宽、高都为空时，则和原视频保持一致<br />• 宽为空，高不为空，则按高的值对原视频等比例缩放<br />• 宽不为空，高为空时，则按宽的值对原视频等比例缩放<br />• 宽、高都不为空时，则根据宽、高来缩放 |
