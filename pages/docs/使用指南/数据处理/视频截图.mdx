支持视频截图，您可通过控制台管理截图预设。本文为您介绍视频截图、截图类型、使用说明、截图模板和截图参数，以及如何管理截图预设。
​
# **简介**

视频截图，是指在指定视频特定的位置进行图像截取，并生成图片文件。为了提高易用性，视频点播提供了截图模板，用户可先将相关参数配置成截图模板，在发起截取任务时选择截图预设即可。
​
## 截图类型

当前支持两种截图类型：单张截图、多张截图。

- **单张截图**

  ![](https://dora-doc.qiniu.com/WX20201214-142100.png)

  > - 对指定视频截取1张图片，可设置开始截图的时间点、截取的图片宽高、图片格式等。
  >
  >
  > - 适合用来设置视频封面。

  ​


- **多张截图**

  ![](https://dora-doc.qiniu.com/WX20201214-142116.png)

  > - 对指定视频截取一定数量的图片，可设置截图的开始时间和结束时间、截图的总数、截图的时间间隔以及截取的图片宽高、图片格式等。
  > - 适合用于对视频内容做分析、识别、审核等。

  ​

  注意：当截图开始时间或者结束时间超出视频时长范围时，会导致截图失败。

  ​

# **使用方式**

视频截图的使用流程如下：

1. 创建截图预设，具体操作，请参见控制台[截图预设](https://portal.qiniu.com/dora/media-gate/preset?activeTab=screenshot)。

2. 使用创建好的截图预设，去创建工作流，具体操作请参考控制台[工作流](https://portal.qiniu.com/dora/media-gate/template/create)。

   ​

# **视频截图参数说明**

视频截图是用于从视频流中截取指定时刻的单帧或者多帧画面并按指定大小缩放成图片。

下面对各类型参数的关键参数含义及取值范围进行说明。

#### 1. 单张截图(vframe)

| 参数             | 必填 | 说明                                                         |
| ---------------- | ---- | ------------------------------------------------------------ |
| 截图格式(format) | 是   | 输出的目标截图格式，支持jpg、png                             |
| 开始时间(offset) | 是   | 指定截取视频的时刻，单位：秒，精确到毫秒。                   |
| 图片大小(w×h)    | 否   | • 宽取值范围 [1，3840]，高取值范围 [1，2160]，均为整数<br />• 宽、高都为空时，则和原视频保持一致<br />• 宽为空，高不为空，则按高的值对原视频等比例缩放<br />• 宽不为空，高为空时，则按宽的值对原视频等比例缩放<br />• 宽、高都不为空时，则根据宽、高来缩放 |

#### 2. 多张截图(vsample)

| 参数                   | 必填 | 说明                                                         |
| ---------------------- | ---- | ------------------------------------------------------------ |
| 截图格式(format)       | 是   | 输出的目标截图格式，支持 jpg、png 两种格式                   |
| 截图间隔时间(interval) | 否   | 默认值为5秒，指定采样间隔，单元：秒<br />• 请输入正整数，值不超过100 |
| 开始时间(ss)           | 是   | 采样总时，单位：秒，精确到毫秒<br />• 时：不超过50；分：不超过59；秒：不超过59 |
| 结束时间(t)            | 是   | 指定截取视频的时刻，单位：秒，精确到毫秒<br />• 时：不超过50；分：不超过59；秒：不超过59 |
| 图片大小(w×h)          | 否   | • 宽取值范围 [1，3840]，高取值范围 [1，2160]，均为整数<br />• 宽、高都为空时，则和原视频保持一致<br />• 宽为空，高不为空，则按高的值对原视频等比例缩放<br />• 宽不为空，高为空时，则按宽的值对原视频等比例缩放<br />• 宽、高都不为空时，则根据宽、高来缩放 |
