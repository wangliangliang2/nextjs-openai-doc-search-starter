# 阿里云侧操作
- 登录阿里云 ECS 管理控制台。
- 在左侧导航栏，选择实例与镜像 > 镜像。
- 在顶部菜单栏左上角处，选择地域。
- 在自定义镜像页签，选中待共享的自定义镜像。在操作栏，单击共享镜像。
- 在共享镜像对话框中，输入待共享用户的账号 ID（联系七牛云技术支持获取）并选中提示信息，然后单击共享镜像。

提示：阿里云账号 ID 的获取方式为：将鼠标移至控制台右上角的用户头像，在弹出的用户信息框中，如果标识了账号为主账号，则显示的账号 ID 即为阿里云账号 ID。

# 七牛云侧操作
- 联系七牛云技术支持将共享过来的镜像复制（复制镜像将在指定地域生成快照，会产生额外费用，请知悉）
- 登录七牛云 QVM 管理控制台。
- 在左侧导航栏，选择云主机 > 镜像。
- 选中共享的镜像，点击操作栏的「创建实例」

# 异常情况排查：
1、创建云主机实例无法选中共享的私有镜像
1. 确保云主机实例地域是否和镜像地域相同
2. 检查是否是镜像缺少 cloudinit 标记
- 登录源服务器
- 运行以下命令检查是否已安装 cloudinit.
``` language
which cloud-init
```
无路径输出信息表示未安装，您需要安装阿里云版cloud-init.
- 在服务器中安装 cloud-init 参考文档 https://help.aliyun.com/document_detail/57803.html 
- 如还有问题，请[提交工单](https://support.qiniu.com/tickets/category)联系七牛云技术支持
