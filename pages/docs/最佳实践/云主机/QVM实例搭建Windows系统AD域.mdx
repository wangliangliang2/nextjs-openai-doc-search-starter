<div><main role="main">         <article role="article" aria-labelledby="title_g9k_ycj_3dq">            <div>               <p class="shortdesc">活动目录AD（Active Directory）是微软服务的核心组件。AD能实现高效管理，例如批量管理用户、部署应用和更新补丁等。许多微软组件（例如Exchange）和故障转移群集也需要AD域环境。本文以Windows Server 2012 R2 Datacenter操作系统为例，介绍如何搭建AD域。
               </p>               <div>                  <div>                     <h1 font-weigh = "bold">前提条件</h1>                  </div>                  <ul class="ul" id="ul-l56-3sa-4jg">                     <li>安装者必须拥有管理员权限。</li>                     <li>安装分区必须为NTFS分区。</li>                     <li>实例必须支持DNS服务。</li>                     <li>实例必须支持TCP/IP协议，并且需要有固定私网IP地址。 
                        <div>                           <div><i class="icon-note note"></i></div>                           <div><strong>说明</strong> 建议使用固定IP地址，防止重启实例后IP地址发生变化。本文采用的是专有网络VPC，手动修改IP地址会导致IP失效。
                           </div>                        </div>                     </li>                  </ul>               </div>               <section class="section context" id="context-bu3-459-jc5">                  <div>                     <h1 font-weigh = "bold">背景信息</h1>                  </div>                  <ul class="ul" id="ul-e8a-qcp-5wt">                     <li>名词解释： 
                        <ul class="ul" id="ul-btm-bbb-r41">                           <li>DC：Domain Controllers，域控制器</li>                           <li>DN：Distinguished Name，识别名 </li>                           <li>OU：Organizational Unit，组织单位 </li>                           <li>CN：Canonical Name，正式名称 </li>                           <li>SID：Security Identifier，安全标识符</li>                        </ul>                     </li>                     <li>实例组网信息： 
                        <ul class="ul" id="ul-yzy-gt5-sx4">                           <li>网络类型：专有网络VPC</li>                           <li>交换机私网网段：192.168.100.0/24</li>                           <li>是否启用网关：是
<img  class="image break" id="image-b8e-rzr-524" src="http://qey2kbifv.hn-bkt.clouddn.com/2020-09-17/6ZPf7NSV8qWhlyG9ICM1.png" width="600" alt="VPC组网"></li>                        </ul>                     </li>                     <li>域名信息： 
                        <ul class="ul" id="ul-02e-595-24y">                           <li>域名：lyonz.com</li>                           <li>DC：192.168.100.105</li>                           <li>需要加入域的客户端（Client）IP地址：192.168.100.106
<img  class="image break" id="image-4ti-6v0-5u8" src="http://qey2kbifv.hn-bkt.clouddn.com/2020-09-17/321kB58aQzZmPOiFcfMp.png" width="600" alt="域名信息"></li>                        </ul>                     </li>                  </ul>               </section>               <section class="section" id="section-pii-rq8-lqj">                  <h1 font-weigh = "bold">注意事项</h1>                  <ul class="ul" id="ul-shv-epy-aye">                     <li>关于域控： 
                        <ul class="ul" id="ul-fo6-mlq-41u">                           <li>如果您使用自定义镜像创建QVM实例部署域控制器，必须按照本文中的步骤修改SID。如果您使用公共镜像，已经默认修改了SID，无需手动修改。 </li>                           <li>七牛云不推荐您使用已有的域控制器创建自定义镜像来部署新的域控。如果必须使用，请注意新建实例的主机名（hostname）和创建自定义镜像之前实例的主机名必须保持一致<span>，否则可能会报错<code>服务器上的安全数据库没有此工作站信任关系</code></span>；您也可以在创建实例后修改成相同的主机名，解决此问题。
                           </li>                        </ul>                     </li>                     <li>关于客户端：七牛云不推荐您使用已加入域的客户端实例来创建自定义镜像<span>，否则新镜像创建的实例会报错<code>服务器上的安全数据库没有此工作站信任关系</code></span>。如果确实需要，建议您在创建新的自定义镜像前先退域。
