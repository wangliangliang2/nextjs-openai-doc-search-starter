目前七牛API调用审核接口有两个版本：v1版本、v3版本
我们推荐您使用v3版本，如果您目前在使用v1版本，建议您更换至v3版本

v1与v3版本区别
* v3版本是线上环境最新版本，v1是旧版本，不提供维护。内容审核存量/增量审核，目前都是用的v3版本
* v3与v1接口参数有所差异，比如 图片审核参数中审核类型，v3是params.scenes，v1是params.type
* v3版本审核结果准确度更高
