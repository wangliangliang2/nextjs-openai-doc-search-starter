您在使用内容审核API之前，需要做三个准备工作：

1.注册七牛云账号。如果您已有七牛云账号，请跳过此步骤。
2.获取需要进行内容审核的文件外链url。
3.生成七牛鉴权。如果您对七牛云 bucket 中的文件，采用`<fop>`的方式调用，请跳过此步骤。

# 1. 注册七牛云账号
请用户打开[七牛云官网](https://www.qiniu.com/)，点击注册，个人账号或企业账号均可。我们会在一个工作日内完成实名认证。注册成功后，登录[七牛云开发者平台](https://portal.qiniu.com/signin)。


# 2. 获取文件url

对于非七牛云 bucket的文件，请用户准备好文件的url。对于存储在七牛云 bucket 中的文件，请参考如下操作获取文件的外链url:

1.如果用户未将图片上传到七牛云 bucket，请根据[对象存储快速入门](https://developer.qiniu.com/kodo/manual/1233/console-quickstart)建立 bucket 并将图片上传。

2.获取单个文件 url：

点击“复制外链”获取选定图片 url

![](https://dn-odum9helk.qbox.me/FoTIhsjMdq0Br2OtVIVuwULhehHv)

3.批量获取文件 url 可通过[命令行工具](https://developer.qiniu.com/kodo/tools/1300/qrsctl)来完成。


# 3. 生成七牛鉴权
内容审核API支持如下方式调用：
* 基于API URL发起HTTP/HTTPS POST请求。采用该方式时，您首先需要获取七牛接口鉴权签名，根据 AccessKey 和 SuccessKey 生成鉴权，以便进行身份验证，方法详见[接口鉴权](https://developer.qiniu.com/dora/kb/3702/QiniuToken)。
