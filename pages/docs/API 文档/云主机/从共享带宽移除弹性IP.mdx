# 请求路径
> `PUT` /v1/vpc/common_bandwidth/:cb_id/remove_eip/:eip_id

# 请求参数

| 名称 | 类型 | 位置 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- | ----- |
| region_id | string | body | 是 | 地域 ID |
| cb_id | string | path | 是 | 共享带宽包 ID |
| eip_id | string | path | 是 | 弹性 IP 实例 ID, 要求和共享带宽包的地域一致 |


# 返回值

| 名称 | 类型 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- | ----- |
| request_id | string | 是 | 请求记录 ID |


# 示例
### 正常返回示例
```json
{
    "request_id": "04F0F334-AC1"
}
```
