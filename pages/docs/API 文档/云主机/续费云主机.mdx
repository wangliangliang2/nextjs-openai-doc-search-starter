### 请求示例
```json
{
    "region_id": "cn-shenzhen",
    "cost_charge_type": "PrePaid",
    "cost_period":1,
    "cost_period_unit":"Month",
    "client_token": "473469C7-AA6F-4DC5-B3DB-A3DC0DE3C83E"
}
```

# 返回值
| 名称 | 类型 | 是否必须 | 描述 |
| ----- | ----- | ----- | ----- |
| success | boolean | 否 | 是否操作成功 |
