# 请求路径
> `GET` /v1/cm/region


# 返回值
| 名称 | 类型 | 描述 |
| ----- | ----- | ----- | ----- |
| request_id | string | 请求 ID |
| data | \[]Region | 区域集合 |

### Region
| 名称 | 类型 | 描述 |
| ----- | ----- | ----- | ----- |
| region_id | string | 地域 ID |
| local_name | string | 地域名称 |
| zones | \[]Zone | 可用区集合 |

### Zone
| 名称 | 类型 | 描述 |
| ----- | ----- | ----- | ----- |
| region_id | string | 地域 ID |
| zone_id | string | 或用区 ID |
| local_name | string | 或用区名称 |
