# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string | 请求 ID
data | { monitor_data: { instance_monitor_data: []ECSMonitorDataItem } } | 

### ECSMonitorDataItem

字段 | 类型 | 描述 
----- | ----- | -----
bps_read | int | 实例系统盘读带宽 Byte/s
bps_write | int | 实例系统盘写带宽 Byte/s
iops_read | int | 实例系统盘 I/O 读操作 次/s
iops_write | int | 实例系统盘 I/O 写操作 次/s
intranet_rx | int | 实例在TimeStamp时刻的Period时间段内，接收的内网数据流量，kbits。
intranet_tx | int | 实例在TimeStamp时刻的Period时间段内，发送的内网数据流量，kbits。
intranet_bandwidth | int | 实例的公网带宽，单位时间内的网络流量，kbits/s
cpu | int | 实例vCPU的使用比例，单位：百分比（%）
