# 请求路径
> `PUT` /v1/vm/sgr/:security_group_id


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
security_group_id  | string | path | 是 | 安全组 ID
region_id         | string | body | 是 | 安全组所属地域ID。您可以调用 DescribeRegions 查看最新的七牛云地域列表。
security_group_name | string | body | 否 | 安全组名称
description | string | body | 否 | 安全组描述


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string | 请求 ID
