# 请求路径
> `POST` /v1/vm/disk

      <td>

      </td>
    </tr>
    <tr>
      <td>zone_id</td>

        <ul>
          <li>如果您不设置 InstanceId，则 ZoneId 为必填参数。</li>
          <li>您不能同时指定 ZoneId 和 InstanceId。</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>size</td>
      <td>integer</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>容量大小，以 GiB 为单位。指定该参数后，其取值必须≥指定快照 ID 的容量大小。取值范围：</p>
        <ul>
          <li>cloud：5~2000</li>
          <li>cloud_efficiency：20~32768</li>
          <li>cloud_ssd：20~32768</li>
          <li>cloud_essd：20~32768</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>disk_name</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>云盘名称。长度为 2~128 个英文或中文字符。必须以大小字母或中文开头，不能以 http:// 和 https:// 开头。可以包含数字、半角冒号（:）、下划线（_）或者连字符（-）。</p>
        <p>默认值：空</p>
      </td>
    </tr>
    <tr>
      <td>description</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>

        <p>默认值：空</p>
      </td>
    </tr>
    <tr>
      <td>encrypted</td>
      <td>boolean/td>
        <td>body</td>
      <td>否</td>
      <td>
        <p>是否加密云盘。默认值：false</p>
      </td>
    </tr>
    <tr>
      <td>disk_category</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>数据盘的云盘种类。取值范围：</p>
        <ul>
          <li>cloud：普通云盘</li>
          <li>cloud_efficiency：高效云盘</li>
          <li>cloud_ssd：SSD 云盘</li>
          <li>cloud_essd：ESSD 云盘
            默认值：cloud</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>performance_level</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>创建一块 ESSD 云盘时，设置云盘的性能等级。取值范围：</p>
        <ul>
          <li>PL1（默认）：单盘最高随机读写 IOPS 5 万</li>
          <li>PL2：单盘最高随机读写 IOPS 10 万</li>
          <li>PL3：单盘最高随机读写 IOPS 100 万
            有关如何选择 ESSD 性能等级，请参见 ESSD 云盘。</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>snapshot_id</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>创建云盘使用的快照。指定该参数后，Size 会被忽略，实际创建的云盘大小为指定快照的大小。2013年7月15日及以前的快照不能用来创建云盘。</p>
      </td>
    </tr>
    <tr>
      <td>instance_id</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>

        <ul>
          <li>设置实例 ID 后，会忽略您设置的 ResourceGroupId、Tag.N.Key、Tag.N.Value、ClientToken 和 KMSKeyId 参数。</li>
          <li>您不能同时指定 ZoneId 和 InstanceId。
            默认值：空，代表创建的是按量付费云盘，云盘所属地由 RegionId 和 ZoneId 确定。</li>
        </ul>

# 返回值
名称 | 类型 | 描述 |
----- | ----- | -----
request_id | string | 请求 ID
data | object{} | 返回数据
data.disk_id | string | 按量创建返回 ID
data.order | string | 预付费创建返回订单号
