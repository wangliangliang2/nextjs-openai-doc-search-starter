# 请求路径
> `GET` /v1/order/unpaid_orders


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
resource_type | string | query | 是 | 资源类型


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
data | \[]OrderInfo | 订单信息集合

### OrderDetail
名称 | 类型 | 描述 
----- | ----- | -----
id           | integer | ID
order        | string  | 订单号
fee          | float   | 订单原价
c_fee        | float   | 订单实际价格
create_time  | string  | 创建时间
update_time  | string  | 更新时间
pay_time     | string  | 支付时间
expired_time | string  | 订单过期时间
status       | integer | 订单状态 1: 未支付 2: 已支付 3:作废, 4: 垫付，5: 后付费
