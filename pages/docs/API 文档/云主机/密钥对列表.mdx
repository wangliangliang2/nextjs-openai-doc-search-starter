# 请求路径
> `GET` /v1/vm/keypair


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
region_id            | string  | query | 是 | 地域 ID。
key_pair_name        | string  | query | 否 | 密钥对名称
key_pair_finger_print| string  | query | 否 | 密钥对的指纹
page                 | integer | query | 否 | 页码。默认：1
size                 | integer | query | 否 | 分页大小，最大100。默认20


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
page       | integer | 页码
size       | integer | 分页大小
total      | integer | 密钥对总量
data       | \[]Keypair | 密钥对列表

### Keypair
名称 | 类型 | 描述 
----- | ----- | -----
id                    | string | 钥匙对 ID
key_pair_name         | string | 密钥对的名称
key_pair_finger_print | string | 密钥对的指纹
private_key_body      | string | 密钥对内容
public_key_body       | string | 公钥内容
created_at            | string | 创建时间
