# 请求路径
> `GET` /v1/vm/switch/:v_switch_id


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
v_switch_id | string | path  | 是 | 交换机 ID
region_id   | string | query | 是 | 地域 ID


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
data       | VSwitchInfo | 虚拟交换机详情

### VSwitchInfo
名称 | 类型 | 描述 
----- | ----- | -----
v_switch_id                | string  | 交换机的ID
vpc_id                     | string  | 交换机所属VPC的ID
status                     | string  | 交换机的状态，取值：Pending：配置中， Available：可用
cidr_block                 | string  | 交换机的IPv4网段
zone_id                    | string  | 交换机所属的可用区 ID
region_id                  | string  | 交换机所属的区域 ID
available_ip_address_count | integer | 交换机中可用的IP地址数量
description                | string  | 交换机的描述信息
v_switch_name              | string  | 交换机的名称
is_default                 | boolean | 是否为默认交换机，只能在创建的时候插入该字段
ipv6_cidr_block            | string  | 交换机的IPv6网段
created_at                 | string  | 创建时间
updated_at                 | string  | 更新时间
