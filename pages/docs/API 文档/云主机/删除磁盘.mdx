磁盘列表

# 请求路径
/v1/vm/disk/:disk_id

# HTTP Method
DELETE

# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
disk_id     | string | path  | 是 | 磁盘 ID
region_id   | string | query | 是 | 实例所属的地域 ID



# 返回值
| 名称 | 类型 | 是否必须 | 描述 |
| ----- | ----- | ----- | ----- |
| request_id | string | 是 | 请求 ID
