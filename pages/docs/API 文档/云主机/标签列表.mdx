# 请求路径
> `GET` /v1/tag/meta


# 请求参数
<table>
  <thead>
    <tr>
      <th>名称</th>
      <th>类型</th>
      <th>位置</th>
      <th>必须</th>
      <th>描述</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>with_detail</td>
      <td>bool</td>
      <td>query</td>
      <td>否</td>
      <td>
      是否返回标签详细信息，默认 false
      </td>
    </tr>
  </tbody>
</table>


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string | 请求 ID
data | \[]Tag | 标签列表

### Tag
<table>
  <thead>
    <tr>
      <th>名称</th>
      <th>类型</th>
      <th>描述</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>name</td>
      <td>string</td>
      <td>
        <p>标签名</p>
      </td>
    </tr>
    <tr>
      <td>description</td>
      <td>string</td>
      <td>
        <p>描述信息</p>
      </td>
    </tr>
    <tr>
      <td>resource_count</td>
      <td>int</td>
      <td>
        <p>with_detail == true 时会返回</p>
      </td>
    </tr>
  </tbody>
</table>


## 返回值示例
```json
{
  "data": [
    {
      "name": "proxy",
      "description": "proxy resources",
      "resource_count": 10
    }
  ],
  "request_id": "N8cGF3EvRnQOSFMW"
}
```
