# 请求路径
> `POST` /v1/vm/instance/:instance_id/security_group/leave


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
instance_id       | string | path | 是| 主机实例 ID
region_id         | string | body | 是 | 安全组所属地域ID。您可以调用 DescribeRegions 查看最新的七牛云地域列表。
security_group_id | string | body | 是| 安全组ID。您可以调用 DescribeSecurityGroups 查看您可用的安全组。


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string   | 请求 ID
