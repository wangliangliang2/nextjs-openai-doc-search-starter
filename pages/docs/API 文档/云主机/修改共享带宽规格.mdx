# 请求路径
> `PUT` /v1/vpc/common_bandwidth/:cb_id/modify

# 请求参数

| 名称 | 类型 | 位置 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- | ----- |
| region_id | string | body | 是 | 地域 ID |
| cb_id | string | path | 是 | 共享带宽包 ID |
| bandwidth | int | body | 是 | 新的带宽大小 |
| cost_charge_mode | string | body | 是 | 计费方式, PayByBandwidth 或 PayBy95 |

# 返回值

| 名称 | 类型 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- | ----- |
| data | OrderInfo | 否 | 订单详情 |
| request_id | string | 是 | 请求记录 ID |

### OrderInfo 订单详情
| 名称 | 类型 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- |
| order | string | 否 | 订单号, 选择预付费返回该参数 |
