# 请求路径
> `PUT` /v1/vm/instance/:id/spec

# 请求参数

| 名称 | 类型 | 位置 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- | ----- |
| region_id| string | body | 是 | 地域 ID |
| id | string | path | 是 | 主机实例 ID |
| instance_type | string | body | 是 | 新的主机规格，查规格参考文档: [查看实例规格详情列表](https://developer.qiniu.com/qvm/5594/openapi-ecs-describe-instance-type) |

# 返回值

| 名称 | 类型 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- | ----- |
| request_id | string | 是 | 请求记录 ID |

# 示例
### 正常返回示例
```json
{
    "request_id": "04F0F334-AC1"
}
```
