# 请求路径
> `GET` /v1/vm/image/:image_id


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
image_id  | string | path  | 是 | 镜像 ID
region_id | string | query | 是 | 实例所属的地域 ID


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string | 请求 ID
data | Image | 镜像详情
