# 请求路径
>`DELETE` /v1/security/bgpip/resource/:resource_id/level4rule_v2/delete

# 请求参数
| 名称 | 类型 | 位置 | 必须 | 描述 |
| ----- | ----- | ----- | ----- | ----- |
| X-Qiniu-Regionid | string | header | 是 | 地域 id, 取值：shanghai (大陆通用地域)|
| resource_id | string | path | 是 | 高防实例id |
| rule_id_list | [] string | body | 是 | 需要删除的规则id列表 |


# 返回值
| 名称 | 类型 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- |
| request_id | string | 是 | 请求记录 id |

<br>

# 示例
### Body示例
```json
{
    "rule_id_list": ["rule-1xxx"]
}
```

<br>

### 正常返回示例
```json
{
    "request_id": "04F0F334-AC1"
}
```
