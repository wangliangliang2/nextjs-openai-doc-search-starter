# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
data       | object{} | 创建 EIP 返回信息
data.allocation_id | string | 按量创建返回 ID
data.order | string | 预付费创建返回订单号



## 返回值示例
```json
{
  "data": {
    "order": "edc5413cc610305704a8aaddca"
  },
  "request_id": "N8cGF-xihhWd5VMW"
}
```

{
