# 请求路径
> `GET` /v1/tag/meta/:name


# 请求参数
<table>
  <thead>
    <tr>
      <th>名称</th>
      <th>类型</th>
      <th>位置</th>
      <th>必须</th>
      <th>描述</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>name</td>
      <td>string</td>
      <td>path</td>
      <td>是</td>
      <td>
      标签名称
      </td>
    </tr>
  </tbody>
</table>


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string | 请求 ID
data | TagInfo | 标签信息

  <thead>
    <tr>

      </td>
