# 请求路径
>`PATCH` /v1/security/bgpip/resource/:resource_id/elastic_limit


# 请求参数
| 名称 | 类型 | 位置 | 必须 | 描述 |
----- | ----- | ----- | ----- | -----
| X-Qiniu-Regionid | string | header | 是 | 地域 id, 取值：shanghai (大陆通用地域) |
| resource_id | string | path | 是 | 高防实例 id |
| elastic_limit | string | body | 是 | 弹性防护阈值，单位 Mbps, 取值[0 30000 40000 50000 60000 70000 80000 90000 100000 150000 200000 250000 300000],<br> 当取值为0时表示关闭弹性防护，否则取值需高于实例的保底防护带宽 |

<br>


# 返回值
| 名称 | 类型 | 是否必须 | <center>描述 |
| ----- | ----- | ----- | ----- |
| request_id | string | 是 | 请求记录 id |

<br>

# 示例
### Body参数示例
```json
{
	"elastic_limit": 30000
}
```

### 正常返回示例
```json
{
    "request_id": "04F0F334-AC1"
}
```
