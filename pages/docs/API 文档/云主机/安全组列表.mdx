# 请求路径
> `GET` /v1/vm/sgr


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
region_id           | string  | query | 否 | 地域 ID。
security_group_id   | string  | query | 否 | 安全组 ID
security_group_name | string  | query | 否 | 安全组名称
vpc_id              | string  | query | 否 | VPC ID
page                | integer | query | 否 | 页码。默认：1
size                | integer | query | 否 | 分页大小，最大100。默认20


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
page       | integer | 页码
size       | integer | 分页大小
total      | integer | 安全组总量
data       | \[]SecurityGroup | 安全组列表
