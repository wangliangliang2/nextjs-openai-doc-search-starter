# 请求路径
> `POST` /v1/vm/sgr/:security_group_id/ingress


# 请求参数
<table>
  <thead>
    <tr>
      <th>名称</th>
      <th>类型</th>
      <th>位置</th>
      <th>必须</th>
      <th>描述</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>security_group_id</td>
      <td>string</td>
      <td>path</td>
      <td>是</td>
      <td>
        <p>安全组 ID。您可以调用 DescribeSecurityGroups 查看您可用的安全组。</p>
      </td>
    </tr>
    <tr>
      <td>region_id</td>
      <td>string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>安全组所属地域 ID。您可以调用 DescribeRegions 查看最新的七牛云地域列表。</p>
      </td>
    </tr>
    <tr>
      <td>auth_type</td>
      <td>string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>授权类型，ip_cidr: IP地址或ip段，security_group_id: 安全组id, ipv6_cidr: ipv6地址</p>
      </td>
    </tr>
    <tr>
      <td>auth_list</td>
      <td>[]string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>授权列表，只能是ip地址段、ipv6地址或者安全组id。<br>支持一次性授权多个相同授权类型的值。</p>
      </td>
    </tr>
    <tr>
      <td>port_range_list</td>
      <td>[]string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>端口范围列表，支持一次性设置多个端口范围。
      <ul>
          <li>TCP/UDP 协议：取值范围为 1~65535。使用斜线（/）隔开起始端口和终止端口。例如：1/200</li>
          <li>ICMP 协议：-1/-1</li>
          <li>GRE 协议：-1/-1</li>
          <li>IpProtocol 取值为 all：-1/-1</li>
        </ul>
</p>
      </td>
    </tr>
    <tr>
      <td>ip_protocol</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>传输层协议。取值大小写敏感。取值范围：</p>
        <ul>
          <li>tcp</li>
          <li>udp</li>
          <li>icmp: 仅支持 IPv4 地址。</li>
          <li>gre</li>
          <li>all：支持所有协议</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>policy</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>设置访问权限。取值范围：</p>
        <ul>
          <li>accept（默认）：接受访问。</li>
          <li>drop：拒绝访问，不返回拒绝信息，表现为发起端请求超时或者无法建立连接的类似信息。</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>priority</td>
      <td>integer</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>安全组规则优先级，数字越小，代表优先级越高。取值范围：1~100 默认值：1</p>
      </td>
    </tr>
    <tr>
      <td>description</td>
      <td>string</td>
      <td>body</td>
      <td>否</td>
      <td>
        <p>安全组规则的描述信息。长度为 1~512 个字符</p>
      </td>
    </tr>
  </tbody>
</table>


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
