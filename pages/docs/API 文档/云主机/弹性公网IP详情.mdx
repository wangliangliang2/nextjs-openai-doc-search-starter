# 请求路径
> `GET` /v1/vm/eip/:allocation_id


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
allocation_id | string | path  | 是 | 弹性公网 IP 的 ID
region_id     | string | query | 是 | 地域 ID


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
data       | EIPInfo | EIP 列表
