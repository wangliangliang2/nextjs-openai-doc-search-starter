# 请求路径
> `GET` /v1/vm/snapshot

# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
total | integer | 总数
page | integer | 页码
size | integer | 分页大小
request_id | string | 请求 ID
data | \[]Snapshot | 快照列表
