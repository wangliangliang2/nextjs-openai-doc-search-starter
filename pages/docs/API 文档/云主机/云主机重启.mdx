# 请求路径
> `POST` /v1/vm/instance/:instance_id/reboot


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述
----- | ----- | ----- | ----- | -----
instance_id | string  | path  | 是 | 云主机实例ID 
region_id   | string  | body  | 是 | 实例所属的地域 ID
force_stop  | boolean | body  | 否 | 重启ECS实例前，是否强制关机。<br/>取值范围：<ul><li>`true`：强制关机。相当于典型的断电操作，所有未写入存储设备的缓存数据会丢失。</li><li>`false`（默认）：正常关机。</li></ul>


# 返回值
重启成功 http 状态码为200

| 名称 | 类型 | 描述 |
| ----- | ----- | ----- |
| request_id | string | 请求 ID |
