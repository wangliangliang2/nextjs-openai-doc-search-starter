# 请求路径
> `POST` /v1/vm/eip/:allocation_id/associate


# 请求参数
<table>
  <thead>
    <tr>
      <th>名称</th>
      <th>类型</th>
      <th>位置</th>
      <th>必须</th>
      <th>描述</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>allocation_id</td>
      <td>string</td>
      <td>path</td>
      <td>是</td>
      <td>
        <p>绑定云产品实例的 EIP 的 ID。</p>
      </td>
    </tr>
    <tr>
      <td>region_id</td>
      <td>string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>要绑定云产品实例的 EIP 所属的地域 ID。</p>
      </td>
    </tr>
    <tr>
      <td>instance_type</td>
      <td>string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>要绑定 EIP 的实例的类型，取值：</p>
        <ul>
          <li><code>Nat</code>：NAT 网关。</li>
          <li><code>SlbInstance</code>：负载均衡 SLB。</li>
          <li><code>EcsInstance</code>：云服务器 ECS。</li>
          <li><code>NetworkInterface</code>：辅助弹性网卡。</li>
          <li><code>HaVip</code>：高可用虚拟 IP。</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>instance_id</td>
      <td>string</td>
      <td>body</td>
      <td>是</td>
      <td>
        <p>要绑定 EIP 的实例 ID。</p>
        <p>支持输入 NAT 网关实例 ID、负载均衡 SLB 实例 ID、云服务器 ECS 实例 ID、辅助弹性网卡实例 ID、高可用虚拟 IP 实例 ID。</p>
      </td>
    </tr>
  </tbody>
</table>


# 返回值
名称 | 类型 | 描述 
----- | ----- | -----
request_id | string  | 请求 ID
