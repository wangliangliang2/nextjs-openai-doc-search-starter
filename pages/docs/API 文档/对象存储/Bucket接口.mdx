空间管理相关的接口：

| 接口   | 说明 |
| -----|-------------------|
|[mkbucketv3](/kodo/api/1382/mkbucketv3)        | 创建新的存储空间|
|[drop](/kodo/api/1601/drop-bucket)        | 删除指定存储空间|
|[domains](/kodo/api/3949/get-the-bucket-space-domain)        | 获取一个空间绑定的所有域名列表|
|[image](/kodo/api/3966/bucket-image-source) | 为存储空间指定一个镜像回源网址，用于取回资源|
|[private](/kodo/api/3946/set-bucket-private)| 设置空间访问权限|
|[putBucketTagging](/kodo/api/6314/put-bucket-tagging)        | 设置空间标签|
|[getBucketTagging](/kodo/api/6315/get-bucket-tagging)        | 查询空间标签|
|[deleteBucketTagging](/kodo/api/6316/delete-bucket-tagging)        | 删除空间标签|
