Multipart Upload相关的接口：

接口   | 说明 
 -----|-------------------
[initiateMultipartUpload](/kodo/api/6365/initialize-multipartupload)        | 通知服务端开启分块上传任务,得到全局唯一任务 UploadId
[uploadPart](/kodo/api/6366/upload-part)        | 分块上传数据，需指定的任务 UploadId
[completeMultipartUpload](/kodo/api/6368/complete-multipart-upload)        | 完成整个文件的分块上传，需指定的任务 UploadId
[abortMultipartUpload](/kodo/api/6367/abort-multipart-upload) | 中止分块上传任务，并且删除已经上传的块，需指定的任务 UploadId
[listParts](/kodo/api/6858/listparts)|列举指定 UploadId 所属的所有已经上传成功 Part
