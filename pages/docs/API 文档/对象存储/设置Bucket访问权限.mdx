<a id="private-description"></a>
# **描述** 

本接口用于设置 Bucket 访问权限。目前 Bucket 有两种访问权限：公开和私有。创建 Bucket 时默认为公开空间。

<br>

<a id="private-requests"></a>
# **请求**
<a id="private-requests-syntax"></a>

### **请求语法**

```
POST /private?bucket=<BucketName>&private=<Private> HTTP/1.1
Host: uc.qiniuapi.com
Content-Type: application/x-www-form-urlencoded
X-Qiniu-Date: 20060102T150405Z
Authorization: Qiniu <AccessToken>
```

<br>

<a id="private-request-parameters"></a>
### **请求参数**
| 参数名称 | 必填 | 说明 |
| --- | --- | --- |
| bucket | 是 | 空间名称|
| private | 是 |  <li>`0` 公开</li> <li>`1` 私有</li>|

<br>

### **请求头**
该请求操作的实现使用了所有操作的公共请求头。有关详细信息，请查阅[公共请求头](/kodo/api/3924/common-request-headers)。

<br>

### **请求内容**
该请求操作的请求体为空。
