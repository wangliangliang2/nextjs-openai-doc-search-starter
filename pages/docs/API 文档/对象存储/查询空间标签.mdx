<a id="getbucketTagging-description"></a>
# **描述** 

本接口用于查询指定空间已设置的标签信息。

<br>

<a id="getbucketTagging-requests"></a>
# **请求**
<a id="getbucketTagging-requests-syntax"></a>
### **请求语法**

```
GET /bucketTagging?bucket=<BucketName> HTTP/1.1
Host: uc.qiniuapi.com
Content-Type: application/x-www-form-urlencoded
X-Qiniu-Date: 20060102T150405Z
Authorization: Qiniu <AccessToken>
```


<br>

<a id="getbucketTagging-request-parameters"></a>
### **请求参数**

| 参数名称 | 必填 | 说明 |
| --- | --- | --- |
| BucketName | 是 | 空间名称|


<br>

<a id="getbucketTagging-request-headers"></a>
### **请求头**
该请求操作的实现使用了所有操作的公共请求头。有关详细信息，请查阅[公共请求头](/kodo/api/3924/common-request-headers)。

<br>

<a id="getbucketTagging-request-elements"></a>
### **请求内容**
该请求操作的请求体为空。
