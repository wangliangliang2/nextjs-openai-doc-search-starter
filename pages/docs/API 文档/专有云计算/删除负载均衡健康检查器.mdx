# 请求路径
> `DELETE` /api/rio/v1/network/loadbalancer_health/:health_id

# 请求参数
名称 | 类型 | 位置 | 必须 | 描述 
----- | ----- | ----- | ----- | ----- 
x-rio-zone-id | string | header | 是 | 地域 ID
health_id       | string | path   | 是 | 负载均衡健康检查器 ID

# 返回值
名称 | 类型 |  描述
----- | ----- | -----
request_id	| string  | 请求 ID
