# 请求路径
> `PUT` /api/rio/v1/network/loadbalancer_certificate/:cert_id

# 请求参数
名称 | 类型 | 位置 | 必须 | 描述 
----- | ----- | ----- | ----- | ----- 
x-rio-zone-id           | string | header | 是 | 地域 ID
cert_id                 | string | path | 是 | 证书 ID
server_certificate_name | string | body | 是 | 证书名称

# 返回值
名称 | 类型 |  描述
----- | ----- | -----
request_id	| string  | 请求 ID

## 请求值示例
```json
{
    "server_certificate_name":"modify_cert_name"
}
```

## 返回值示例
```json
{
    "request_id":"HtPfEgzO4NVWyskW"
}
```
