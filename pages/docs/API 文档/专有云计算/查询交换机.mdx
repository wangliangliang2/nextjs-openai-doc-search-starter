# 请求路径
> `GET` /api/rio/v1/network/vpc/:id/vswitch

# 请求参数
名称 | 类型 | 位置 | 必须 | 描述 
----- | ----- | ----- | ----- | ----- 
x-rio-zone-id              | string | header | 是 | 地域 ID
id             | string | path | 是 | 专有网络 ID
vswitch_ids     | []string | query | 否 | 要查询的交换机 ID
marker           | string | query | 否 | 查询起始标记
limit            | int | query | 否 | 查询数量

# 返回值
名称 | 类型 |  描述
----- | ----- | -----
request_id	| string  | 请求 ID
Data    |  []VSwitchInfo | 交换机信息

## VSwitchInfo
名称 | 类型 |  描述
----- | ----- | -----
zone_id	| string  | 地域 ID
vswitch_id    | string | 交换机 ID
vpc_id        | string | 专有网络 ID
status    | string | 交换机 状态，Pending: 等待中，Available: 可用
cidr    | string | 交换机网段
vswitch_name    | string | 交换机名称
description    | string | 交换机描述
available_ip_count    | int | 交换机中可用的IP地址数量
resource_count    | int | 资源数量
created_at    | big int | 创建时间戳

## 请求示例
> `GET` /api/rio/v1/network/vpc/:id/vswitch
