# 请求路径
> `GET` /api/rio/v1/network/eip

# 请求参数
名称 | 类型 | 位置 | 必须 | 描述 
----- | ----- | ----- | ----- | ----- 
x-rio-zone-id              | string | header | 是 | 地域 ID
marker                | string | query | 否 | list marker
limit                 | int    | query | 否 | 主机限制数量
status                | string | query | 否 | EIP状态 过滤： <ul><li>`Creating` 创建中 </li><li>`Associating` 关联中</li><li>`Unassociating` 解除关联中 </li><li>`InUse` 使用中 </li><li>`Available`  可用状态 </li></ul>
server_id            | string | query | 否 | 实例 ID
eip_name            | string | query | 否 | 弹性公网 名称
eip_id            | string | query | 否 | 弹性公网 ID
eip_address           | string | query | 否 | 弹性公网 地址

# 返回值

| 名称       | 类型       | 描述         |
| ---------- | ---------- | ------------ |
| request_id | string     | 请求 ID      |
| data       | []EIPInfo | 弹性公网信息列表 |

## EIPInfo 磁盘信息

| 名称               | 类型       | 描述                                                                                                                                                                                                                                                                                                                                                |
| ------------------ | ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| zone_id            | string     | 地域 ID                                                                                                                                                                                                                                                                                                                                             |
| server_id            | string     | 主机 ID                                                                                                                                                                                                                                                                                                                                             |
| server_name          | string    | 主机 名称                                                                                                                                                                                                                                                                                                                                            |
| eip_id      | string     | 弹性公网 ID                                                                                                                                                                                                                                                                      |
| eip_name          | string     | 弹性公网 名称                                                                                                                                                                                                                                                                               |
| eip_address             | string     | 弹性公网 地址                                                                                                                                                                                                                                                                                                                                          |
| bandwidth               | uint    | 带宽                                                                                                                                                                                                                                                                                                                                            |
| status                | string | query | 否 | EIP状态 过滤： <ul><li>`Creating` 创建中 </li><li>`Associating` 关联中</li><li>`Unassociating` 解除关联中 </li><li>`InUse` 使用中 </li><li>`Available`  可用状态 </li></ul>|
| attach_instance_type             | AttachInstanceType | 绑定实例类型                                                                                                                                                                                                                                                                                                                                           |
| created_at           | int64     | 创建时间                                                                                                                                                                                                                                                                                                                                            |
| updated_at         | int64     | 更新时间                                                                                                                                                                                                                                                                                                                                            |
