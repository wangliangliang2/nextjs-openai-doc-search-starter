# 请求路径
> `GET` /api/rio/v1/compute/warn/:id

# 请求参数
名称 | 类型 | 位置 | 必须 | 描述 
----- | ----- | ----- | ----- | ----- 
x-rio-zone-id   | string | header| 是 | 地域 ID
id              | string | path | 是 | 告警组 ID

# 返回值
名称 | 类型 |  描述
----- | ----- | -----
request_id	| string  | 请求 ID
data        | Warn    | 告警组详情

## Warn 告警组详情
名称 | 类型 |  描述
----- | ----- | -----
name         | string           | 名称
warn_id      | string           | ID
status       | string           | 状态：<ul><li>`OK`：正常</li><li>`INSUFFICIENT_DATA`：数据不足</li><li>`ALARM`：告警中</li></ul>
alert_user   | string           | 告警联系人
phone_list   | []string         | 电话号
email_list   | []string         | 邮箱
enable       | string           | 启用状态：<ul><li>`1`：启用</li><li>`0`：禁用</li></ul>
rules        | []WarnRule       | 告警规则
instances    | []WarnInstance   | 告警实例

### WarnRule 告警规则
名称 | 类型 |  描述
----- | ----- | -----
rule_type               | string  | 指标：<ul><li>`CPUUtilization`：CPU使用率，单位：%</li><li>`disk_readbytes`：磁盘每秒读取的字节数，单位：MB/s</li><li>`disk_readiops`：磁盘每秒的读请求数量，单位：次/s</li><li>`disk_writebytes`：磁盘每秒写入的字节数，单位：MB/s</li><li>`disk_writeiops`：磁盘每秒的写请求数量，单位：次/s</li><li>`diskusage_utilization`：磁盘使用率，单位：%</li><li>`memory_usedutilization`：内存使用率，单位：%</li><li>`networkin_errorpackages`：设备驱动器检测到的发送错误包的数量，单位：个/s</li><li>`networkin_packages`：网卡每秒发送的数据包数，单位：个/s</li><li>`networkin_rate`：网卡的上行带宽，单位：M/s</li><li>`networkout_errorpackages`：设备驱动器检测到的接受错误包的数量，单位：个/s</li><li>`networkout_rate`：网卡的下行带宽，单位：M/s</li><li>`netin_dropped_packet_rate`：网络流入丢包率，单位：个/s</li><li>`netout_dropped_packet_rate`：网络流出丢包率，单位：个/s</li><li>`networkout_packages`：网络包流入速率，单位：个/s</li><li>`diskio_read_delay`：磁盘IO读延迟，单位：ms</li><li>`diskio_write_delay`：磁盘IO写延迟，单位：ms</li></ul>
rule_flag               | string  | 告警开启状态：<ul><li>`1`：启用</li><li>`0`：禁用</li></ul>
medium_threshold        | integer | 中度阈值
seriousness_threshold   | integer | 严重阈值
time_interval           | integer | 时间间隔，单位：s，默认 60s
statistics              | string  | 告警数据统计类型，默认`max`：<ul><li>`max`：最大值</li><li>`min`：最小值</li><li>`avg`：平均值</li></ul>
comparison_operator     | string  | 比较规则，默认`>=`：<ul><li>`>`：大于</li><li>`>=`：大于等于</li><li>`!=`：不等于</li><li>`<`：小于</li><li>`<=`：小于等于</li></ul>


### WarnInstance 告警实例
名称 | 类型 |  描述
----- | ----- | -----
instance_id     | string | 云主机 ID
instance_name   | string | 云主机 Name
instance_type   | string | 资源类型：<ul><li>`cloud`：云主机</li></ul>
