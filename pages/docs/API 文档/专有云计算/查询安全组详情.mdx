# 请求路径

> `GET` /api/rio/v1/compute/security_group/:id/detail

# 请求参数

| 名称          | 类型     | 位置   | 必须 | 描述                                                                                          |
| ------------- | -------- | ------ | ---- | --------------------------------------------------------------------------------------------- |
| x-rio-zone-id | string   | header | 是   | 地域 ID                                                                                       |
| id            | string   | path   | 是   | 安全组 ID                                                                                     |
| direction     | []string | query  | 否   | 流量方向：<ul><li>`all` : 全部</li><li>`ingress` : 入方向</li><li>`egress` : 出方向</li></ul> |

# 返回值

| 名称       | 类型          | 描述       |
| ---------- | ------------- | ---------- |
| request_id | string        | 请求 ID    |
| data       | SecurityGroup | 安全组对象 |

## SecurityGroup 安全组对象

| 名称                 | 类型                | 描述           |
| -------------------- | ------------------- | -------------- |
| zone_id              | string              | 地域 ID        |
| security_group_id    | string              | 安全组 ID      |
| security_group_name  | string              | 安全组名称     |
| security_group_rules | []SecurityGroupRule | 安全组规则列表 |
| description          | string              | 安全组描述信息 |

## SecurityGroupRule 安全组规则对象

| 名称                     | 类型   | 描述                                                                                                                          |
| ------------------------ | ------ | ----------------------------------------------------------------------------------------------------------------------------- |
| security_group_rule_id   | string | 安全组规则 ID                                                                                                                 |
| protocol                 | string | IP 协议类型：<ul><li>`all` : 全部</li><li>`tcp` : TCP</li><li>`udp` : UDP</li><li>`icmp` : ICMP</li><li>`gre` : GRE</li></ul> |
| direction                | string | 流量方向：<ul><li>`all` : 全部</li><li>`ingress` : 入方向</li><li>`egress` : 出方向</li></ul>                                 |
| remote_security_group_id | string | 远端安全组 ID                                                                                                                 |
| from_port                | integer    | 最小端口号                                                                                                                    |
| to_port                  | integer    | 最大端口号                                                                                                                    |
| cidr                     | string | 安全组描述信息                                                                                                                |
| type                     | string | 安全组授权类型：<ul><li>`1` : 安全组</li><li>`2` : Ipv4 地址</li></ul>                                                        |
| created_at               | integer  | 安全组规则创建时间                                                                                                            |
