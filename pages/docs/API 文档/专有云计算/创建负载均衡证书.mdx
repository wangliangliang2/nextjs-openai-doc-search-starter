# 请求路径
> `POST` /api/rio/v1/network/loadbalancer_certificate


# 请求参数
名称 | 类型 | 位置 | 必须 | 描述 
----- | ----- | ----- | ----- | ----- 
x-rio-zone-id           | string | header| 是 | 地域 ID
server_certificate_name		| string 	| body | 是 | 证书名称
server_certificate			| string 	| body | 是 | 公钥证书
private_key					| string 	| body | 是 | 私钥证书


# 返回值
名称 | 类型 |  描述
----- | ----- | -----
request_id	| string             | 请求 ID
