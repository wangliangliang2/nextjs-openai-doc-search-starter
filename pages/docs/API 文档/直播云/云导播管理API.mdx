## 简介

云导播管理API，对应云导播的增删改查(CRUD)，使用七牛账户密钥进行鉴权(鉴权规则参考[HTTP请求鉴权](/pili/api/2772))

## API列表

- [创建云导播 POST /v1/casters](/pili/api/4709)
- [更新云导播 POST /v1/casters/<CasterID\>](/pili/api/4711)
- [删除云导播 DELETE /v1/casters/<CasterID\>](/pili/api/4712)
- [开启云导播 POST /v1/casters/<CasterID\>/start](/pili/api/4713)
- [关闭云导播 POST /v1/casters/<CasterID\>/stop](/pili/api/4714)
- [云导播列表 GET /v1/casters](/pili/api/4715)
- [云导播信息 GET /v1/casters/<CasterID\>](/pili/api/4716)
