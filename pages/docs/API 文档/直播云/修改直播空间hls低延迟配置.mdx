# 接口描述

对目标直播空间下的直播流开启或关闭hls低延迟

# 请求参数

```
POST /v2/hubs/<hub>/hlsplus
Host: pili.qiniuapi.com
Authorization: <QiniuToken>  
Content-Type: application/json 

{
    "hlsPlus":<true>
}
```

|      参数         |      类型          |  必填   |                  描述                      |
| -------------------- | -------------------- | ----------- | ------------------------------------------ |
|       Hub           |    string           | 是    |        直播空间名                                    |
|   HlsPlus           |      bool           | 是   |         是否开启 hls 低延迟                   |

# 返回数据

空

# 示例

请求体:

```
POST /v2/hubs/testhub/hlsplus HTTP/1.1
Host: pili.qiniuapi.com
Authorization: <QiniuToken>  
Content-Type: application/json 

{
    "hlsPlus":true
}
```

返回体:

```
200 OK
```
