

# 请求包
**备注：如果是海外的pub转推任务，请将Host改为：pili-hw-pub.qiniuapi.com**

```
GET /tasks?marker=123&limit=123&name=abc
Host: pili-pub.qiniuapi.com
Authorization: <QiniuToken>

```

| 参数名称 | 必填 | 说明 |
| ----------- |---------------------|------------------------------------------ |
| marker | 否 | 字符串，定位标识。 |
| limit | 否 | 整型，请求数目，上限1000。 |
| name | 否 | 字符串，任务名称和任务描述模糊匹配。 |
