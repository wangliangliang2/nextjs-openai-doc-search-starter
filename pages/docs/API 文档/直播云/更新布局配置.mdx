# 请求包
```
POST /v1/static/casters/<CasterID>/layouts
Host: pili-caster.qiniuapi.com
Authorization: <StaticKey>
Content-Type: application/json
{
  "layout": 0,
  "title": "",
  "overlay": {},
  "text": {}
}
```

# 请求参数

参数名称 | 必填 | 字段类型 | 取值范围 | 说明
 ---|---|---|---|---
CasterID | 是 | string | - | 云导播唯一ID
layout | 否 | int | [0,7] | 布局ID
title | 否 | string | - | 布局标题，可留空
overlay | 否 | object | - | 画中画配置，key为对应的监视器频道，详细参数见[画中画配置(overlay)详细参数](/pili/api/4735#overlay)
text | 否 | object | - | 文字水印配置，key为文字水印标题，详细参数见[文字水印配置(text)详细参数](/pili/api/4735#text)


# 返回包
```
200 OK
```


# 返回参数

- 详细参数说明见[云导播参数说明](/pili/api/4735#caster)
