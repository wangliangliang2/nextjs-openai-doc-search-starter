    <div><p>rtc-pub 允许用户将视频文件转推到房间内，供房间内的其他用户订阅。</p>
<h2 id="接口鉴权"><a href="#%E6%8E%A5%E5%8F%A3%E9%89%B4%E6%9D%83" ariaHidden="true" tabIndex="-1"><span className="icon,icon-link"></span></a>接口鉴权</h2>
<p>与开启房间相同，需要在路径上带上token=<roomtoken>。room token 必须是合法的房间token，并具有管理员权限。</roomtoken></p>
<h2 id="限制条件"><a href="#%E9%99%90%E5%88%B6%E6%9D%A1%E4%BB%B6" ariaHidden="true" tabIndex="-1"><span className="icon,icon-link"></span></a>限制条件</h2>
<p>目前转推的文件必须使用 H264 视频编码及 AAC 音频编码（采样率为44100Hz），flv封装。不符合条件的文件在转推时可能出现无法观看、音视频与源文件有较大差异等异常，可转码后再使用转推服务。</p>
<h2 id="建立转推任务"><a href="#%E5%BB%BA%E7%AB%8B%E8%BD%AC%E6%8E%A8%E4%BB%BB%E5%8A%A1" ariaHidden="true" tabIndex="-1"><span className="icon,icon-link"></span></a>建立转推任务</h2>
