# 创建模板 (POST Template)

## 请求定义

```
POST /v1/template HTTP/1.1
Host: sms.qiniuapi.com
Authorization: <QiniuToken>
Content-Type: application/json
```

### 请求头

> 请参考 [请求认证](/sms/api/5842/sms-api-authentication)


### 请求数据

```json
{
    "name": string,
    "template": string,
    "type": string,
    "description": string,
    "signature_id": string,
    "is_oversea": bool
}
```

参数名称 | 类型 | 必填 | 说明
--------|------|------|------
name | string | 是 | 模板名称
template | string |是 | 模板内容 可设置自定义变量，发送短信时候使用，参考:${code}
type | string | 是 | 模板类型，取值范围为: `notification` (通知类短信), `verification` (验证码短信), `marketing` (营销类短信), `voice`(语音类短信)
description | string | 是 | 申请理由简述
signature_id| string | 是 | 已经审核通过的签名
is_oversea | bool | 否 | 是否是 国际/港澳台 模版

-----|-----
