# 查询签名 (GET Signature)

## 请求定义

```
GET /v1/signature HTTP/1.1
Host: sms.qiniuapi.com
Authorization: <QiniuToken>
Content-Type: application/x-www-form-urlencoded
```

### 请求头

> 请参考 [请求认证](/sms/api/5842/sms-api-authentication)

### 请求参数

该接口定义以下请求参数:

名称 | 必填 | 描述
---- | ---- | ----
audit_status | 否 | 审核状态, 取值范围为: "passed"(通过), "rejected"(未通过), "reviewing"(审核中)
page | 否 | 页码。默认为 1
page_size | 否 | 分页大小。默认为 20


## 响应定义

### 该接口正确处理请求时返回如下 JSON 数据

```json
{
	"items": [{
		"id": string,
		"signature": string,
		"source": string,
		"audit_status": string,
		"reject_reason": string,
		"created_at": int64,
		"updated_at": int64
	}...],
	"total": int,
	"page": int,
	"page_size": int,
}
```
