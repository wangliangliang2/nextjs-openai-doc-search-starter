# API 错误码

### 数据格式

当错误发生时，Qiniu SMS Service 服务返回如下 JSON 数据:

```json
{
	"error": "NoSuchObject",
	"message": "The specified object does not exist.",
	"request_id": "uZyOLgh5jN-_DJ8V"
}
```

- HTTP Status Code: 3xx, 4xx or 5xx
- Content-Type: application/json

### JSON 数据字段定义如下

名称 | 类型 | 描述
---- | ---- | ----
error | string | 错误码
message | string | 错误信息
request_id | string | 错误对应的请求唯一 ID，在您提交工单时，请提供 `request_id`

### 错误列表
错误码 | 错误信息描述
---------|-------
ServerInternalError | 发生了内部错误。重试您的请求，但如果问题仍然存在，请[提交工单](https://support.qiniu.com/tickets/new/)。
InvalidQueryValue | 无效查询。
MalformedJSON | JSON 格式不正确。
NotFoundError | 未找到.
FreezedUserError | 您的账户被冻结了。请联系您的销售。
InactivatedUseError | 您的帐户没有开通短信服务，请[提交工单](https://support.qiniu.com/tickets/new/)或者联系您的销售。
AccountNeedCertified | 您的帐户需要认证。
AccountBalanceNotEnough | 您的账户余额不足。
