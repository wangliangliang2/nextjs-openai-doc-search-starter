**视频监控（qvs）API 返回结果说明**

调用 API 服务后返回数据采用统一格式：

返回的 HTTP 状态码为 2xx，代表调用成功。 调用成功返回的数据格式为json。
返回的 HTTP 状态码为 4xx、5xx或6xx，代表调用失败。错误状态下会返回详细的错误信息：
```json
 {
  "code": <errorCode>,
  "error": "<errorInfo>"
}
```
