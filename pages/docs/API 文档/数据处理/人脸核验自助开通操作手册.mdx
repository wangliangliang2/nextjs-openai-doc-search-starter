
# 1 开通入口

（1）点此链接立即访问：https://portal.qiniu.com/faceid/sdk-auth
（2）官网页，人脸核验产品，点击【立即使用】

![](https://dn-odum9helk.qbox.me/FvjwtQeXiHES8N_E7k8aDPDWQ-A6)

（3）控制台左上角选择人脸核验产品

![](https://dn-odum9helk.qbox.me/FmFjO4zL11vRie6VF3-kl8vQOud0)

# 2 开通步骤一：开通方式选择

授权列表页点击【新增授权】

（1）如您的业务需使用纯API调用的方式，请勾选【纯API调用】点击下一步，可开通使用人脸核验API的相关服务。

（2）如您的业务需要嵌入人脸核验相关服务的SDK，请勾选【获取SDK授权】点击下一步，我们更推荐使用SDK的方式。
 ![](https://dn-odum9helk.qbox.me/FslKTpGnmFsN6R1zeqAHZS2wYl_u)
 
# 3 开通步骤二：SDK授权

授权类型：SDK的授权类型，可选动作活体或者光线活体
生效时间：默认立即生效
应用名称：填写应用名称
应用场景：根据应用的实际场景填写
系统平台：请根据应用的实际平台进行选择
应用包名：请填写应用的实际包名，注意此处包名填写完成之后不能修改，请谨慎填写
服务协议：请阅读并勾选服务协议
填写完成上述信息，点击下一步

![](https://dn-odum9helk.qbox.me/FswGRS0UMerJ6lTrsZ315mzjLx4s)

# 4 开通步骤三：授权成功
在本页面可下载SDK，查看API调用文档，以及查看按量付费的具体价格，或者购买资源包
【返回列表】按钮回到已授权应用的授权列表页
![](https://dn-odum9helk.qbox.me/Fj91vxmcQoyPhSZRR7Z9-xcEe82e)

# 5 授权列表页面说明
点击控制台的人脸核验产品，默认进入授权列表页面。
（1）在本页面可查看已授权的app列表的具体信息，操作下载SDK，删除SDK的授权信息。
（2）右上角点击【按量付费】查看计量价格，
（3）点击【购买资源包】可购买人脸核验相关资源包
（4）【使用文档】可查看API调用文档
（5）【Demo下载】可下载Demo体验人脸核验的相关功能。
（6）点击【新增授权】进入新增授权页。

![](https://dn-odum9helk.qbox.me/FvqtNlE3CinLI9w9nTJbslnLlxh8)
