# **简介**
imageAve接口用于计算一幅图片的平均色调，并以`0xRRGGBB`形式返回。  



## **限制说明**

- 同步处理 
  - 原图支持 20MB 以内的图片
  - 处理前的图片`w`和`h`参数不能超过10000像素

- 持久化处理
  - 图片文件无大小限制
  - 处理前的图片`w`和`h`参数不能超过10000像素

<a id="imageAve-request"></a>

# 接口规格

`imageAve`



# **示例**

在Web浏览器中输入以下图片地址：

```
https://o6nalx2hr.qnssl.com/0.jpg?imageAve
```

返回结果（内容经过格式化以便阅读）：

```
{
    "RGB": "0x85694d"
}
```
