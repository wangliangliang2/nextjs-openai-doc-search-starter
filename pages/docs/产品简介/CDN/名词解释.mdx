# CNAME域名

在七牛云Portal创建加速域名后，会给域名分配一个 “CNAME域名”（例：example.qiniudns.com）。用户需要在域名服务商处配置一条主机记录为 example 的 CNAME 记录类型，并将需要记录值指向 example.qiniudns.com 。
生效后域名就会正式指向七牛云 CDN ，该域名所有的请求都开始享有CDN加速效果。

# DNS

即 Domain Name System ，域名解析服务。把域名转换成为网络可以识别的 IP 地址（人们习惯记忆域名，但机器互相只认 IP ），能够使人更方便地访问互联网，它们之间的转换工作称为域名解析。

# 边缘节点

距离最终用户最近的 CDN 节点，提高响应速度和连接速度，提高用户浏览网页视频的体验，边缘节点的主要工作是将访问量较大的网页内容和音视频等文件保存在服务器前端的专用缓存设备上。
