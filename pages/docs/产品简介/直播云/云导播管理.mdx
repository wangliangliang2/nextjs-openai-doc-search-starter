#  <span id="tutorial">创建导播台</span>
1. 登录直播云控制台，选择**云导播**

<img src="https://dn-odum9helk.qbox.me/FtUwNKRV7J8TNv9hCrvPuBsN8qaY" width = "90%" height = "90%" align=center />

2. 点击**创建导播台**，新创建的导播台将默认为开启状态，开启时长为1小时
3. 输入导播台名称，静态鉴权Key，选择画布尺寸，提供480P，720P，1080P三种尺寸

<img src="https://dn-odum9helk.qbox.me/Fp5UvfwIKV58DkDbL_auOf1r09Ac" width = "90%" height = "90%" align=center />

#  <span id="tutorial1">修改导播台信息</span>
1. 登录直播云控制台，选择**云导播**
2. 点击 **导播台名称**
可设置的字段包括：
  * 静态鉴权Key：静态鉴权 Key 是用来登录相应导播台的唯一密码
  * 临时令牌：可生成一个MD5加密令牌，用于导播台的加密登录，令牌将会在设置时间之后失效
  * 存储空间和访问域名：配置资源存储信息后，可在终端导播台直接上传本地静态资源

#  <span id="tutorial2">删除导播台</span>
如果某个导播台不再需要使用，可以通过**删除**完成导播台的删除操作。删除为不可逆操作，请谨慎
1. 登录直播云控制台，选择**云导播**
2. 点击**导播台名称**
3. 点击**删除**，输入导播台ID
4. 输入控制台登录密码后完成删除

<img src="https://dn-odum9helk.qbox.me/FlrsCtuy6a6ctKqHJaF2ik0SxcaT" width = "90%" height = "90%" align=center />

#  <span id="tutorial3">定时开启和关闭导播台</span>
支持通过定时时钟，设置导播台的开启时间和开启时长。设置后，导播台将在设置好的开始时间启动，在运行指定时长后自动关闭。运行期间中，允许对剩余时长时间进行调整。
1. 登录直播云控制台，选择**云导播**
2. 点击右侧操作的**开启**按钮

<img src="https://dn-odum9helk.qbox.me/Fu0AK-Cshv8EyoP-_kzYHT0QsjhI" width = "90%" height = "90%" align=center />

4. 开启定时任务，设置开启时刻和开启时长
<img src="https://dn-odum9helk.qbox.me/FiS6E702q9nUJnUKciJH63UlU8cS" width = "90%" height = "90%" align=center />

6. 点击右侧操作的**关闭**按钮，可以提前关闭导播台
