# 基于OpenStack 开发
七牛云 QEC 专有云以业界主流的 OpenStack 平台为核心，融入七牛云对客户需求的深刻理解，保持了与七牛云 QVM 公有云操作平台一致的操作体验，针对 Nova （计算管理）、Cinder（块存储）、Manila（文件存储）、Neutron（网络管理）、Glance（镜像管理）等 OpenStack核心组件进行大量深度地优化和开发。
# 大规模场景支撑能力
系统在单一集群部署规模上不断突破，助力用户构建简单、高速、高可扩展的开源云基础设施，以支撑快速、灵活和大规模的业务创新与部署。
# 异构资源统一管理能力 
系统代码同源异构，针对不同架构处理器进行适配，兼容 X86、ARM、MIPS、ALPHA 等架构处理器和 CentOS、麒麟、UOS 等操作系统。单一集群支持多架构云主机、裸机。
# 全面的异构设备加速能力
系统具备丰富的异构设备加速能力，提供 GPU、NVMe、FPGA、USB、光驱等广泛的硬件设备的直通接入服务，满足 AI、图像处理、CDN、大数据等场景的使用要求。
# 与公有云一致的使用体验
QEC专有云前台控制台将应用作为一个整体，以产品的服务和管理为核心，在使用体验上与七牛云 QVM  公有云控制台保持了一致的使用姿势，降低了专有云使用门槛，提供了灵活丰富的产品配置服务，可以快捷生成所需资源和服务，进一步提升云服务的便捷性和易用性。
