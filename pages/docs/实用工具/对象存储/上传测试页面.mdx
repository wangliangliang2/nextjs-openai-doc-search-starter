# 上传速度检测

<br>

在遇到上传慢或上传不了问题时，需要用户通过[提交工单](https://support.qiniu.com/question)提供有效地信息，发给我们来帮助排查。为了方便用户操作，七牛云提供了简单的 web 端检测工具。

<br>


# 使用方法

<br>

打开 [测速页面](https://kodo-utils.qiniu.com/upload-test) ，选择您指定的区域点击**开始测试**查看该区域的测速结果。
查看当前七牛云对象存储 Kodo 所支持的[存储区域](/kodo/product_introduction/1671/region-endpoint-fq)。
