**Q：如何购买内容审核资源包？**
A：购买入口：https://portal.qiniu.com/financial/respack/censor

**Q：如何查询内容审核资源包使用情况？**
A：
1.进入财务中心 - 订单列表：https://portal.qiniu.com/financial/orders

2.找到对应三鉴资源包，点击详情
![](https://dn-odum9helk.qbox.me/FoiZJvYxbtmu1bIWaPs_NZLy8Fu6)

3.在右边名目处，点击「xxx资源包」
![](https://dn-odum9helk.qbox.me/FuOb_lfaql5PNHPsY3kO5e_sf_c5)

4.在详情处可查看截止上个月底的已使用量、余量
![](https://dn-odum9helk.qbox.me/FlXK2veK-Nna0vwLyPAw0sfF-I0j)

5.进入财务系统 - 实时消费明细，可查看截止今天本月的资源包抵扣量
![](https://dn-odum9helk.qbox.me/Fr9G3CLtPD6OdhrvxN_xHcQnnQz2)

6.`总使用量 = 截止上个月底的余量 - 截止今天本月的资源包抵扣量`
比如
截止上个月底的余量 = 45426500张
截止今天本月的资源包抵扣量 = 10615600张
则总使用量 = 45426500 - 10615600 = 34810900张
