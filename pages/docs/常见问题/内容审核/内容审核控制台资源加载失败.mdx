内容审核控制台，在复核时资源加载失败，常见问题及解决办法如下：

**1. Chrome 浏览器用户请检查网站安全设置，允许显示混合内容**
Chrome 80 版本开始，会逐渐屏蔽网站的混合内容（https 网站加载 http 的图片视频资源），如果对应的存储空间不能提供 https 的域名，则需要用户手动允许加载混合内容
操作如下：

点击地址栏右边的锁图标 -> 网站设置 -> 权限 -> 不安全内容 -> 设置为允许

![](https://dn-odum9helk.qbox.me/FvVojMoPVzvtsLTW38XfkzLuIBHh)


![](https://dn-odum9helk.qbox.me/FkDuqC4hVCSmMENawix4E2BW62lf)


**2. 检查存储空间是否有可用域名，并且域名可用**
内容审核控制台将通过存储空间设置的可用域名访问资源，**并且优先使用默认域名**

![](https://dn-odum9helk.qbox.me/FtjmscnFewqqhwYF-Hvszp1ORp5U)

**3. 请检查对应存储空间中的资源是否存在，已删除的资源将无法显示**
