# 问题描述

支付宝认证失败，提示支付宝账号已被认证，或者提示授权不成功。

（支付宝认证请参考：https://developer.qiniu.com/af/manual/4062/quick-start-personal-authentication ）



# 问题原因 

这样的报错原因基本有两种：

## 原因一

您之前有其他的七牛账号绑定过该支付宝账号。你可以登录你的支付宝账号，点击 我的->设置->账号管理->账号授权 查看你的认证产品。详情如下：

![](https://dn-odum9helk.qbox.me/FsZoOSm-vZ5ZZEfH1VoV_W2QrahU)

如果你支付宝账号已经认证过，请您使用银行转账验证方式进行验证。

## 原因二

授权失败可能是支付宝账号的某些不可抗拒原因，详情您可以咨询支付宝客服电话：95188 ，或者你提交工单，将您的授权失败的页面截图以及支付宝账号通过提交工单的方式提交给我们，这边会帮您查看授权失败的原因。
