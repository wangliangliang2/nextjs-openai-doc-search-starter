# 账单
### 什么是账单？
后付费产品每月产生的消费金额
### 什么时间出账？
每月5号出上月账单，例如10月份账单会在11月5号出账
### 在哪里查看？
[管理控制台>费用>财务中心>账单和消费详情](http://https://portal.qiniu.com/financial/summary)

![账单图](https://dn-odum9helk.qbox.me/FpyIDTIg1X75XlEMXrchTGyAHTBi)
### 如何查看账单消费详情？
点击「账单编号」可查看账单详情，详情中包含产品使用量、价格以及消费金额

![账单详情](https://dn-odum9helk.qbox.me/Fs7zWlYVLoYmQr6M254XfOpV3otA)
# 结算单
### 什么是结算单？
包含后付费账单和预付费订单的消费金额
### 在哪里查看？
[管理控制台>费用>财务中心>月结算单](http://https://portal.qiniu.com/financial/settlement)
![月结算单](https://dn-odum9helk.qbox.me/Fqq5MF2z-qiQxr2vNJbkigD48yCA)
### 满足哪些个性化需求？
* 时间筛选：可点击「本月」、「上月」、「上季度」或者自定义时间选择所需查看时间范围
* 产品筛选：可在「产品类型」选择框内查看所选产品的消费情况
* 支付状态：全部、已支付、未支付
* 交易类型：结算单（包含账单和订单）、账单、订单
* 查看明细：点击「详情」按钮可查看结算单明细
* 导出消费明细：点击「导出消费明细」可导出按照条件筛选的消费详情到本地
