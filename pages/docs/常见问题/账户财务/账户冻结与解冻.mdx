# 1.冻结条件

当前账户余额小于当前消费金额时，系统自动对账户执行冻结操作:

- 七牛云服务将不可用。
- 账户冻结期间，因保存数据而使用的存储空间仍将计算费用。

# 3.解冻方法  

请前往[帐号充值](https://portal.qiniu.com/financial/recharge)及时完成现金充值。 用户可参考账户过去使用量数据，计算未来三个月的预估费用，用户可选择一次性充入等额现金,以便享受稳定的服务。

# 4.账户冻结后的影响

- 当账号冻结时，七牛云服务不可用。

- 账户被冻结后,七牛云将保留账户中的所有数据连续 30 个自然日(自冻结之日起计算)。

- 账户被冻结超过连续 30 个自然日后,视同用户默认放弃数据所有权,七牛云之后将有权定期删除账户中的所有数据,并且不可恢复。

- 在账户冻结、且保留数据期间，如用户想要取回数据，请先完成现金充值缴清账户欠费款项，然后联系七牛云技术支持进行后续沟通。

- 冻结帐号期间，因保存数据而使用的服务仍在计算费用，例如：存储服务。自服务欠费冻结日起，该服务所占用的存储空间资源仍会继续按日计费。

- 七牛云不鼓励客户拖欠费用和账单。若账号因欠费而被冻结，您仍应将欠缴的费用付清，同时支付因冻结期间存储文件和任何资源所造成的额外费用。否则，七牛云将保留按照您未付费用的万分之五的标准，按日向您收取滞纳金的权利。按照规范制度和法律，七牛云有权力采取法律途径追回欠款。


# 5.账户被冻结后,充入抵用券/现金券为何不能解冻?

- 抵用券/现金券不等同于现金充值，无法解除账户的冻结状态，账户欠费冻结期间，请充入现金进行解冻。

- 已充入现金会立刻用于缴清账户欠费款项，并重新判断是否继续冻结；不满足继续冻结条件的,账户将10分钟后解冻。解冻完成后,后续扣费将会优先扣除抵用券/现金券。

- 欠费期间所充值的抵用券余额仅用在下月消费时抵扣，并且不能抵消现金余额的欠费。
