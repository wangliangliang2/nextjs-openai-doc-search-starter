# 如何查看七牛账单中提到的数据？
- 在七牛的报表中心，通过时间段的选择，可查看对应时间段的汇总数据。在 [财务概览](https://portal.qiniu.com/financial/overview) 和 [账单和消费详情](https://portal.qiniu.com/financial/summary) 中查看。

- 进行账单核对时，可将报表中的时间选择调整至和账单计费时间一致，查看汇总数据进行对照。
