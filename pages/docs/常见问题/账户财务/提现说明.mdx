# 可提现金额说明
## 可提现金额 = 实际现金余额 - 欠票金额 - 未支付金额 - 提现中的金额
- 实际现金余额：扣除充值赠送和实时消费后金额。含充值赠送的提现请查看[含赠送金的提现说明](http://developer.qiniu.com/af/kb/6450/give-gold-deposits-containing-withdrawal)。
- 欠票金额：发票超开金额，即开票金额大于充值金额。（需要将超开的发票退回至七牛，方可提现该部分金额。[如何退票？](http://developer.qiniu.com/af/kb/6339/invoice-refund-process)）
- 未支付金额：已出账，未支付金额。
- 受理中的金额：正在受理中的提现金额总和。
# 如何申请提现？
1. 在[财务概览](http://portal.qiniu.com/financial/overview)页面点击「提现」按钮，进入提现申请页面。
![image.png](https://dn-odum9helk.qbox.me/FttAOXgL03AXl8Xh2U02HYi9v1S-)
2. 点击「申请提现」按钮，填写申请提现表单。
![image.png](https://dn-odum9helk.qbox.me/FpwB5tGlGTJIGGcxb7nU0afrZDnr)
3. 填写表单信息，并提交申请。
- 提现金额：必填信息，且填写的提现金额不能大于可提现金额。
- 收款方开户名：您在银行的账户名称（为保证您的资金安全，收款方开户行信息需和实名认证一致。）
- 收款方银行名称：您的开户银行名称
- 收款方银行网点名称：您开户银行的支行网点名称
- 收款银行账号：您的收款银行卡号（请注意不要填写错误，影响收款）
![image.png](https://dn-odum9helk.qbox.me/FtIGvKsu7G3_Xx38AxKoNKdiebNL)
4. 提交申请，进入提现流程，您可在提现记录中查看您的提现进度。（线下退回预计需要5~15个工作日退回到您的银行账户）
- 打款金额：实际打款到您的银行账户金额。
- 状态：
受理中：提交提现申请，七牛内部审核。
待打款：线上扣费，等待打款。
已打款：已打款，等待银行到账。
已驳回：申请被驳回，鼠标在「已驳回」状态上可查看驳回理由。
- 提现时间：打款到您填写的收款银行账户时间。（实际到账时间以银行为准）
![image.png](https://dn-odum9helk.qbox.me/FgzO_iK0_9i_u1noHq8Hf4PJ22o_)
# 为什么我不能申请提现？
- 未实名认证、个人认证升级企业认证审核中的用户不能申请提现。请先前往[实名认证](http://portal.qiniu.com/user/profile)。
- 可提现金额小于等于0的用户不能申请提现。
