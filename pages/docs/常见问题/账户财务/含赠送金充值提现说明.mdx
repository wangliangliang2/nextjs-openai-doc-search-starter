# ** 1 充值赠送金概述**

-充值赠送金只能支付按量计费的产品，不可开票、提现。

-赠送金是默认您的此次充值的现金全部用于消费的情况下所给予的赠送，当该笔充值未全部消费而申请提现时，此次赠送无效，在可提现金额中将减去对应的赠送金额。

# ** 2 含赠送金充值提现实例 **
## ** 2.1 单笔含赠送金充值提现 **
充值金额 = 1000元

赠送金额 = 300元

消费金额 = 500元

提现前现金余额 = 800元

最大可提现金额 = 充值金额 - 消费金额 = 1000元 - 500元 = 500元

提现后现金余额 = 0元

备注：该笔充值赠送无效。

** 若您申请提现金额小于最大可提现金额，则调整账户现金余额。**
示例：

申请提现金额 = 300元

提现后现金余额 = 最大可提现金额 - 申请提现金额 = 500元 - 300元 = 200元

## **2.2 多笔含赠送金充值提现**
### **例1 消费金额 <= 第一笔充值金额**
充值金额(1) = 1000元

赠送金额(1) = 300元

充值金额(2) = 1000元

赠送金额(2) = 300元

消费金额 = 500元

提现前现金余额 = 2000元

最大可提现金额 = 充值金额1 + 充值金额2 - 消费金额 = 1000元 + 1000元 - 500元 = 1500元

提现后现金余额 = 0元

备注：第一、二笔充值赠送均无效。

**若您申请提现额小于最大可提现金额，则调整账户现金余额。**
示例：

申请提现金额 = 300元

提现后现金余额 = 充值金额(1) + 赠送金额(1) + 充值金额(2) - 消费金额 - 申请提现金额 = 1000元 + 300元 + 1000元 - 500元 -300元 = 1500元

备注：第一笔充值赠送有效，第二笔充值赠送无效。

**若 充值赠送(2) < 客户申请提现金额 < 最大可提现金额**
申请提现金额 = 1100元

提现后余额 = 充值金额(1) + 充值金额(2) - 消费金额 - 申请提现金额 = 1000元 + 1000元 -500元 - 1100元 =400元

备注：两笔充值赠送均无效。

### **例2 第一笔充值金额 < 消费金额 <= 第一笔充值金额 + 第一笔赠送金**
充值金额(1) = 1000元

赠送金额(1) = 300元

充值金额(2) = 1000元

赠送金额(2) = 300元

消费金额 = 1100元

提现前现金余额 = 1500元

最大可提现金额 = 充值金额(2) = 1000元 

提现后现金余额 = 200元

备注：第一笔充值赠送有效，第二笔充值赠送无效。第一笔现金余额200元可继续用于消费。

**若您申请的提现金额小于最大可提现金额，则调整账户现金余额。**
示例：

申请提现金额 = 500元

提现后现金余额 = 充值金额(1) + 赠送金额(1) + 充值金额(2) - 消费金额 - 申请提现金额 =1000元 + 300元 + 1000元 - 1100元 - 500元 =700元

**若您申请提现金额大于最大可提现金额，则两笔充值赠送均无效。**
示例：

申请提现金额 = 1200元

可提现金额 = 充值金额(1) + 充值金额(2) -消费金额 =1000元 + 1000元 - 1100元 = 900元

提现后现金余额 = 0元

### **例3 消费金额 >= 第一笔充值金额+第一笔赠送金**
充值金额(1) = 1000元

赠送金额(1) = 300元

充值金额(2) = 1000元

赠送金额(2) = 300元

消费金额 = 1500元

提现前现金余额 = 1100元

最大可提现金额 = 充值金额(1) + 赠送金额(1) + 充值金额(2) - 消费金额 = 1000元 + 300元 + 1000元 - 1500元 = 800元

提现后现金余额 = 0元

备注：第一笔充值赠送有效，第二笔充值赠送无效。

**若您申请提现金额小于最大可提现金额，则调整账户现金余额。**
示例：

申请提现金额 = 500元

提现后现金余额 = 最大可提现金额 - 申请提现金额 = 800元 - 500元 = 300元

## **2.3 无赠送金和含赠送金充值并存**
充值金额(1) = 300元（无充值赠送）

充值金额(2) = 1000元

赠送金额(2) = 300元

消费金额 = 1000元

提现前现金余额 = 600元

最大可提现金额 = 充值金额(1) + 充值金额(2) - 消费金额 = 300元 + 1000元 -1000元 = 300元

提现后现金余额 = 0元

备注：按照充值时间顺序扣费，第二笔充值赠送无效。充值赠送金是否为有效情况和2.2例子相同。

**若您申请提现金额小于最大可提现金额，则调整账户现金余额。**
示例：

申请提现金额 = 200元

提现后现金余额 = 最大可提现金额 - 申请提现金额 = 300元 - 200元 = 100元

### **提示：**
上述例子中不涉及含抵用券的情况。
