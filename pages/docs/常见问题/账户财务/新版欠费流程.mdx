# 一、欠费规则
当账户的可用额度小于0时，将进入欠费流程。
关于可用额度的更多说明请参考 [新版财务概况](https://developer.qiniu.com/af/manual/5997/the-new-financial-situation)

# 二、欠费流程
**1.欠费通知。**

* 欠费当天七牛云会向您发送欠费的邮件和短信通知。

**2.欠费保护期。**

* 欠费后立即进入保护期，保护期内七牛云会**每天**向您推送一次提醒。
* **保护期时长：**保护期时长根据您的认证状态有所不同，未认证用户保护期1天，个人认证用户保护期3天，企业认证用户保护期5天。
* **保护期内限制：**保护期内，您使用的服务/功能将受到一定的限制，包括无法使用余额购买预付费产品，无法新增和修改CDN域名，无法开启云主机按量付费资源，对象存储内无法新建存储空间、绑定域名、跨区域同步等操作，详情参考文档[欠费状态下各产品的限制功能](https://developer.qiniu.com/af/manual/6650/owe-the-protection-limit-function)。
* **解除保护期：**保护期内充值达到充值要求后，您的账号将自动恢复正常。

**3.欠费冻结。**

* 保护期结束后，七牛云会向您发送冻结的邮件和短信通知，您的账号将直接被冻结。
* 冻结期间，因保存数据而使用的服务仍在计算费用，例如：存储服务。自服务欠费冻结日起，该服务所占用的存储空间资源仍会继续按日计费。
* 当您的充值金额达到最低充值要求后，您的账号将自动解冻，恢复正常。
* 冻结期间，如果您想要取回数据，请先完成现金充值缴清账户欠费款项，然后联系七牛云技术支持进行后续沟通。
* 若账号因欠费而被冻结，您仍需付清欠缴的费用，同时支付因冻结期间存储文件和任何资源所造成的额外费用。否则，七牛云将保留按照您未付费用的万分之五的标准，按日向您收取滞纳金的权利。按照规范制度和法律，七牛云有权力采取法律途径追回欠款。

**4.资源清空。**

* 账户被冻结超过连续 30 个自然日后，视同您默认放弃数据所有权，七牛云将有权定期删除账户中的所有数据，并且不可恢复。

# 三、欠费后如何处理
* 欠费后请尽快进行充值，充值入口：财务中心->[账户充值](https://portal.qiniu.com/financial/recharge)。
* 充值金额：不得低于当前账户的可用额度，建议适当多充值，避免继续欠费。

# 四、常见问题
1.账户被冻结后,充入抵用券/现金券为何不能解冻?

* 抵用券/现金券不等同于现金充值，无法解除账户的冻结状态，账户欠费冻结期间，请充入现金进行解冻。
* 已充入现金会立刻用于缴清账户欠费款项，并重新判断是否继续冻结；不满足继续冻结条件的,账户将10分钟后解冻。解冻完成后,后续扣费将会优先扣除抵用券/现金券。
* 欠费期间所充值的抵用券余额仅用在下月消费时抵扣，并且不能抵消现金余额的欠费。
