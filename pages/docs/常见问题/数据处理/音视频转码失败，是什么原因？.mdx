- 首先 [获取转码的 persistent ID](https://developer.qiniu.com/dora/kb/3643/how-to-acquire-persistent-id ) ，然后通过 [持久化处理状态查询接口](https://developer.qiniu.com/dora/manual/1294/persistent-processing-status-query-prefop) ，查看简要的报错原因。
- 或者浏览器直接访问 `http://api.qiniu.com/status/get/prefop?id=` 后跟 persistent ID 。
- 如果对处理失败原因有疑问，可以带着 persistent ID ，提交工单咨询。
