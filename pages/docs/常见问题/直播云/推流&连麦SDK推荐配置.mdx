# 使用直播SDK的机型配置推荐
由于直播推流以及连麦都是高消耗的操作，所以对手机的CPU以及内存消耗占用较高。如果手机机型太低配，可能造成直播效果不佳，所以此文档目的意在介绍下推荐的直播机型配置

## 安卓
### 推荐机型：
三星S7
HTC M9w
华为P9/华为荣耀8
小米5/小米5s/红米Note 4X/红米Note3
LG Nexus 5X
vivo X9
OPPO R9s


### 推荐配置：
版本：Android原生4.3以上（录屏、美颜和连麦功能需要Android原生5.0以上系统支持）
处理器：骁龙808处理器平台
内存：2GB
摄像头（后置/前置）: 1300万像素/500万像素
显示屏：4.3英寸（1920*1080分辨率）

## IOS  
### 推荐机型及配置：
iPhone SE 及以上  （录屏、美颜和连麦功能需要iPhone6s及以上系统才能保证良好的体验效果）
