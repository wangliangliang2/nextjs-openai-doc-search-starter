## Q：如何查看某条路的历史直播记录？
可以直接调用该接口进行查询：https://developer.qiniu.com/pili/api/2778/live-history
 
<br/>

## Q：该接口要传入 start 和 end 是什么意思？
比如主播在如下时段直播：
 ```
 1点 到 3点
 4点 到 6点
 7点 到 9点
 ```
 如果您调用接口的时候 start 和 end 分别传的是 1点 和 6点，那么返回的结果会有两套数组：
 
 start：1点 end：3点；

start：4点 end：6点
 
 那么 7点 到 9点 这条记录就没包括在上面的返回结果里面，另外传入的时间是十进制UNIX时间戳，单位为秒

<br/>

## Q：为什么我调用了接口后，发现没有返回 start 和 end 的数组？
可能因为如下原因：

- 调用接口的时候，指定 start 和 end，其中这两个时间内没有任何推流记录
- 之前推的流不是 视频 h264 + 音频 aac/mp3 格式导致
