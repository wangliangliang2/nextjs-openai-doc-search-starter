# 背景
由于近期出现不法分子通过七牛 CNAME 域名传播违规资源的情况，受相关信息监管部门要求，七牛 CDN 将调整用户主动操作下线或欠费冻结后的域名解析逻辑。

# 调整说明
2021年10月10日之前，用户主动下线或欠费冻结加速域名后，加速域名会被解析回用户源站。为避免相关合规风险，自2021年10月11日起，**下线或欠费冻结的域名会解析至不可用地址。**
因此，在用户主动下线或欠费冻结之后，将无法通过加速域名访问源站内容。

# 应对方案
若希望在域名下线后，可以继续通过加速域名访问源站，可自行修改 DNS 配置，将加速域名的 CNAME 记录指向您的源站即可。
