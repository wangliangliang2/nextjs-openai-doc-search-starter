-----
<div id="Q0"></div>
**Q:CDN 是否支持 websocket 加速？**
&emsp;暂不支持 websocket 加速。

<div id="Q1"></div>
**Q:CDN 是否支持 HTTP2 访问？**
&emsp;支持，在 控制台 - CDN - 域名管理 中，修改 https 配置可以开启 http2 访问。

<div id="Q2"></div>
**Q:微信小程序要求的TLS版本必须大于等于1.2怎么处理？**
&emsp;域名需要先开启 HTTPS，然后开启 HTTP 2 功能，HTTP 2.0 默认使用 TLS1.2 及以上版本。

<div id="Q3"></div>
**Q:CDN 是否支持 Gzip 处理？**
&emsp;支持。

<div id="Q5"></div>
**Q:CDN 支持指定端口访问吗？**
&emsp;CDN 域名仅支持默认的 80/443 端口访问，不支持自定义端口访问，但是 CDN 配置回源时，支持回源到您的服务器的指定端口。

<div id="Q6"></div>
**Q:CDN 支持支持海外访问加速吗？**
&emsp;支持，您可以在创建域名时，选择覆盖范围为 海外/全球 ，如果您的域名已经创建，您可以通过 **[工单系统](https://support.qiniu.com/tickets/new)** 反馈给我们，我们将在后台为您修改。
&emsp;大陆：仅使用国内的加速节点，海外用户访问将被调度到国内节点。
&emsp;海外：仅使用海外的加速节点，国内用户访问将被调度到海外节点。
&emsp;全球：使用覆盖全球的加速节点，根据用户所在位置自动调度到最优加速节点。
