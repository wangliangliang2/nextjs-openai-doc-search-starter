# 运营商劫持投诉

首先需要确认是否被劫持了。
劫持一般分为 DNS 劫持和内容劫持。如果遇到以下两种现象：
1. 域名解析到七牛的 cname 地址，但是跟我们确认过，域名实际解析到的节点非七牛 CDN 节点。  
2. 文件内容被篡改，加入了广告链接、跳转链接等。  

投诉内容格式参考：

>有客户反馈在北京移动被劫持了，访问到的节点不是七牛云存储公司的CDN节点（或者文件被篡改）。
>
>测试链接如下：`<这里提供劫持链接>`，麻烦帮忙解决

# 联通劫持

1、找到联通在本地的最高等级投诉热线

2、确定投诉内容，投诉时，不要向接线员描述这些细节，而要告诉他们：当我访问某个域名时，经常跳到奇怪的站点

3、收集证据，浏览器截图、录屏动画、网页源代码、Fiddler HttpWatch Wireshark等工具的抓包数据都可以作为证据。

4、拨打投诉热线，发送证据，并确认投诉的问题已经发送邮件，请检查是否收到。
