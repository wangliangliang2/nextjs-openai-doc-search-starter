| Q1: 如何查询访问量最大的CDN资源？| 
| ---- |
| 可以在 [**七牛控制台 - 访问分析**](https://portal.qiniu.com/cdn/statistics/log/access) 界面查询 top100 资源URL，操作详情参考：[**统计分析**](https://developer.qiniu.com/fusion/3846/data-statistics-fusion) |

<br>

| Q2: 如何查询每个资源的访问量和访问次数？| 
| ---- |
| 查询每个资源的访问量和访问次数，需要您下载CDN日志，根据CDN日志自己统计分析下。<br>CDN日志下载方式：<br>  1. 在 [**七牛控制台**](https://developer.qiniu.com/fusion/manual/3847/cdn-log-fusion) 下载。<br>  2. 调用 [**API接口**](https://developer.qiniu.com/fusion/api/1226/download-the-log) 下载。|
