# Q：未备案冻结的域名如何解冻？
&emsp;备案完成后，可以先在「**[域名信息备案管理系统](http://beian.miit.gov.cn/publish/query/indexFirst.action)**」查询备案是否已经公示，确认已经公示后，在 **七牛开发者平台** 页面选择 [**CDN**](https://portal.qiniu.com/cdn/overview) ，选择 [**域名管理**](https://portal.qiniu.com/cdn/domain) ，点击您需要解冻域名后的 **解冻** 字样，即可发起解冻任务。备案完成后，工信部一般会延迟1个工作日左右进行公示，解冻域名将以实际公示情况为准。

# Q：如何查询备案？
&emsp;您可以通过「[**域名信息备案管理系统**](http://beian.miit.gov.cn/publish/query/indexFirst.action)」对备案信息进行查询。
&emsp;查询方式：左侧「公共查询」-「备案查询」-「备案信息查询」，在网站域名处输入您的备案主域名，正确验证码后点击提交，如果域名没有备案信息，会提示`「没有符合条件的记录」`。

# Q：为什么域名之前备案过了还会被未备案冻结？
<a id="Q3"></a>
&emsp;备案吊销是指对备案信息做收回并注销，由有管辖权力的部门停止原来准许进行某项活动的对象停止该项活动并收回准许文本的执行过程，一般带有强制性。起因基本是由于该对象违背或违反了发放文本时的约定所致，七牛检测到备案信息失效后，会对域名执行冻结操作。

# Q：备案吊销后如何处理？
<a id="Q4"></a>
&emsp;确认备案被吊销后，您可以在原域名备案处，重新进行备案。

# Q：某些工具可以查询到备案，但还是被冻结了？
&emsp;网络上的备案查询工具都有一定的延迟，需要以工信部官网的公示结果为准，查询方式参考「<font color=#4671D5><a href="#Q1">如何查询备案</a></font>」。
