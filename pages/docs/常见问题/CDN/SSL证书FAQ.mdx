| Q1: 为什么我无法上传自有证书或证书上传后不可用？| 
| ---- |
|证书有效期需要≥30天，少于30天的证书是不能用于七牛 CDN 的。|

<br>

| Q2: 我申请了七牛的 SSL 证书，为何申请 SSL 证书一直待确认,有时候需要好几天？|
| ---- |
| 在证书订单页面，点击订单最后的 “详情” ，确定验证流程是否配置完成。<br>当选择 DNS 验证方式，域名除 TXT 记录外，还有 A 记录、cname 记录等记录类型，验证是不会通过的。这种情况需要重新提交订单，选择文件验证方式。 |

<br>

| Q3: 因为解析冲突，我不能选择 DNS 验证方式；因为服务器不支持 .well-known 这种目录，文件验证不能通过，我如何通过验证呢？|
| ---- |
| 参考文档：[域名已经解析如何通过验证](https://developer.qiniu.com/ssl/kb/3890/domain-name-already-how-do-parsing-by-verifying)<br>文档中的第一种方法，可以解决两种情况同时存在的问题。|

<br>

|Q4: 为什么申请 SSL 证书失败呢？|
| ---- |
| CA 中心验证的过程是完全黑盒的，具体失败原因无法得知。从经验看，原因有以下几种可能性：<br>1. 网站所有人与证书申请人姓名不同。<br>2. 域名中含有 “非法” 字符，比如域名中不能有 test 。<br>3. 域名涉及非法信息 / 被举报 / 被 CA 中心添加黑名单。|

<br>

|Q5: 为什么申请 SSL 免费证书配置时间很长？|
| ---- |
| 通常情况下证书会在配置在 30 分钟内颁发并完成配置，期间域名访问不受影响，部分配置不可修改。但证书颁发受 CA 中心的影响，证书颁发时间不可控，若较长时间仍未配置完成，请提交工单咨询。|
