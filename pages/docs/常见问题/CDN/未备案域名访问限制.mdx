七牛 CDN 支持为海外业务客户提供加速服务，但对于未备案域名访问有以下限制。
## 国内 ip 无法访问未备案域名
对于海外未备案域名，七牛 CDN 可提供在纯海外区域（不含中国大陆）的加速访问；但受到相关监管规则限制，目前国内 ip 默认无法访问未备案的海外加速域名。
如果实际业务需要域名可被国内 ip 访问，建议您在[工信部域名信息备案管理系统](http://beian.miit.gov.cn/state/outPortal/loginPortal.action)进行备案，然后重新将域名配置为全球加速。

如有其他配置需求，可联系七牛销售进行申请和业务审核。

## 备案吊销情况说明
原有已备案的纯海外加速域名如果备案被吊销，七牛将同样禁止国内 ip 访问。因此对于有备案的海外加速域名，强烈建议转为全球加速区域。
