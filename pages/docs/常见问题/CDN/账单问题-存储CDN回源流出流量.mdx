# 问：账单中「标准存储CDN回源流出流量」是什么含义？
答：
下图为CDN访问的简化流程图，后续步骤号码将对应流程图中标号。
 <img src="https://dn-odum9helk.qbox.me/FlzgLWCIX8jz_s2Ua41TjYP-HzAN" width=800 />

## 场景1：CDN节点「**没有缓存**」时，需要回源站拉取资源并缓存，访问步骤为：
1： 客户访问端发送访问请求到CDN节点。
2： CDN节点进行回源，发送请求到七牛云存储。
3： 七牛云存储返回数据给CDN节点。
4： CDN节点缓存数据，并由CDN返回数据给客户端。
计费说明：该过程中，产生的计费项目为流程 3 产生的**存储CDN回源流出流量** + 流程 4 产生的**CDN访问流量**。

## 场景2：CDN节点「**有缓存**」时，访问步骤为：
1： 客户访问端发送访问请求到CDN节点。
4： CDN节点返回数据给客户端。

计费说明：该过程中，产生的计费项目为流程 4 产生的**CDN访问流量**。所以，并不是每次请求都会产生这两项计费。如果CDN有缓存没有进行回源，就不会产生**存储CDN回源流出流量**。

-----

# 问：如何区分 存储CDN回源流出流量 和 CDN访问流量 呢？
答：**存储CDN回源流出流量**是由七牛的存储机房流出的流量（流程3），**CDN访问流量**是由CDN节点流出的流量（流程4）。

-----

# 问：为什么账单中 存储CDN回源流出流量 比 CDN访问流量 还高呢？
答：CDN回源流出流量高的可能原因有
1.  业务主要是视频业务，但是创建域名时域名场景选择了网页平台，下载时边下边播，回源时完整回源，导致即使只观看一小部分，也会产生完整的回源流量费用。`建议空间可以绑定不同域名，视频业务使用点播平台域名播放`，域名场景不影响使用，但会针对特定场景有一定的优化。
2.  资源访问时，如果`每次都带了不同的query参数`，或者`把缓存时间设置非常小`，会导致强制CDN节点回源，所以每次访问都会产生了**存储CDN回源流出费用**。

-----

# 问：可以关闭存储CDN回源流出流量吗？
答：不能，CDN节点至少需要有1次回源拉取行为才能完成缓存，在此过程中存储产生的流量都需要进行计费。
