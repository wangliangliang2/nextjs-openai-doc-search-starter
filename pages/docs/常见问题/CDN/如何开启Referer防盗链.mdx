登录 [七牛开发者平台](https://portal.qiniu.com/create) ，在左侧点击 [ CDN ](https://portal.qiniu.com/cdn/overview)，在域名管理中，找到需要开启 Referer 防盗链的域名，点击** 配置**：  

![](https://dn-odum9helk.qbox.me/FhVoHF_8RxtA8Ar03a7dp65s6HKh)

下拉弹窗，找到 访问控制 -> Referer 防盗链，点击** 修改配置**：  

![](https://dn-odum9helk.qbox.me/FgFsb9oKSOruOYyychkswdwvVA4q)

点击** 开启 **，然后配置具体的黑白名单规则：  

![](https://dn-odum9helk.qbox.me/Fnu0mzKrO-s_wnp-0jVn3nb5PLRz)
