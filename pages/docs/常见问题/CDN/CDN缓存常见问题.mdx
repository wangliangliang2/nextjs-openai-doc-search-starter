
## 缓存刷新和文件预取有什么区别？

缓存刷新：是指通过管理控制台或者 API 接口的方式将缓存在 CDN 节点的资源文件删除或标记为过期，对于新的请求会回源站服务器获取新的文件。

文件预取：是提前将资源文件缓存在 CDN 节点，新的请求过来之后可以直接命中，无需回源服务器获取资源文件，以低源站压力，提升首次访问体验。

## 刷新缓存后要多久生效？

缓存刷新全网生效时间一般为 10 分钟。

## 使用七牛云对象存储服务，文件更新需要手动刷新吗？

需要。使用了七牛云对象存储服务时，存储资源更新（替换或删除）后，需要手动刷新触发 CDN 节点刷新缓存，否则可能会导致依然能够访问到原文件的情况。

## 发现通过七牛云 CDN 访问的资源和源站资源不一致怎么办？

通过管理控制台或 API 接口进行缓存刷新。

### 管理控制台：
