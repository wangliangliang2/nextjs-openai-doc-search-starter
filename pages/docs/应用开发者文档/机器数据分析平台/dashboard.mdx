通过 `<dashboard> `元素定义仪表盘。

```
<!-- 可以通过编写XML来自定义dashboard -->
<dashboard>
  <!-- your xml code here -->
</dashboard>
```
	
### 属性

|属性|说明|类型|默认值|
|:--:|:--:|:--:|
|theme|dashboard 主题风格，可选值为“dark”、"light"|string|“light”|
|refreshInterval|dashboard 中的图表刷新频率|string|无|
|time|dashboard 中的全局时间|time|无|
