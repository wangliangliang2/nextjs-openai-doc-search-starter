* `<option>`元素定义图表的通用属性，系统为不同图表支持不同属性。

### 父元素

所有图表类型均支持`<option>`元素。

### 属性

**二维图表类型的公共 `<option>`标签的属性**
	
|属性|可选项|说明|默认值|
|:--:|:--:|:--:|:--:|
|name|style.mergeType<br>style.dataLimit<br>style.stackMode|数据的合并方式，支持 sum、avg、min、max、newest、oldest 类型值<br>当数据量很大时，限制图表中显示的数据量<br>是否设置数据堆叠方式，设置了数据堆叠后， 同个类目轴上系列配置相同的 stack 值后，后一个系列的值会在前一个系列的值上相加。|'sum'<br>500<br>'none'|

示例：

```
<option name="style.mergeTyp">sum</option>
```

	
**二维图表每种图表类型特有的`<option>`标签的属性：**

1.折线图、面积图：没有特有`<option>`标签的属性。

2.柱状图：

|属性|可选项|说明|默认值|
|:--:|:--:|:--:|:--:|
|name|style.orientation<br>style.showBarLabel|柱状图显示的方向，支持可选值为 horizontal: 柱状图水平方向上显示; vertical: 柱状图垂直方向上显示<br>是否在柱形条上显示数值以及显示的位置，支持可选值 true: 显示数值，且在柱状条的中间显示; false: 隐藏数值; top: 显示数值，且在柱状条的顶端显示|'horizontal'<br>false|

3.散点图
