`<time>`元素定义时间对象。支持通过`<earliest>`元素和`<latest>`元素设置初始开始时间、结束时间，通过设置开始时间和结束时间表示出相对时间、绝对时间、和预设的时间范围。

### 父元素

* `<search>`
* `<operation-bar>`元素的`<initial-value>`子元素
* `<action-bar>`元素的`<initial-value>`子元素

### 子元素

* `<earliest>`
* `<latest>`

**绝对时间**

* 通过`<earliest>`和`<latest>`元素指定具体的时间戳，表示两个时间戳之间的绝对时间范围。

 ```
<earliest>1572524870871</earliest>
<latest>1572524870891</latest>
```

* `<earliest>`值为 earliest 或不写`<earliest>`元素， 表示在 `<latest>` 的时间之前:

	 * `<earliest>`值为 earliest
	 
   ```
<earliest>earliest</earliest>
<latest>1572524870891</latest>
   ```
	 
 * 无`<earliest>`元素
 
	```
	<latest>1572524870891</latest>
	```
	
* `<latest>`值为 now 或不写`<latest>`元素， 表示在`<earliest>`的时间之后

	* `<latest>`值为 now
	
	```
	<earliest>1572524870891</earliest>
<latest>now</latest>
	```
	
**相对时间**

通过在`<earliest>`和`<latest>`元素指定时长来表示相对时间，时长可以是正数或负数，均表示向前偏移时间，时间单位支持: s(秒)、m(分)、h(时)、d(天)、w(周)、M(月)、Q(季)、y(年) 。

* `<earliest>`和`<latest>`元素均指定时长，如前三分钟到前两分钟：

 ```
<earliest>3m</earliest>
<latest>2m</latest>
```

* `<earliest>`元素指定时长，不写`<latest>`元素或者`<latest>`值为 now ，如前三分钟到现在：

 *  `<latest>`值为 now：
	
   ```	
<earliest>-3m</earliest>
<latest>now</latest>
 ```
 
 * 无`<latest>`元素:
	
   ```
<earliest>-3m</earliest>
   ```
	 
* `<earliest>`元素指定时长，`<latest>`元素指定时间戳 ，如从前三分钟到2019/10/31 21:13:02 的时间范围：
	
 ```
<earliest>3m</earliest>
<latest>1572527582426</latest>
```
