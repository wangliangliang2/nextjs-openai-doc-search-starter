#### @qn-pandora/app-sdk
&nbsp; &nbsp; &nbsp; &nbsp; Pandora平台应用前端开发的基本SDK。提供快速操作Pandora资源能力的Manager；APP开发中常用的工具方法；整体引入Pandora平台页面模块的方法；以及开发自定义展示的基类。具体使用文档请参考：
[https://www.npmjs.com/package/@qn-pandora/app-sdk](https://www.npmjs.com/package/@qn-pandora/app-sdk)

#### @qn-pandora/visualization-sdk
&nbsp; &nbsp; &nbsp; &nbsp; 当您需要定制自定义图表时，您需要使用这个库。它提供了自定义图表所需的基类，该基类约束了自定义图表的插件机制，它提供了完善的hook函数，通过重载这些hook函数，您可以完成自定义图表逻辑。此外，该库还提供了开发自定义图表所需要的常用工具方法，提高开发您的开发效率。具体请参考：[https://www.npmjs.com/package/@qn-pandora/visualization-sdk](https://www.npmjs.com/package/@qn-pandora/visualization-sdk)

#### @qn-pandora/pandora-component
&nbsp; &nbsp; &nbsp; &nbsp; Pandora风格的React基础组件库，提供了常用的基础组件，例如 <font color=#E96900> Button, Select, Input...</font>。如果您使用React框架开发APP，您可以使用它快速搭建出符合Pandora平台风格的APP。具体请参考：[https://www.npmjs.com/package/@qn-pandora/pandora-component](https://www.npmjs.com/package/@qn-pandora/pandora-component)

#### @qn-pandora/pandora-app-component
&nbsp; &nbsp; &nbsp; &nbsp; Pandora风格的React业务组件库，提供了适用于Pandora平台的业务组件，例如 <font color=#E96900>AdvancedTable, Downloader, SearchBar...</font>。如果您使用React框架开发APP，使用业务组件可以大大提高开发效率。具体请参考： [https://www.npmjs.com/package/@qn-pandora/pandora-app-component](https://www.npmjs.com/package/@qn-pandora/pandora-app-component)
