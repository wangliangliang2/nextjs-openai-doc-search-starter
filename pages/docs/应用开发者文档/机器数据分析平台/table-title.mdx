* 对于table类型的图表，可使用 `<table-title>` 元素定义table标题的字体。

### 父元素 {docsify-ignore-all}

在 `<chart>` 元素下，只有当 type 属性指定如下值时，可以使用 `<data-font>` 元素。

* `<chart type="tree-table">`
* `<chart type="grid-table">`

### 子元素

|子元素|说明|默认值|
|:--:|:--:|:--:|
| `<data-font>` |设置表title的字体样式，详细见 `<data-font>` |无|
| `<background-color>` |设置表头的背景颜色 |无|

示例

``` 
<table-title>
  <data-font>
    <font-weight>500</font-weight>
    <font-family>PingFang SC</font-family>
    <color>#C1C7D0</color>
    <font-size>12</font-size>
  </data-font>
  <background-color>#C1C7D0</background-color>
</table-title>

```
