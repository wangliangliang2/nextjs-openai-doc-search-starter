# 产品简介
七牛云播放器 SDK 是一款全自研内核的跨平台 多媒体播放器，该播放器内核由有着18年播放器经验的音视频专家团队研发。目前支持 Android 和 iOS 平台。七牛云播放器 SDK 具有包体小、首开快、播放流畅、使用简单等明显优势可支持直播、点播 场景下的多种视频格式及流媒体协议。

播放器完全免费，不强制绑定任何其他产品，下载地址如下：
[Android 播放器 SDK](https://developer.qiniu.com/pili/sdk/1210/the-android-client-sdk)
Android Github 地址：https://github.com/pili-engineering/PLDroidPlayer `除支持h264外，也支持rtmp和http-flv的h265播放`

[iOS 播放器 SDK](https://developer.qiniu.com/pili/sdk/1211/ios-playback-end-the-sdk) 
iOS Github 地址：https://github.com/pili-engineering/PLPlayerKit  `除支持h264外，也支持rtmp和http-flv的h265播放`

[小程序播放器]
[推荐小程序原生播放器](https://developers.weixin.qq.com/miniprogram/dev/component/live-player.html#%E7%A4%BA%E4%BE%8B%E4%BB%A3%E7%A0%81)    `仅支持视频编码h264`

[web播放器]
推荐西瓜播放器 <br>[web http播放器](http://v2.h5player.bytedance.com/generate/) 适用于http地址 `仅支持视频编码h264`
[web https 播放器](https://v2.h5player.bytedance.com/generate/)适用于https地址               `仅支持视频编码h264`


**webrtc 快直播低延迟专用播放器SDK：** `仅支持视频编码格式h264`
