七牛实时音视频云是七牛基于 WebRTC 的自研实时音视频产品；为客户提供低延时、跨平台、高品质、可定制化的一站式解决方案，帮助客户快速支持一对一视频通话、多人会议、直播连麦、旁路直播等多种业务场景。七牛实时音视频云提供全平台的客户端 SDK（包括 Web 端、Windows 端、Android 端、iOS 端）、多语言的服务端 SDK、实时互动流媒体，并提供丰富的云端功能；目前七牛实时音视频云已正式上线。

（1）全平台互通的客户端 SDK
提供高性能的 Android、iOS、 Web、Windows SDK，全平台互通。负责音视频的采集、编解码，支持发布、订阅、踢人、静音、自动重连的基本音视频通话功能。

（2）多语言的服务端 SDK
提供 Java、PHP、Python、Node.js、Golang 语言 SDK，负责鉴权、房间管理、应用管理、合流配置等功能，实现云端能力的快速接入。

（3）丰富的云端功能
提供服务端合流、实时落存储、鉴黄鉴暴恐鉴政等功能，实现更广的应用场景。

（4）实时互动流媒体线路
网状去中心化的拓扑结构，可快速扩容。全球 180 各数据中心，保证高可用、低延迟的音视频通话体验。
