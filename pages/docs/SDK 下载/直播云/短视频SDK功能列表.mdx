# 产品简介
七牛短视频 SDK，方便开发者快速实现短视频拍摄、剪辑、编辑、合成、分发功能。我们提供 80+ 功能供开发使用，申请 **0 元体验** 可 [点击此处](https://qmall.qiniu.com/template/MTE1) 获得产品试用权限。
