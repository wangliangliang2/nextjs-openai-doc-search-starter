pdr-python-sdk是Pandora的Python版本的SDK，提供简单方便的Python编程接口，实现Pandora插件平台对象的基本操作方法，从而实现pandora数据分析能力的扩展，可以用来进行自定义算子、自定义api、数据存储等功能的开发。
pdr-python-sdk支持的底层Python版本为Python3。

- 开发自定义算子：[自定义算子开发流程](/express/sdk/7282/custom-operators-sdk)
- 开发自定义API：[自定义API开发流程](/express/sdk/7273/custom-api-sdk)
