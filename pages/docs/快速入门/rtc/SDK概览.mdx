<p><strong>QNRTC-Uniapp</strong> 是基于 <strong><a href="https://uniapp.dcloud.io/">UNIAPP</a></strong>、 <strong><a href="https://developer.qiniu.com/rtc/8764/an-overview-of-the-android-sdk">QNRTC-Android</a></strong> 以及 <strong><a href="https://developer.qiniu.com/rtc/8830/an-overview-of-the-ios-sdk">QNRTC-iOS</a></strong> 三者所实现的。前端可通过 <strong>QNRTC-Uniapp</strong> 所提供的 <strong>JS 插件</strong> 与 <strong>Native 插件</strong> 调用原生 <code>Android</code> 以及 <code>iOS</code> 的 RTC 接口来实现实时音视频通话。</p>
<p>其中包括但不限于以下场景</p>
<ul>
<li>在线教育</li>
<li>主播观众连麦</li>
<li>视频会议</li>
<li>一对一社交</li>
<li>狼人杀等依赖实时音视频通话的游戏</li>
<li>在线客服</li>
<li>在线抓娃娃</li>
</ul>
