### <br/> **引言**</br>

<br/> 使用七牛开发者平台，您可以使用直观的界面对存储空间进行相应的操作。在登录七牛开发者平台之前，确保您已经注册了七牛云账号。如果您还没有七牛云账号，可通过以下步骤创建七牛云账号。

- 打开 https://portal.qiniu.com/ ，选择 **注册**。
- 遵循在线使用说明。

### <br/> **只需三步，即可按需创建存储空间**</br>

<br/> 1.登陆七牛云开发者平台，点击左侧【对象存储】
![image.png](https://dn-odum9helk.qbox.me/FkvlmMTnSjNQKGJqLaFhw6w0Kz2i)

<br/> 2.点击进入【空间管理】后，点击【新建空间】
![image.png](https://dn-odum9helk.qbox.me/FqxjrF35OGEey7b26PKiWmKAQIti)

<br/> 3.自定义存储空间名称、选择存储区域和访问控制，点击【确定】
![image.png](https://dn-odum9helk.qbox.me/FvDO853Az3irzPMezCMicS2pDPzb)
